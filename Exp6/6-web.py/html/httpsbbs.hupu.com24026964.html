（原文链接：不买OW！我们给你准备了四双一样拉风的休闲鞋！）
要说最近休闲鞋市场最炙手可热的鞋款，大家能想到的绝对不是某一双鞋，而是两个品牌——OFF-WHITE 和Nike。两个品牌的联名鞋款在市场的炒价少则是原价的两倍，多则直接冲着五位数而去。而这样的现象让许许多多的人将脚下的鞋子变成了自己财富的象征（虽然许多人都是在用父母的钱买……），而这其实并不是一件太好的事。



不过，发生都发生了，我们也只能静观其变（毕竟，没有钱……）。在“椰子化”和“OFF-WHITE化”的趋势越发可怕的情况下，依旧有一些高颜值休闲鞋款出现在消费者面前。那么，在不同的消费级别前，有什么样的鞋款能让你潮出独特呢？




2500-3000元级：adidas Yeezy 700
关键词：350都量产了，只能买700装逼了

在“人人有yeezy”的疯狂阶段开始之后，我相信你每天都能在街上看到那么一两双白椰子。在Yeezy 350v2开始逐步由限量走入量产之后，身为adidas爱好者/侃爷死忠粉/就是不怎么喜欢耐克的你，应该用2500多预算买一双什么呢？
当然是Yeezy 700了！



市场行情并不算太贵，但是Yeezy 700的货量和配色数远远没有350 V2多，在这一点上这双鞋也能让你的钱花的够值。当然了，就舒适度而言，中底内置Boost绝对不会让你失望，而颜值上……自然要看你的穿搭功力了！



街头稀有度：★★★☆
颜值：★★★★
性价比：★★★

adidas Yeezy Boost Runner   700

1100-2500元级：Nike React Element 87
关键词：詹皇上脚

说到React Element 87，大家自然会想到之前詹皇穿着上脚的那双土黄色。而这也正是它从那么多React跑鞋兄弟中脱颖而出的理由之一。不过，除了有巨星带货之外，还未被玩腻的“半透明”鞋面设计，配以全掌React的高端配置，以及相比Zoom Fly更偏向机能风的外形，无不是这双鞋得以在二级市场保持地位的原因。



当然了，如果你不喜欢半透明设计，但是却爱上了这双鞋的外形，他的兄弟款式Element 55也是个不错的选择。但如果你想要更加小众的版本，选择一些高桥盾的联名配色也是是十分不错的。



街头稀有度：★★★★
颜值：★★★★
性价比：★★★★

Nike React Element  87
900-2000元级：adidas ACE 16.1+ Ultra Boost
关键词：adidas的“吕布”

adidas ACE 16.1+ Ultra Boost对于一些篮球爱好者而言可能有些小众，但这双融合了足球鞋元素的Ultra Boost则是两年前就来到了许多潮鞋玩家的“狩猎范围”之内。从最早的“彩虹”配色，到之后粉色的KITH联名无一不是颜值极高。而在有了非限量配色之后，这双鞋也受到了更多球鞋爱好者的欢迎。



而在今年的世界杯结束之后，adidas依旧在继续扩大ACE 16.1+ Ultra Boost的配色队伍。至于是否能够复制Nike“吕布”的套路，成为阿迪潮流鞋款的战力，就要看adidas对于这双鞋市场口碑的维护以及配色推出频率的把控了。



街头稀有度：★★★★★
颜值：★★★★
性价比：★★★★☆

adidas ACE 16.1+ PureControl Ultra Boost

500-900元级：Nike Cortez
关键词：Kendrick Lamar

其实Cortez就数量而言应该是几双鞋里最普通的一款了，但是每当Cortez与其他品牌出现联名的的时候，往往也是比较难抢的款式之一。在最近两年成为说唱界话题人物的Kendrick Lamar对Cortez就情有独钟，在从Reebok来到Nike后的几年里先后推出了三双合作款Cortez，每一双都会让人感叹到“这小子的设计团队有、东西”。



而近期的Kendrick Lamar x Nike Cortez也悄悄登场。这双名为“House Shoes”的鞋款一改往日的戾气，以灰色和橙色为主色调示人。在极为生活化的配色背后，无论是后跟鞋带收紧的鞋型设计，还是巨大的外置鞋舌，都可以看出这双鞋真的是为“方便舒适的日常生活”而生。设计出挑，而却生活化十足，这样的鞋款的确值得令人欣喜。



稀有度：★★★
颜值：★★★★
性价比：★★★★

Nike Classic  Cortez
