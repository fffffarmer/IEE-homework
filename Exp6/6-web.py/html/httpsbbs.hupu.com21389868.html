前言：最近看到论坛好多人对这个网站充满好奇 在美帝闲得无聊便发篇帖子和大家唠唠嗑 如果有不对的地方或者有疏漏的地方多多包涵 有什么关于这些的问题都会一一回复
首先 Stockx这个网站对新用户都是有一个20刀减免的福利的

如图所示 

如图所示 当你需要买鞋的时候 把它发给你的折扣码输入图二的 Enter Discount Code 之后 就会自动减免20刀 一双鞋的邮费是13.95 刀 所以 这个20刀不仅减了邮费 并且还帮你省了6刀 但是需要注意的事 这个折扣码是有时间限制的 大概半个月就会失效 之前有看到有的人因为折扣码失效去找客服重新要回了 不过我还是建议不买就不需要注册账号 看准了再买 不过如果是竞价的话到没什么 因为有些鞋并不是你竞价了卖家就一定会成交的 这取决于你的价格卖家是否满意  那么怎么样才能再去拿这个20刀的折扣呢 原理很简单 再去开个新号 但是 由于你的旧号绑定了你目前的地址 你的新号就注册不了 所以这时候你要再去编辑一下你的旧号 把地址XJB乱写一通 让系统取消了你旧号的地址 这样才可以注册新号 最好能把你的支付方式再换一下 楼主曾经为了卡这个折扣换了五个地址 不过现在嫌麻烦也不想弄了
然后就是大家所关心的怎么捡白菜了 首先点进APP 看到这个Lowest ask 就是卖家的最低出价



点进去之后就是这样的一个界面 第一的选项的你要选什么品牌的鞋 第二个是你的尺码 第三个是你的价位




这样找白菜就方便多了 我是不会告诉你们我在上面买过40刀的消失的飞翼 150刀的ZK7的 
买完鞋子后呢Stockx就会发来一封邮件恭喜你 之后要做的就是等了 一般卖家我遇到的基本上都是两天内发货的 因为Stockx和UPS是合作伙伴 所以他们并不需要支付邮费 而且一般都是隔日达 鉴定大概也需要两天 到最后兜兜转转到你手上大概10-15天不等 
一般来说 Stockx的品控相对于Goat 来比较 是没他们那么严的 但我也没出现过有很严重的情况 不过GOAT上面的卖家职业的比较多 所以口碑比较好 然后呢鞋子一旦到你手上基本上就没有退货的可能了 除非有很严重的瑕疵或者是Fake 目前还没见过论坛和贴吧的人说Stockx上有假 所以基本上可以放心 然后竞价这个 在卖家接受之前你任意时候撤销你的出价 但一旦卖家接受了 就不可以取消了 所以不要觉得竞价好玩就去胡乱的拍 我来上一个活生生的例子


看到这个价格莫名想笑有没有 这价格都够我入两双大师之路了 然后呢竞价一定要对比最近的出价再给出你的价格 


如同所示 点击VIEW ALL ASK之后回进入到这个界面 然后就是最近的成交价了 右边Sale Price那里旁边有个倒三角 点一下是从最高价往下看 再点一下是从最低价往下看
