五年苦练，终成冠军——记万米接力与学院跑步氛围的建设

环境学院2017年04月18日电（通讯员 童心）4月6日15时30分，清华大学在紫荆雕塑园举办了第五届马约翰杯万米接力比赛，现场聚集了来自26个院系的400余人，场面宏大，蔚为可观。每个院系分别派出10名队员（8男2女）参加比赛，按照接力形式，每人绕雕塑园跑1圈，长约1000米，以10人跑步总用时作为最终成绩进行排名。经过激烈的角逐，环境学院参赛队员取得了乙组冠军、全校季军的历史最佳成绩，五年来首次实现了在该项目上零冠军的突破。汽车系、数学系分获乙组亚军和季军。

此次万米接力，我院派出陈悦（女）、金欣欣（女）、莫博超、王琦、孙若水、童心、张京辉、赵律童、谢昌益、赵凌浩上场参赛，分别位于第一棒至第十棒。在前两棒女生结束后，我院稍落下风。第三棒至第七棒的四位同学奋起直追，咬牙坚持，不断缩小与前面的差距。第八棒赵律童同学实力不俗，一举反超数人，奠定了我院乙组第一的位置。后两棒的同学超越自我，奋力完赛，最终稳稳保住冠军宝座。我院总用时约28分16秒，比去年快了约40秒。感谢各位参赛队员、高添教练与体育部后勤同学的付出，让我院迎来首个万米接力乙组冠军。

 

从左至右分别为：陈悦、金欣欣、莫博超、王琦、孙若水、童心、张京辉、赵律童、谢昌益、赵凌浩、教练高添

环境学院体育氛围浓厚。近年来，由于跑步的大众化程度越来越高，2016年2月26日，在院学生会体育部的大力支持下，热爱跑步的赵律童等同学成立了环境学院跑步俱乐部，简称“环跑”。“环跑”以让大家跑步锻炼为主要目的，让跑步爱好者们找到一起跑步的伙伴，互相督促、互相鼓励，养成跑步的好习惯。“环跑”每周不定期组织2-4次跑步活动，跑步形式包括自由跑或分配速跑，根据大家的能力和身体状况会调整活动的强度，并在教练的指导下进行跑前热身、跑后拉伸，通过科学跑步强身健体。“环跑”成立短短一个月内，参加同学就超过100人，院内同学的跑步热情相当之高。“环跑”活动一直持续至今，每学期均有新人加入，活动期间会穿插结合万米接力训练、校园马拉松训练等。
