精品教材

      1.《当代自然辩证法教程》2006年被评为北京市高等教育精品教材。此书2005年1月份由清华大学出版社出版，适合于研究生的必修课自然辩证法课程。

       该教材的编写基于对自然辩证法学科发展和教学内容改革方面进行的学术探讨，以及前期的教学实践探讨，将部颁教学要求与本校的教学实践结合起来，将案例教学的实践结合进来。该教材的每一章，附上一到两个案例，以及本章的思考题。从而促进研究生的自学，变被动学习为主动学习，以及强化自然辩证法教学的研讨性将起到积极作用。这是该教材尝试进行的一个创造，也是该教材最鲜明的特点。

     2 .由余谋昌等与我所雷毅老师合著的《环境伦理学》入选教育部2008年度普通高等教育精品教材。

清华大学优秀教材

    1.《科学技术概论（第二版）》获2008年清华大学优秀教材评选一等奖。该书由胡显章、曾国屏、李正风、吴金希、蒋劲松、高亮华、

 雷毅、张成岗编写，高等教育出版社出版，适用于本科生教学。

     2.《科学技术史二十一讲》获2008年清华大学优秀教材评选一等奖。该书由刘兵、杨舰、郭奕玲、戴吾三、蒋劲松编写，清华大学出版社出版，适用对象本科生。

     3.《当然自然辩证法教程》获2008年清华大学优秀教材评选二等奖。该书由曾国屏、高亮华、刘立、吴彤、李正风等编写，清华大学出版社出版，适用于研究生的教学。
