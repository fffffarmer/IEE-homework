环境学院团委

系团委在系党委和校团委领导下开展工作，以服务青年成长成才为工作的出发点，是广大青年同学的群众性组织。

目前，环境系团委经过多年的不断积累，传承和发展，趋于成熟，为广大青年创造了成长成才的舞台，在我系的人才培养工作中发挥了重要的作用。

按照“高效精简”的组织原则，组织机构设置如下：

以组织组，实践组和宣传组为例，简单做一介绍如下：

组织组：组织组是校系与基层支部联系的纽带。组织组向来本着“立足于同学，服务于同学”的宗旨，在紧紧围绕学校和系团委工作中心的同时，不断加强自身建设，完善和落实各项工作，力求及时反映同学的心声，带动各支部脚踏实地的推动我系团的各项工作。组织组常务性工作有每年一次的团支部工作等级评估、每两年一次的全系团员代表大会和平时的支书组委例会，以及团费收缴等工作。

实践组：以“学知识，长才干，做贡献”为目的，实践组充分利用各种资源，组织和开展各种假期的社会实践活动。尤其是我系的暑期社会实践活动从2004年开始全面突破，队伍数量质量均有较大水平提高，如2005年我系的社会实践开始向多年级合作、本研合作，师生合作的方向发展。经过充分的前期准备，共有13个支队，17个分队，足迹遍及12个省市自治区（包括澳门特别行政区），参与人数达150人。其中校级重点支队3支，本研结合支队4支，本研和教师结合支队2支。

在全校总结评比中，我系的“循环经济零距离”专业实践支队连续两年荣获校级“金奖支队”，同时获得“北京市专业特色实践支队”荣誉称号，同时，我系还荣获两支校级“银奖支队”、三支校级“铜奖支队”。

宣传组：宣传组主要负责每两星期一期的《信息简报》和每学期一次的《环境人》专刊，还有平时团委各项活动的宣传工作。《信息简报》及时的报道系团委的工作和各支部开展的各项活动，系刊《环境人》主要收录一学期各学生组织有代表性的活动纪实以及有关专业和环保方面的好文章。目前新近创办了《十日谈》短期刊物，力求及时、准确把握校园和系内外热点，进一步满足同学需求。
