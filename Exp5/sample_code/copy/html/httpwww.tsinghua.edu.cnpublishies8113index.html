            新经济与新产业研究中心

                              中心简介
 

我国正处在全球产业结构调整的背景中，发展新兴产业是适应我国建立创新型国家的总体发展目标和发展趋势，在引领传统产业升级，提升中国的软实力等方面，新兴产业承担着不可替代的重要作用。
在中国发展的关键阶段，急需一个能整合中国产业资源、促成产业跨领域技术创新与产业创新、帮助优秀产业项目孵化落地、培养产业各级各层面人才的产学研一体化机构，以推动中国各地新经济与新产业的健康、持续、蓬勃发展，清华大学作为中国的百年品牌高等学府，承担起这一任务责无旁贷。
新经济与新产业研究中心，作为中国新经济与新产业领域的重要机构，整合清华大学校内外与新经济与新产业相关的优势学科资源和研究成果，服务于中国新经济与新产业的蓬勃发展。中心吸收国内外一流研究机构的最新研究成果，将中心建设成为具有一流学术力量、较高教育咨询水平和项目运营能力的中国新经济与新产业研究与发展重镇。
中心愿与各级政府、企业界、投资界、研究专家和从事新经济与新产业的有志之士一起，为发展我国新经济与新产业做出贡献。
                                                                 

 

                                                              人员简介


王安富，中心副主任
中山大学管理学博士，笔名王宣喻。清华大学理论经济学博士后，比利时鲁汶大学访问学者，主要研究方向为新经济与新产业、战略管理、华商家族企业、企业理论等。主持国家自然科学基金项目2项，主持国家自然科学重点项目子课题、教育部长江学者团队子课题2项，主持教育部面上课题2项，主持博后面上基金和特等资助项目2项，以及其它省部级、厅级课题10多项。发表学术论文40多篇，出版和即将出版专著5部。国家自然基金、教育部课题评审专家库成员，部委和广东各省厅专家库成员。为部委和地方政府完成规划项目和政策研究20项，为国企和中小民企提供管理咨询服务和培训讲座60多项。

罗贞礼，农业经济学博士，中组部“西部之光”（清华大学）访问学者。中心研究室主任、研究员。
研究方向：区域经济、新经济与新产业、文化创意产业等。
曾在大学、科研院所、国有大型企业、上市公司、地方党委政府工作。主持或承担国家、省及市级重点研究课题与规划项目《湘南山地生态旅游可持续发展战略研究》、《赣州发展定位研究》、《深圳益田国际创意工场概念性规划》、《三级城市文化综合体产业研究》、《澳门创意产业年度发展报告2011》等10多项；发表学术论文“虚拟土战略与区域土地资源可持续利用的社会化管理”（冰川冻土）、“边缘区域经济协同发展理论与实践体系研究”（贵州社会科学）、“以新兴产业敲开跨越发展之门”（人民日报（理论版））、“战略性新兴产业:欠发达地区跨越式发展的战略突破口”（光明日报（理论版））等90余篇；出版《可持续发展背景下区域农业结构调整研究》（长沙:湖南人民出版社, 2005.7）、《边缘区域经济发展研究》（长沙:湖南人民出版社, 2007.4）、《新材料产业发展分析及策略研究》（北京:科学出版社, 2013.3）等专著。

张凯昭，中心课题研究部主任
研究方向与领域为：国民休闲的基础研究（重点：休闲社会学与休闲经济学）；国民休闲的应用研究（重点：政策、公共服务与产业。行业：旅游〈休闲〉、文化〈休闲〉与体育〈休闲〉）。
具体研究体现：新的社会分层（或分众）结构中的社区休闲与社群休闲；休闲发展与特色城市的转型（主要有文化型城市与文化型城市的建设指标体系、体育健康型城市与体育健康型城市的建设指标体系、休闲旅游型城市与休闲旅游型城市的建设指标体系）；休闲教育（学校、社会；多层、分众）；城市现代服务业的发展；集成旅游与经纪旅游的发展；一体化趋势下城市与乡村俱乐部经济及其产业的发展；区域农业、都市农业及现代农业服务业的发展；中国式休闲（狭义养生、广义养生与后现代全息养生）等。
主要研究课题与成果有：北京通州区“十二五”文化发展规划；上海浦东现代服务业发展研究报告；重庆九龙坡区文化创意产业发展规划；山东泰山旅游产业发展规划；河南禹州钧瓷文化创意产业规划；贵州省遵义市国家科技园区产业发展规划；贵州省湄潭农业产业园区发展规划；江苏省宿迁市生态体育发展规划；《中国重建“成人节”的生产力影响与理想（休闲实践）》、《大理州农、游集成发展的选择模式》、《中国式休闲——养生与全息养生》等论文。

李昶，博士
研究方向：宗教文化产业、文化产业政策、文化资源开发整合与利用、文化产业园区/基地规划与运营、城市特色文化建设路径等。参编《世界文化产业园》、《我国周边国家概况》等多部著作，发表近百篇文化产业研究文章。
社会兼职：曾任联合国军事观察员，现任工信部中国电子商会创意专业委员会副会长、中国宗教经济研究会常务理事、文化部中国传统文化促进会高级顾问等。

王旭光，中心副主任(兼)

清华同衡规划设计研究院经济金融研究所所长，清华大学学士、美国普度大学硕士。曾长期供职于清华大学教育基金会、美国施贵宝制药公司、美国惠氏制药公司、美国先灵葆雅制药公司等企事业单位，并曾兼任大纽约地区清华校友会副会长等职。主要从事政策、产业、科技和金融交叉创新研究，在科技成果转化与孵化、产业发展战略、政府引导基金、产研融互动共兴、县域发展研究等领域有着丰富的研究和实践经验。
