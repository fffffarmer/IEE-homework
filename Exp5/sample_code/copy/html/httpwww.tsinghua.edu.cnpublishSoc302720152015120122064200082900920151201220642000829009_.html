胡庆钧先生唁电

惊悉中国社会科学院民族学与人类学研究所研究员、学部委员胡庆钧先生不幸于2015年11月26日病逝，享年97岁。作为清华大学的民族学、人类学同仁，我们感到万分悲痛！在此，我谨代表清华大学社会科学学院人类学与民族学研究中心及社会学系，向胡庆钧先生的逝世表示沉痛的哀悼！并向胡先生的家人表示深情的问候！

胡先生于1940年进入西南联大社会学系，深受潘光旦先生的影响。1944年，获得硕士学位后，先在中央研究院做助研，继而应邀前往云南大学社会学系担任讲师，讲授人类学课程。1947年，胡先生调到国立清华大学人类学系。人类学系于1949年并入社会学系之后，胡先生在潘光旦、吴泽霖、费孝通先生领导的社会学系民族组工作。1952年，胡先生调入中国科学院近代史研究所。从1950年到1985年，他八上凉山，获得了大量令人信服的第一手材料，出版《明清彝族社会史论丛》、《凉山彝族奴隶制社会形态》等，译著《巴西印第安部落的黄昏》等。胡先生治学严谨、著述丰硕，在学术上造诣深厚，他将自己的生命奉献给中国的民族学和人类学，为民族学、人类学在中国的恢复和发展呕心沥血。他的不幸逝世，是中国民族学、人类学界的重大损失。我们将继承胡先生的遗志，为促进中国民族学和人类学的发展努力工作，以高水平的教学和学术研究告慰先生的英灵。

胡庆钧先生安息！


张小军（主任）
清华大学社会科学学院
人类学与民族学研究中心
2015年11月30日
