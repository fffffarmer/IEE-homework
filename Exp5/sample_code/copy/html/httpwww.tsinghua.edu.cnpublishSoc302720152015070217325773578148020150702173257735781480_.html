宋柏萱（Priscilla Song）教授做客费孝通学术系列讲座谈“从人类学的视角来了解干细胞疗法”

2015年6月16日下午，美国圣路易斯华盛顿大学宋柏萱（Priscilla Song）教授做客“费孝通学术系列讲座”，从人类学的视角来谈干细胞疗法。本次讲座由社会科学学院人类学与民族学研究中心主办，系第17讲。

宋柏萱教授通过其在北京某医院的田野经历，检视了全球干细胞疗法背后的文化和伦理问题。宋教授指出所谓的“海外干细胞旅游”实际上是流氓医生在剥夺绝望的病人。不同国家不同的伦理价值、规章制度、文化规范和财政考量使得对再生医疗的跨国探索成为可能。宋教授的研究超越了过分简化的刻板印象，提供一个干细胞科学的民族志描述，指出道德、社会经济及关于希望的地缘政治格局正在支撑着新的生物医疗和沟通技术。

讲授最后，来自校内外的师生与讲者进行了深入的互动。
