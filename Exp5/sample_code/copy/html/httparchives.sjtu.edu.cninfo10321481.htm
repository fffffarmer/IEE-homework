抚一纸情怀，书一片心声——“笺之语—上海交通大学收藏名人手札撷英展”开展

浏览次数：    日期：2018-06-28    撰稿：何嘉玲    摄影：陈晓俊

6月28日，由上海交通大学档案馆（党史校史研究室）举办的“笺之语——上海交通大学收藏名人手札撷英展”在徐汇校区董浩云航运馆拉开帷幕。

1936年5月15日蔡元培致黎照寰校长函

1947年8月14日钱学森致曹鹤荪函

在百廿年的办学历程中，交通大学曾与无数近现代各领域先驱人物、大师学者、社会名流相遇、相识、相交，期间留下了一封封有细节、有温度、有韵味的笔墨书札。此次展览从档案馆2000余卷历史档案里的数千书信中撷出近现代名人与交大往来的手札精华26通，均为首次对外公开展出。主要有“民国教育之父”蔡元培、“海上闻人”杜月笙、“布衣将军”冯玉祥、“七君子”之一沈钧儒、“状元实业家”张謇、“中国人口学第一人”马寅初以及中国航天事业奠基人钱学森等。细细品读这些妙文雅信，便可走进带着历史温度的笔墨世界，在字里行间感受老交大的办学理念与治校风格。

观展之余，主办方还专设“临摹席位”，邀请观众写下自己与交大的故事，留下观展感言，体会亲手书写信札的情怀和乐趣。

此次名人手札展最大程度地挖掘并发挥书信中蕴含的文化价值、道德价值、社会价值，在弘扬中国传统文化的基础上，助推广大青年学子、教职员工、海内外校友以及社会各界人士，成为坚定的文化传播者、继承者，为交大的发展、祖国的兴盛不懈奋斗。

此次展览为期3个月，时间2018年6月28日-2018年9月28日。
