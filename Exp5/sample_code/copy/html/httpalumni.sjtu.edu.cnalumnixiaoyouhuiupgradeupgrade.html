关于向上海交通大学校友开放统一身份认证jAccount和校友邮箱的通告

    一、为了给校友提供更优质的服务，从2015年4月3日9时起，面向所有取得上海交通大学学历学位的校友开放上海交通大学jAccount统一身份认证及校友邮箱(注册流程请参见第四点)。上海交通大学jAccount统一身份认证是广大在校师生的电子通行证，一个账户，可以登录学校为在校师生提供的所有信息系统。jAccount统一身份认证对校友开放后，学校、各院系和各级校友组织为校友开发的互联网应用都可通过jAccount登录。未来基于互联网的校友服务平台将实现“一个校友、一个账号、单点登录、终身有效”。

    二、即日起，新注册的校友邮箱地址为@alumni.sjtu.edu.cn，原@sjtu.org的校友终身信箱将不再接受新用户注册，但可正常使用至2016年3月31日，请校友们及时备份相关邮件。

    三、新邮箱容量5G，统一使用jAccount（上海交通大学统一身份认证）登录。

    四、相关注册流程如下：

    1、进入 https://register.alumni.sjtu.edu.cn ，选择jAccount注册，进入后按照提示步骤完善相关信息，即可开通校友jAccount和终身邮箱。详细注册步骤可参考：注册及服务使用说明。

    五、登录邮箱方式如下：

    1、使用网页 https://mail.alumni.sjtu.edu.cn ，用户名和密码使用jAccount账号和密码。

    2、使用邮箱客户端登录需要更改邮箱地址为：用户名@alumni.sjtu.edu.cn，POP3/IMAP为mail.alumni.sjtu.edu.cn，SMTP为mail.alumni.sjtu.edu.cn，用户名和密码使用jAccount账号和对应密码。

    3、可以通过校友会网站 http://alumni.sjtu.edu.cn 中的邮件链接直接点击登录, 用户名和密码使用jAccount账号和密码。

    六、建议通过IE8及以上版本、FireFox、Chrome浏览器登录，屏幕分辨率建议1024以上。

    七、交大邮箱迁移到同名校友邮箱请参考 http://net.sjtu.edu.cn/info/1079/1304.htm ，友情提醒：原有交大jAccount账号的校友请申请和交大邮箱同名的校友邮箱。

    八、注册校友jAccount和校友邮箱相关问题请参考 注册常见问题汇总 。

    九、应届毕业的校友请在确认在校手续都办理完成后再申请校友账号。

    十、其他相关事宜请联系上海交通大学校友会，邮件：alumni@alumni.sjtu.edu.cn ，电话：4001801896、021-54742208。
