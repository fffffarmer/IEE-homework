向Rachel Carson女士学习，做环境保护的先锋——余刚在环境学院2012届本科生研究生毕业典礼上的讲话

（2012年7月4日）

院长 余刚

各位2012届毕业生、各位老师、各位毕业生的亲友：

大家下午好！

昨天上午和今天上午，学校先后举行了研究生和本科生毕业典礼暨学位授予仪式。今天下午，我们在这里隆重举行环境学院2012届本科生和研究生毕业典礼。首先，我代表全院教职员工对环境学院2012届96名本科生、79名硕士生和20名博士生顺利完成学业表示热烈的祝贺！对不辞辛劳培育你们的班主任、辅导员、研究生导师、任课老师、行政管理人员以及你们的亲友表示衷心的感谢！

刚才班主任老师和同学代表在发言中回顾了你们在清华学习期间的难忘岁月；的确，2012届毕业生在学期间的表现也给我留下了深刻的印象：获得了“北京市先进班集体”，领导的绿色协会荣膺2009“绿色中国年度人物”，获得“清华大学特等奖学金”，成为伦敦奥运会火炬手，作为主力两次蝉联男子足球马约翰杯冠军……你们为环境学院的发展做出了贡献！

在刚刚过去的六月，在里约热内卢召开了“联合国可持续发展大会”，通过了大会文件《我们憧憬的未来》，为实现可持续发展奠定了坚实基础。与此同时，美国、英国、德国、日本等发达国家的政府、大学、民间机构组织了丰富多彩的活动，来纪念半个世纪前出版的一本被公认为全球环境保护运动的奠基之作——《寂静的春天》。在50年前的六月，《纽约客》杂志分三期连续刊登了这本当年11月才正式出版的著作。这些纪念活动，加深了我对这本环境保护经典名著及其作者Rachel Carson女士的认识，她们对人类的思维和行为产生了深远的影响，拉开了全球环境保护运动的序幕。

今天，在你们毕业之际，我愿以“向Rachel Carson女士学习，做环境保护的先锋”为题和你们分享三点感受，并作为我对你们的期望。

第一，我们要学习Rachel Carson女士热爱自然的高尚情操。

Rachel Carson女士从小就对大自然和野生动物怀有浓厚的兴趣，并立志成为一名作家。她在宾夕法尼亚女子学院读书时选择了英文专业，但是在大学三年级时她转到了生物系，这在科学界鲜有女性的时代是一个非常大胆的举动。她1929年大学毕业，1932年在约翰-霍普金斯大学以优异成绩获得海洋生物学硕士学位。毕业之后她先在马里兰大学教授动物学，同时在海洋生物实验室从事研究工作。1936年她被聘为美国渔业管理局的水生生物学家。她追随心灵深处对自然的热爱，在随后15年的工作中撰写了大量有关自然资源和环境保护的教育材料，出版了《海风下》、《我们周围的海洋》、《海洋的边缘》等著作，展现了她崇尚自然、热爱自然、尊重自然的高尚情怀。她在日后谈到写《寂静的春天》这本书的动机时说：“作为一名自然主义者，当我最珍视的一切都受到威胁时，将它们写成书就是我能做的最重要的事情”。

同学们已经接受了多年环境专业的教育，对人与自然的关系应该有正确的理解，应该继承“热爱我环境、光大我事业”的优良传统。在全社会环境意识仍然不高的今天，我们要学习Rachel Carson女士热爱自然的高尚情操，做环境保护的先锋，带动更多的人来热爱自然、敬畏自然、保护环境。

第二，我们要学习Rachel Carson女士实事求是的科学态度。

早在1945年，Rachel Carson女士就开始关注过度使用化学农药对环境的破坏，她写了一篇关于杀虫剂危害的文章投给一本著名杂志，但是没有引起重视。1958年，有关DDT等杀虫剂危害的证据越来越多，促使她决定动笔写下《寂静的春天》这本书，以期让社会关注DDT等杀虫剂的危害。在接下来的四年中，她本着实事求是的科学态度，开展了大量的调查研究。她发现底特律市在榆树上喷洒杀虫剂，导致知更鸟死亡，而且其体内检测出DDT，这些知更鸟以蚯蚓为食，而蚯蚓摄取了那些洒有杀虫剂的落叶。在此基础上，她认为在物种相互关联的自然环境里，仅仅以昆虫或害虫为目标的化学杀虫剂会迅速被其它生物体吸收并进入食物链，导致鸟儿死亡和生态系统被破坏。她提出了“没有鸟儿的歌声，寂静的春天随之来临”的科学预言。她预见到化学公司的反驳，因此研究调查方法非常严谨，她详细列出了所有的资料来源和以及审阅过手稿的专家名单，长达55页，使批评人士几乎找不到任何事实上的错误。

50年后的今天，无论是全球环境还是国内环境，面临的问题更加错综复杂。在今后的工作中，我们要学习Rachel Carson女士实事求是的科学态度，做环境保护的先锋，勇于面对纷繁复杂的环境保护难题，用我们的专业知识和综合能力去探寻破解难题的方略。

第三，我们要学习Rachel Carson女士不畏艰难的奋斗精神！

当年开展DDT等农药危害调查以及写作《寂静的春天》的时候，Rachel Carson女士强忍着因癌症切除乳房的痛苦，接受着放射治疗，期间还承受了母亲去世所带来的悲伤，以及因身体十分虚弱被阻止继续工作酸楚，以极大的毅力完成了著作的编写。《寂静的春天》的出版，被众多的化学公司视为严重威胁，招致杀虫剂贸易协会成员的愤怒嚎叫；化学工业界将Rachel Carson女士称为“一个歇斯底里的女人”……面对众多非议，Rachel Carson女士平静而理性地接受采访，并在多个国会委员会上作证，还呼吁建立一个“杀虫剂委员会”或监管机构来保护人和环境免受化学农药的危害。她不畏艰难地与化学工业界抗争，始终相信自己至少能起到一点作用。事实上，她的这种抗争精神已化为永恒的巨大力量！她推动美国在1970年成立了“环境保护署”，推动联合国1972年6月在斯德哥尔摩召开了“联合国人类环境会议”，推动联合国大会在1972年12月作出决议成立“环境规划署”，进而于1992年在里约热内卢召开“联合国环境与发展大会”，并于今年6月召开“联合国可持续发展大会”。在《寂静的春天》出版39年后的2001年，Rachel Carson女士在书中所提到的DDT等大多数化学物质已被统称为“持久性有机污染物”而被《斯德哥尔摩公约》在全球范围进行控制。

50年后的今天，尽管人类社会对自然环境重要性的认识大幅提高，但地球环境的状况却在不断恶化，全球范围内发达国家和发展中国家对可持续发展的看法不尽相同，一个国家内部不同部门、不同行业、不同地区对环境保护的态度相差甚远，环境保护和可持续发展面临着更大的挑战。因此，我们要学习Rachel Carson女士不畏艰难的奋斗精神，做环境保护的先锋，勇敢地与任何破坏环境的行为做斗争，用我们的力量为人类社会的可持续发展做出自己的贡献。

同学们，今年校庆期间，我们成立了“清华大学校友会环境协会”，与去年成立的“华东地区环境学院校友会”、“华南地区环境学院校友会”以及早先成立的“海外环境学院校友会”共同构成了环境学院加强毕业生联系和交流的网络；今年四月我们和“华东地区环境学院校友会”在上海组织了“清华大学华东环境论坛”，下半年我们还要在广东举行“清华大学华南环境论坛”；“环环相扣”系列活动不断开展，学院工作简报等信息不断发送给广大校友，海内外“清华环境人”更加紧密地联系在一起。

同学们，你们在2012年顺利完成学业并毕业，使遍布全球的“清华环境人”这个大家庭又增加了近200名新成员，“清华环境人”这个团队的力量变得更加强大！环境学院永远是你们的学术之家，永远是你们的坚强后盾！无论你身在何方，环境学院的全体教职员工都希望和你们永远保持联系！和你们永远在一起！

谢谢大家！
