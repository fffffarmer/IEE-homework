刘钝，清华大学社会科学学院科学技术与社会研究所特聘教授。1947年7月生于湖北武汉，先后就读于内蒙古师范大学数学系和中国科学院研究生院。曾任中国科学院自然科学史研究所所长（1997－2005）、中国科技史学会理事长（2004-2008）、《自然科学史研究》主编（1999－2004）、国际东亚科学技术与医学史学会主席（2002－2005）、国际科技史学会主席（2009-2013）等职。现为《科学文化评论》双月刊和《科学春秋》公众号主编，中国科技大学人文与社会科学学院院长,国际科学史研究院院士（Effective Member），剑桥大学邱吉尔学院海外研究员（Overseas Fellow）。专业领域是中国数学史，对科学社会史、科学文化也有所涉猎。编撰有《大哉言数》、《文化一二三》、《中国科学和科学革命——李约瑟难题及其相关问题研究论著选》等。
