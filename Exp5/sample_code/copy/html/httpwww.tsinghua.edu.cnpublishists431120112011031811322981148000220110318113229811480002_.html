俄罗斯科学院科技史所所长做客“海外名师讲堂”

2011年3月4日，俄罗斯科学院科技史所所长、俄罗斯宇航员尤里·巴图林（Yury Baturin）教授在主楼接待厅作客第95期清华大学“海外名师讲堂”。此次演讲是清华大学百年校庆活动“百场报告”之一。

巴图林教授演讲的主题是21世纪超长期航天事业发展预测。教授首先回顾了他为清华师生所作的 “海外专家短期讲学”的六讲要点，然后重点介绍了俄罗斯有关超长期航天发展预测的最新研究成果——《21世纪航天学》的主要内容，包括预测学方法和在21世纪未来90年间，每十年将发生的航天大事件。《21世纪航天学》完成于2010年5月，长达八百多页，是在鲍里斯·切尔托克院士（俄罗斯航天事业创始人之一）倡议下，由巴图林教授主持召集50多位来自各领域的俄罗斯学者通力完成的巨著。

巴图林教授为清华师生播放了自己拍摄制作的俄罗斯宇航员参加培训以及太空生活的纪录片。

清华学生就“国家间航天合作”、“国际航天法”、“空间碎片的分责处理”、“国际空间站的知识产权”、“太空争夺”等自己感兴趣的问题同巴图林教授进行了热烈地讨论。

科学技术与社会研究所王巍副教授担任主持。尤里·米哈伊洛维奇·巴图林（Yury Baturin），法学博士，现任俄罗斯科学院科技史所所长，俄罗斯联邦安全委员会委员，莫斯科大学教授。曾作为俄罗斯宇航员两次登上太空，完成20多项科学实验工作，被俄罗斯总统授予“俄罗斯联邦英雄”称号。在航空物理学、控制论、法学、新闻学、航天国际法学、航天未来预测学等方面均有建树。
