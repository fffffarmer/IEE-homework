我所师生参加“第16届第十六届全国科学哲学学术会议”并组织“科学实践哲学”分论坛

8月25-26日，由中国科学哲学专业委员会与山东大学哲学与发展学院所联合组织的“第16届第十六届全国科学哲学学术会议”在山东大学威海分校召开。我所吴彤教授、曹南燕教授、王巍教授等十余名师生参加了此次年会，并首次在年会中组织开设了“科学实践哲学”分论坛的讨论。吴彤教授、曹南燕教授分别做了题为“科学实践哲学视野中的观察”和“科学发现与建构论”的大会报告，王巍教授作为中国科学哲学专业委员会秘书长也全程参与了会议的筹备与组织工作。

附：部分师生报告题目

  • 王  巍：科学模型并非小说虚构
  • 蒋劲松：基于儒家伦理学进路反对吃猫狗肉的论证
  • 陈虎平：基因选择、群选择与社会合作
  • 苏  丽：在现象学视角下对阿瓦•诺伊行动主义知觉论的批判
  • 李芳芳：论哈拉维赛博格宣言到伴生物种宣言的转向意义
  • 周伊园：民族动物学及启示
  • 龙缘之：1930年代上海“兽片”热潮研究
