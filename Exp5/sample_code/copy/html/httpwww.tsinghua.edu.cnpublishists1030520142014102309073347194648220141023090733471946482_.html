中国科协-清华大学科技传播与普及研究中心参与主办全国地市级科技馆建设培训班

全国地市级科技馆建设培训班于2014年10月15-17日在合肥科技馆开班。本次培训由中国科协科普部、中国自然科学博物馆协会、中国科协-清华大学科技传播与普及研究中心主办，合肥科技馆承办。原中国科协副主席、党组副书记、中国科协-清华大学科学传播与普及中心理事长徐善衍，中国科协副主席、中国自然科学博物馆协会理事长程东红，中国科协科普部部长杨文志，安徽省科协党组书记王洵，合肥市委副书记凌云，中国科协科普部副部长辛兵，中国自然科学博物馆协会执行副理事长赵有利等领导出席了培训会，另有来自全国31个省、市、自治区的百余名代表参加了培训。

培训班开幕式

开幕式上由中国科协副主席、中国自然科学博物馆协会理事长程东红同志致欢迎辞。在致辞中，她对于科技馆建设和科普工作谈了几点看法：1、科技馆建设要适应科普工作发展面临的新形势和新挑战，紧紧把握住党委政府高度重视科普、广大公众亟需科普这一历史性良机；2、充分认识科技馆建设的重要性和紧迫性；3、科学规划、充分发挥自身优势，建好用好科技馆；4、结合本地实际，积极加强科普公共服务能力建设。 

徐善衍教授作主题报告

培训分主题讲座、专题讲座和分组讨论以及参观考察几个环节。原中国科协副主席、党组副书记、中国科协-清华大学科学传播与普及中心理事长徐善衍教授作了题为“科技馆的时代特征与内容建设的若干问题思考”主题报告。在讲座中，他就科技馆的走向、当代科技馆的特征和社会责任、当代科技馆展教内容和表现形式，科技馆内容建设过程的科学管理等问题进行的深入探讨和讲解。

中国科协科普部杨文志部长就信息技术手段对于科普的价值、原中国自然科学博物馆协会秘书长邵杰就科技馆展览工程工作流程、中国科协-清华大学科学传播与普及中心主任刘兵就科技传播理论的几个问题、中国科协科普部副部长辛兵针对地平线报告对科技场馆教育改革与发展的启示、青海省科技馆馆长徐东向就科技馆资源与科技教育活动、广西壮族自治区科协副主席梁春花着重科技馆科学教育、中国科协-清华大学科学传播与普及中心副主任刘立针对国际科学传播学、合肥科技馆馆长柏劲松对于科技馆可持续发展的创新精神、厦门科技馆馆长郁红萍就激活科技馆资源拓展多样化业务平台、临沂科技馆馆长宋颖就临沂市科技馆建设情况等相关问题做了专题讲座。 

刘兵教授作专题报告

在分组讨论的互动环节，气氛热烈，各个组深入交流了各地的建馆经验。大家一致认为本次培训非常及时，内容丰富，信息量大，“高大上”的理论与“接地气”的措施很好的融合于一体，同时各位代表也表达了对于常态化、分层次科技馆培训的期待。

刘立教授作专题报告

本次培训的讲座人有多年致力于科技馆工作的老领导、有熟悉科普政策的负责同志、有学识渊博的专家学者，也有来自各地方科技馆的负责人，不同视角的交叉比较全面的展示了我国科技馆建设中所遇到的问题并给出了中肯的建议。

总体来看，本次培训班是在我国省级科技馆布局基本完成、地市级科技馆建设方兴未艾的大好局面下开班的，其将为推进我国地市级科技馆事业又好又快的发展产生积极的影响。会议期间，大家参观了合肥科技馆。

全国地市级科技馆建设培训班由本中心徐善衍教授提议。本中心主要承担了培训班培训方案的设计、《会议指南》、《讲义集》的编制工作以及培训班问卷调查的工作。中心徐善衍、刘兵、刘立做了专题报告，刘兵主任还担任了会议主持人。中国科协及有关专家、学员对本中心的工作，表示很满意。

 

王永伟撰稿；刘立 等审阅
