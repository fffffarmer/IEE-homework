    被誉为“亚洲全能天后”

    她的实力像一块温润的玉

    经过岁月的洗礼终散发出迷人光芒

    她是台湾歌演艺三栖的全方位艺人

    ――杨丞琳

     

    用青春呼唤心灵，空中旋转四面舞台打造时空交错的视听盛宴

    本次演唱会的舞台设计可谓全面升级：360度旋转、圆形无死角、升降主舞台、多个超长延伸舞台...还特意增加了左右"旋转臂"，与观众席的距离是前所未有的接近，方便歌迷观看零死角舞台效果！舞美灯光精益求精，光鲜亮丽、炫目震撼又极尽声光效果的视角冲击感令人目不暇接，为观众打造一场震撼无比的视听体验。

     

     

    化身演唱会总策划，深V大露背等全新演出服装悉数亮相
    在这次巡演中，杨丞琳亲自担任总策划，并为自己量身定做了全新演出服装，比如在「年轮说」的音乐中，她身着深V大露背的银紫色服装惊艳亮相。银紫透明材质的头饰则运用「仿生」概念，模拟鸟类翅膀上毛羽的变化，由中心向外层层不对衬堆叠、排列了数种素材，呼应「青春住了谁」歌词第一句的「雏鸟站在悬崖将翅膀晒开」。还有年轮概念注入服装中，在压褶的布料上画出数个同心圆，透过扭转与拉伸的技巧，呈现出压褶后布料特有的蓬松度与空间感。

     

     

    现代舞独舞表演惊艳全场
    刚上任的全能总监杨丞琳表示，"这次想好好唱歌给歌迷听，站在粉丝的出发点去设想所有表演的环节，保证大家看完，不只有看了一场Show！"虽在巡演前，丞琳曾公开说这一次不会跳舞，但这次演唱会仍然精心准备了现代舞独舞环节，使出浑身解数为歌迷制造惊喜，以浓烈转为细腻、悲桑、狂奔的舞姿收服所有歌迷的心。

     

     

    出道十八年不忘初心，新单曲升格制作人百分百表达自己的音乐理念
    杨丞琳是娱乐圈少有的唱跳说皆精，歌演艺三栖的全能艺人，堪称娱乐圈跨界典范。从16岁组合出道到21岁单飞，短短几年内狂揽中港台三地新人奖。第二张个人专辑便热卖一百二十万张……每次出现，我们都能见证她的成长与蜕变。

     

     

    今年的6月1日，杨丞琳推出了全新单曲《忘课》，这也是她主演的热播剧《前男友不是人》里的片尾曲，无论是歌曲还是剧中的角色，都获得了不少粉丝的青睐和追捧。此外，她也通过这个契机升格为制作人统筹自己的音乐，用100%的空间表达自己的音乐理念。

     

    18年的累积，10张专辑，我们一起听过、看过，一起感同身受。以4inlove组合出发的《一千零一个愿望》，一炮而红的第一首个人单曲《暧昧》，独特的琳式情歌《左边》、《匿名的好友》、《带我走》、《雨爱》……首首都是传唱度极高的作品，也成为青春记忆里必不可少的经典。要问青春住了谁，80、90后的青春里一定住有杨丞琳。
