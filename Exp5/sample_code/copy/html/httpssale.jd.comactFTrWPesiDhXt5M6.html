    儿童家具风险提示

    儿童家具区别于普通家具，儿童家具执行GB 28007-2011《儿童家具通用技术条件》，普通木家具执行GB/T 3324-2008《木家具通用技术条件》。儿童家具标准严格，标准是强制性，在重金属限制多，在做工要求、稳定性（固定装置）、安全性（从柜内侧轻碰能打开）、警示语等多方面的要求严于木家具标准。儿童家具定义为供年龄为3岁～14岁的儿童使用，该年龄段的儿童因年龄较小，自我保护的意识和能力较差。消费者在挑选儿童家具，要选择儿童家具专售区域，产品应注明执行儿童家具标准。另外，还要注意儿童家具的结构安全方面。

    首先，产品不应有危险锐利边缘及危险锐利尖端，棱角及边缘部位应经倒圆或倒角处理。存在边缘及尖端的产品，儿童使用时较易产生危险，锐利的边缘和尖端容易对儿童的身体造成创伤性伤害。其次，产品除在离地面高度或儿童站立面高度1600mm以上的区域外，不应使用玻璃部件；管状部件外露管口端应封闭；产品中抽屉、键盘托等推拉件应有防拉脱装置，防止儿童意外拉脱造成伤害；由于儿童因为年龄较小，较为顽皮好动，高桌台及高度超过600mm的柜类产品重心较高，当门、抽屉等部件在开启的状态下，儿童攀爬后较易发生倾翻，从而造成儿童的身体伤害。因此，所有高桌台及高度大于600mm的柜类产品，应提供固定产品于建筑物上的连接件，并在使用说明中明示安装使用方法。另外，标准规定，产品应在使用说明中明确标示产品适用年龄段。如产品有需安装、有折叠或调整装置、有升降气动杆的均应标示警示语，防止产品出现误用。

    鞋类商品风险警示

    在近期商品质量抽检工作中，北京市工商局对鞋类商品进行了抽检。抽检结果显示市场上部分成人鞋商品质量存在问题，涉及的主要问题为成人鞋的标识、帮底剥离强度、耐折性能、勾心等项目不符合相关标准要求。北京市工商局已对不合格商品的销售者依据《产品质量法》的相关规定进行立案调查，相关商品的销售者应做好相同生产者相同型号不合格商品的退市工作。公示的不合格商品生产企业和有关经销单位应按照相关法律法规的要求主动采取措施，保护消费者合法权益。消费者在选购鞋类商品时应予以注意，切勿选购相同生产者相同型号的不合格商品。

    家庭卫生杀虫用品风险警示

    在近期商品质量抽查检验中，北京市工商局对家庭卫生杀虫用品类商品进行了质量抽检。发现“癞蛙”、“强达”等品牌家庭卫生杀虫用品类商品质量不合格，涉及的主要问题为有效成分使用要求、毒理、有效成分含量及允许波动范围、烟尘量、连续燃点时间、标志、净含量、标志等项目不符合相关标准要求。北京市工商局已按照有关法律法规对本次抽查检验中销售不合格商品的经销单位立案调查。本次公示的不合格商品生产企业和有关经销单位应按照相关法律法规的要求主动采取措施，保护消费者合法权益。消费者在选购家庭卫生杀虫用品类商品时应予以注意，切勿选购相同生产者相同型号的不合格商品。
