用行动去诠释“热爱我环境”——余刚在环境学院2011级新生开学典礼上的讲话

（2011年8月29日）

院长 余刚

各位2011级新同学，老师们、同学们：

在2011级本科生军训和研究生强化专业教育紧张进行之际，今天晚上我们在这里隆重举行环境学院2011级新生开学典礼。首先，请允许我代表环境学院，再次对198名环境学院的本科生和研究生、18名深圳研究生院的研究生和24名地学中心的研究生进入环境学院学习表示热烈欢迎！

在改革开放以来的30多年中，清华大学为国家培养了3000多名高层次环境专业人才，包括2000多名本科生、1000多名硕士生和300多名博士生，这些优秀的“清华环境人”正活跃在环境保护及其相关领域，服务于我国乃至全球的环境保护事业。在一代又一代“清华环境人”的成长过程中，环境学院形成了优良的传统和团队文化，特别是上世纪八十年代的学子提出的“热爱我环境，光大我事业”的口号，得到了众多“清华环境人”的认同并代代相传。今天，我想和你们分享我对“热爱我环境”的三点认识。

首先，“热爱我环境”就是热爱我们的自然环境。由水、大气、土壤、植物、动物等构成的自然环境是人类赖以生存的物质基础，人类是自然环境的产物，而人类的活动又影响着自然环境。中国古代哲学“天人合一”的思想强调了天和人和谐一致的关系，季羡林先生在其著作中列举了孔子、孟子、董仲舒、程颐、张载、老子、庄子、墨子等思想家对“天人合一”思想的论述, 认为这一思想是东方文明的主导思想，是东方综合思维模式的具体表现。《老子》第二十五章的“人法地, 地法天，天法道，道法自然”也很好地阐释了人与自然环境的依赖和制约关系。然而，自西方工业革命以来，人成了自然环境的主宰，对资源的索取超过了自然环境的承载力，污染物的排放量也超过了自然环境的容量，导致人与自然环境关系的尖锐化，人类赖以生存的地球濒临失衡。“热爱我环境”，就是要正视自然环境的价值，热爱自然环境，遵循人与自然环境的依赖关系，始终做人与自然环境和谐共生的践行者。

第二，“热爱我环境”就是热爱我们的环境专业。与数学、物理、化学等传统的基础学科相比，环境专业是一个年轻的专业。1972年联合国在斯德哥尔摩召开的“人类环境会议“是全球环境教育运动的开端，1977年联合国教科文组织和联合国环境规划署在前苏联第比利斯召开的政府间环境教育会议是环境教育发展史上重要的里程碑，《第比利斯宣言》明确提出了环境教育的目标包括意识、知识、技能、态度和参与五个方面，拓展了环境教育的内容和方法，为全球环境教育的发展构建了基本框架。我国的环境教育几乎与国际同时起步，1977年近10 所学校设立了环境工程、环境科学、环境监测等专业，1984年教育部正式确定了环境工程、环境科学等作为本科专业开始招生，随后高等环境教育快速发展，迄今数百所大学开设了环境专业，因此环境专业常被评选为最有发展前途的专业。与此同时，我国高等环境教育规模的快速增大导致环境专业的整体质量有待提高，培养的人才难以满足社会的需求，因此环境专业又常被评为就业最差的专业。“热爱我环境”，就是要正确认识环境专业综合性、交叉型、多样化的特点，在基础阶段学习中注重工程与科学结合以及技术与管理结合，具有鲜明的环境学科特色；在专业阶段学习中根据自身的特长与兴趣，选择环境工程、环境科学、环境规划与管理、市政工程、核环境工程等方向，接受分类培养，为发展适合自身特点的环境人生奠定坚实的基础。

第三，“热爱我环境”就是热爱我们的环境学院。尽管环境专业是一个年轻的专业，但我们的环境学院却有着悠久的历史。她源于1928年设立的清华大学市政工程系，在我国环境工程学科的奠基人――陶葆楷先生的带领下，经过几代人的艰苦奋斗，环境学科逐步发展壮大，1977年建立全国第一个环境工程专业，1984年成立环境工程系，1997年更名为环境科学与工程系，2011年发展为环境学院。今天的环境学院拥有实力雄厚的师资队伍，以国际先进的绿色建筑为基地，建有高水平的国家级实验室以及和国际组织、著名环保企业共建的联合研究中心，承担了一大批重要的国家重大研究任务以及国际合作、企业合作研发课题，与哈佛大学、耶鲁大学、斯坦福大学等国际顶尖大学、世界资源研究所等研究机构以及联合国环境规划署等国际组织长期开展实质性的人才培养和科研合作。环境学院还在学校层面发起和推动了清华大学的“绿色大学”建设，学生“绿色协会“更是连续多年位列清华大学十佳社团之首。环境学院长期担任教育部“环境工程教学指导委员会”和“环境科学与工程一级学科教学指导委员会” 的主任单位，带动中国高等环境教育的发展。“热爱我环境”，就是要更快更好地融入环境学院这一集体，弘扬环境学院敢为人先的开拓精神、勇于创新的科学精神和集体利益至上的团队精神，和全体师生一起抓住发展环境学科难得的历史机遇，积极参加各项改革，为建立一个更加强大的环境学院而共同奋斗。

在座的90名本科生、61名硕士生和47名博士生是环境系发展为环境学院后招收的第一届学生，你们加入到环境学院这个优秀的集体将使她更加强大！面对当今国家严峻的环境形势和艰巨的环保任务，期待2011级新同学将其作为今后学习的巨大动力，用行动去诠释“热爱我环境”，充实地度过在校期间的每一天，为日后拓展你们多姿多彩的环境人生打下坚实的基础！

谢谢大家！
