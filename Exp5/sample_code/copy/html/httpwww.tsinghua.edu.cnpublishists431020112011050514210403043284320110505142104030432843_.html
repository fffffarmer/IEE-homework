“科学社会学与科技政策学术研讨会”召开（图）

11月25－26日，“科学社会学和科技政策学术研讨会” 在香山饭店举行，会议由科技部中国科技战略研究院STS研究所和清华大学科技与社会研究所共同举办。来自科技部科技发展战略研究院、清华大学、中国科技大学以及自然科学基金委的研究人员约30人参加了研讨会，就科学社会学与科技政策的相关热点问题进行了深入的探讨。由李正风教授带队的清华研究小组共12人参加了会议，其中有10位代表做了大会报告。

                                     

25日上午的会议由中国科技大学徐飞教授主持。李正风教授做了开场报告，围绕社会科学研究方法以及创新系统理论进行了深入浅出的论述，对社会科学的“建构性预测”、社会科学中的理论、创新系统理论等问题进行了分析。随后，科技部战略研究院科技与社会发展研究所副所长赵延东研究员做了题为《我国科研人员合作网络研究》的报告，该研究以坚实的调查数据为基础，运用整体网络分析法探讨了我国科研人员合作网络的规模、密度等，并提出了政策建议。来自战略研究院的孔欣欣以浙江纽扣产业集群为案例，做了《创新与产业集群的可持续发展----浙江纽扣产业的案例》的报告；李兵博士（清华大学）则以北京为案例，介绍了其博士论文的精髓——《我国建立研发枢纽的条件及发展战略研究》。

下午的会议分别由李正风和赵延东主持。我校科技与社会研究所教师洪伟博士做了题为《异地校企合作的决定因素:一项基于中国专利数据的分析》的报告，该研究数据扎实，论证清晰，获得了与会代表的一致好评；程志波博士以治理理论为框架，介绍了其对“学术共同体”权责的研究；梁永霞博士运用知识图谱方法，绘制了科技政策的“研究地图”，为学科建设提出了若干建议；高璐、尹雪慧也分别就生物技术治理、边界组织为主题，做了学术报告。来自战略研究院的张文霞、何光喜等四位代表也在下午的学术报告中发言，其主题集中在科学家群体的职业认同、对科学家的刻板印象及影响因素、学术不端等问题。

第二天上午的会议由自然科学基金委员会政策局的龚旭处长主持。徐飞教授以我国的院士分布及相关规律为基础，做了以《科学界的中国国情及其对中国科技战略的影响》为题的生动的报告。我校科技与社会研究所张成岗副教授也应邀做了题为《鲍曼现代性社会学中的技术图景》的报告；此外，崔永华博士和丁大尉也分别以《学会在国家创新体系中的作用：IEEE的案例》和《科学信息的开放存取与知识“公有性”的认识回归》为题，和与会代表进行了交流。  

下午，部分参会老师及同学组织了爬山活动，研讨会完满结束。 

本次研讨会是我校科技与社会研究所与科技部科技发展战略研究院共同组织的第一次学术会议，宗旨是促进学术交流、展示研究成果、学习研究方法以及促进学术网络的建设。战略研究院科技与社会发展研究所的研究以社会学方法见长，而我校科技与社会研究所的研究则以科技哲学及STS为基础的政策研究为特征，跨学科的交流为研究者们打开了思考问题的新思路。通过交流，大家就“社会网络”、“学术共同体”、“治理”等问题展开了深入而有益的讨论，为进一步的学术交流与合作奠定了基础。
