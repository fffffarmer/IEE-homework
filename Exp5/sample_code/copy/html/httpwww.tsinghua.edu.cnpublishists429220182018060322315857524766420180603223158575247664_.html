我所毕业生张明君、杨仁杰主讲清华大学科学哲学与技术哲学沙龙第156期

2018年6月1日下午，北京科学哲学论坛2018年第4期暨清华大学科学哲学与技术哲学沙龙第156期在清华大学明斋241会议室成功举办。

宾夕法尼亚大学哲学系博士生张明君做学术报告“因果要素的相对重要性”。在进化生物学中存在着“自然选择是否是影响进化的最重要的因素”的争论。张明君在报告中区分了四种意义上的相对因果重要性，并选择其中最常见的一种进行较为深入的探讨。他认为，在很多情况下，比较不同因果要素的相对重要性的困难，来自于不同因果要素之间的异质性所带来的不可通约性。

卡内基梅隆大学哲学系博士杨仁杰做了点评。他从因果性的最新研究理论出发，认为四种意义的相对因果性定义，其实最终都可以在操控主义的因果理论中得到说明，因此存在统一的因果重要性。

本次沙龙的报告人张明君与评论人杨仁杰都是在清华大学理工科本科毕业，在清华大学科技与社会研究所获得哲学硕士学位，随后到美国名校攻读哲学博士学位。他们共同为清华的师弟妹们带来了国际科学哲学的前沿信息。

本次沙龙由中国自然辩证法研究会科学哲学专业委员会与清华大学科技与社会研究所联合主办。王巍教授主持沙龙。清华大学科技与社会研究所吴彤教授、中国社科院哲学所段伟文研究员、首都师范大学谭笑副教授以及清华大学的师生参加了沙龙。
