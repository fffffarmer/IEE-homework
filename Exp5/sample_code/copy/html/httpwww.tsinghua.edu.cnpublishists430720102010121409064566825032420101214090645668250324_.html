科学哲学与技术哲学沙龙关注“人类社会中的非人类动物”

2010年12月10日下午，清华大学科学哲学与技术哲学第98期沙龙在新斋353举办。本次沙龙邀请了中国人民大学哲学学院现代逻辑与科技哲学研究所所长刘晓力教授做“认知科学对哲学的挑战——延展认知与延展心灵论”的学术报告。

          

刘晓力教授首先介绍了认知科学的新进展，重点分析了克拉克（Andy Clark）和查尔默斯（David J.Chalmers）近期提出的延展心灵与延展认知理论。克拉克与查尔默斯提出，人类的认知与心灵可以超出人的头脑，延展到世界，例如iphone和笔记本电脑。刘晓力教授对延展心灵持批评态度，并且提出了三大论证予以反驳。她对延展认知持部分的同情态度，但是也提出了三点质疑。她最终认为：“我们对认知科学具有决定意义的一些基础性假设，既没有提供足够的令人信服的经验证据，也没有对其展开充分的哲学论证。我们仍然是瞎子在摸象。”

清华大学科技与社会研究所的王巍副教授做了点评。因为刘晓力对延展心灵与延展认知都持批判态度，所以王巍对她的三大论证与三点质疑都做出了回应，认为延展认知与延展心灵论确实都有可取之处。王巍还从科学哲学与技术哲学的角度，提出了不同于心灵哲学的视野。

清华大学科技与社会研究所蒋劲松副教授主持了沙龙。清华大学以及北京兄弟院校的近30名师生参加了沙龙，并进行了热烈的讨论。
