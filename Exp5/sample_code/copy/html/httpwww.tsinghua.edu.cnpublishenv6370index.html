环境污染控制的新理论、新工艺、新技术和新设备

将相关学科的高新技术，如现代生物技术、高效物理化学技术、新型材料技术和先进生态技术等应用于环境污染控制领域，通过相互间的渗透，完善环境污染控制领域的理论基础；将基础理论研究、新技术和新工艺的开发以及设备的研制结合为一体，开发出适合中国国情的高效且低耗的环境污染控制成套技术。

经济－社会－环境资源协调发展的理论与方法

从国民经济的工业布局和产业结构、土地与资源利用等方面研究经济－社会－环境资源可持续发展的方法与理论，为国家各级决策部门提供适合我国国情的环境－经济－社会协调发展战略及技术措施；以软科学研究为先导，以高新技术为支撑，为国家环境保护综合决策服务。

区域清洁生产与资源再生利用

在化工、食品、机械、电力等行业开展清洁生产的研究，提供与我国国情相适应的清洁生产系统的理论框架、评价方法及实施程序；同时大力开展资源再生利用研究，为国家政府和各行业部门推行清洁生产和资源再生利用、发展绿色产业提供指导方法和技术支持。

能源利用和全球环境问题

结合我国国情，主要从能源有效利用方面来研究城市和区域大气及全球环境问题。对燃煤和新能源利用进行全生命周期控制研究，为我国削减能源利用方面的污染及解决全球环境问题提供决策和技术对策；同时加强对气候变化等全球环境问题的研究，分析影响因素及其效应和对策，为国家决策部门在能源环境问题上的战略决策和国际交往提供依据。
