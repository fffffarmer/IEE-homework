       清华大学周培源应用数学研究中心（以下简称"中心"）是清华大学于2002年设立的开展交叉科学研究的实体，其宗旨是：推动数学和经验科学的交叉研究，促进应用数学和经验科学的共同发展，活跃学术思想，开展国际学术交流与合作，培养具有创新能力的杰出人才。

中心聘请国际著名应用数学家、美国科学院院士林家翘先生(已故)为名誉主任，美国布朗大学荣休教授谢定裕先生为首任主任，海内外著名科学家担任顾问或学术委员会委员，并聘请一批海内外知名学者任研究员和客座研究员。

中心以"创新、交叉、综合"为研究活动的主导思想，在数学和经验科学的前沿领域中，选择对未来科技发展可能产生重要影响的并对探索自然规律有深远意义的问题作为研究内容。主要研究领域包括：理论生物学（蛋白质折叠、系统生物学、计算神经科学）、应用数学方法（应用偏微分方程、科学计算、概率统计、非线性波动理论）等。

遵循"精干、择优、流动"的原则，中心积极吸收国内外的科学英才，保证人员结构合理和高效，并为研究人员提供优良的工作条件和宽松的学术环境。中心向校内外一切优秀学生和青年科学工作者敞开大门，每年将在国内外招收博士研究生和博士后研究人员。中心积极与海内外相关研究机构开展学术交流合作。邀请海内外杰出科学家和优秀留学人员来中心访问、讲学和合作研究，并努力创造条件鼓励和资助研究人员出国访问和参加国际学术会议。

新技术革命向科学界提出一系列挑战性的新课题，也为二十一世纪的基础科学研究提出了新的机遇。面对挑战和机遇，清华大学周培源应用数学研究中心将努力加快建设步伐，逐步发展成为具有国际先进水平的、在国内外有重要影响的国家级基础研究基地，最终成为世界性的应用数学研究中心。
