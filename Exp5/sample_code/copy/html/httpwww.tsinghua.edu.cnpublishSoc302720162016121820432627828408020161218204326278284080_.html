清华大学“社会治理与社区建设”研讨会顺利召开

清华大学“社会治理与社区建设”研讨会顺利召开
清华大学社会学系


  2016年12月10日，由清华大学社会学系、中国社科院社会学研究所、中国社科院社会发展战略研究院、中国社会学会社区研究专业委员会、社会科学文献出版社、清华—友成社会创新中心六家单位共同发起举办的“2016年度‘社会治理与社区建设’研讨会”在清华大学主楼接待厅顺利召开。中国社会科学院副院长李培林出席会议并发言；中国社会学会会长、清华大学社会科学学院院长李强，中国社科院社会学研究所所长陈光金做大会开幕致辞。来自国内社会学界相关领域的三十多位重量级专家与学者悉数到场，并就各自所研究与关注的领域进行了发言分享。

 李强老师在致辞中首先作为东道主，对莅临清华大学参会的各位老师表示了欢迎与感谢。其次，李强老师对当前我国社会学界学的三个景象做出了总结：一、学术交流做到中西融汇；二、社会学本土化取得了长足的进展；三、社会学界更为团结与整合。随后，李强老师从“社会治理”这一概念的提出，到当前学界所进行的社区研究与社区实验，对本次会议的主题——“社会治理”与“社区建设”进行了具体的阐述。最后，李强老师向参会的各位同仁介绍了清华大学社会学当前的学科建设情况，从纵向领域回顾了清华社会学的学科发展史，在横向领域介绍了当前清华社会学所涵盖的社会分层与结构、转型研究、城市社会学、社区规划、文化人类学、医学社会学、社会心理学、科学社会学、社会工作等领域，重点推荐了当前清华社会学在清河与大栅栏所开展的社区实验。

 李培林老师代表会议主办方，首先也对到场的诸位学界同仁表示了欢迎与感谢。其次，他重点强调了“社会治理与社区研究”这个议题在当前学术领域及政策层面的重要意义。最后，李培林老师对我国有关社区研究的三个主要领域做出了自己的总结与展望：社区实验的蓬勃开展、转型时期的城乡社区研究以及大数据在社区研究中的应用。
  陈光金老师首先表示了对于各位参会老师的欢迎与感谢。同时，就当前社科院系统的特征——为学术研究提供平台、为社会各界提供智库支持，进行了具体的介绍。陈光金老师重点指出了当前社科院系统与高校在科研领域合作的优势，倡导多领域学术交流合作应实现规范化、机制化、多元化与常态化。最后，陈光金老师对未来哲学社会科学领域赢得更多国际话语权的前景做出了展望。
  随后，会议分为四个发言板块，由参会的各位专家学者就各自关注的主题先后发言，内容几乎涵盖了当前转型时期社会治理与社区建设领域最为关键的议题——从宏观的制度设计到微观的具体操作；从经典理论的反思到社区实验的实践；从具体的资源与绩效到抽象的文化与情感；从城市社区到农村社区。
  详细的29篇发言稿未来将会在微信公众号“清华社会学”连载推出。
  现场旁听的近200多位来自全国各高校及科研院所的学者与学生都表示从本次会议的主题与发言中受益良多——不仅了解到了相关领域研究的全貌，同时也收获了学科最前沿的研究分享。
  未来清华社会学的学科建设会继续坚持与国际研究对话、与国内同行交流，应对各种新的经济、组织、政治和文化现象，追求理论创新；坚持对当代人文社会科学领域的知识加以融会贯通，实现跨学科知识交流与融合；坚持针对转型时期的真问题展开探讨，对中国的新鲜经验进行社会学的概念化建构，成为社会科学的共享知识；坚持在老一辈社会学家开展的社区实验之基础上，继往开来，研究社会，同时改造社会。
