“与张全学长面对面”——环境学院“环环相扣（EnviroNet）”首场活动举办

3月9日上午，清华大学环境学院“环环相扣（EnviroNet）系列活动——与张全学长面对面”在中意环境节能楼举行。上海市环保局局长、环境学院1987届校友张全与师生们分享了自己的人生经历和感悟，并介绍了上海市环保事业的发展和挑战。中国工程院院士郝吉明，清华大学校友总会理事钱锡康，环境学院院长余刚、党委副书记刘建国以及近50名师生参加了座谈会，余刚主持了本次活动。

此次活动为环境学院“环环相扣（EnviroNet）系列活动”的第一场，余刚首先向嘉宾和同学们介绍了开展这一系列活动的背景及意义。今年正值清华大学百年校庆，环境学院借此机会加强清华环境人之间的联系与协作，以便在不同地区各行各业工作的各届毕业生增进交流，形成网络，达到共同发展与进步。“环环相扣”系列活动所体现的正是清华环境人之间相互提携，密不可分的院友之谊。

张全同余刚院长及郝吉明院士合影

随后，张全与大家一起分享了自己的人生经历及感悟。张全说，一踏进校园，熟悉的亲切感便让他怀念起往昔在清华大学的求学时光及生活趣事，虽然许多年过去了，但当年之爱好依然是如今最怀念的事。回顾自己的发展历程，张全风趣地概括道：“我摇过烧杯，画过图纸，跑过现场，一路从技术岗位走向了行政岗位。”谈到自己的本职工作时，张全向大家系统介绍了上海市近几十年来在环境保护方面所取得的成绩、当前面临的主要环境问题以及未来环保工作的新思路。最后，他还特别结合上海市以及我国未来环保工作在管理、科技和产业方面的需求，指出环境专业将大有可为。

全体合影

在互动交流环节中，同学们提问踊跃：上海市建设国际航运中心和金融中心对环保工作提出了哪些挑战？在上海市的环保工作中，政府部门、科研机构和企业各自发挥了怎样的作用？未来中国环保产业发展前景如何？低碳发展对于中国意味者什么？……张全耐心地回答了每个同学的问题。期间，一位研究生向张全赠送了课题组保存的他的几张老照片，把整个活动推向高潮。最后，张全寄语在场同学：“不管将来从事什么行业，专业都是基础，希望同学们充分利用清华大学的丰富资源，专心学习，锤炼品性。

活动最后，余刚再次对张全在百忙之中抽出时间返校与同学们交流表示衷心感谢，并表示环境学院将在今后持续举办多场“环环相扣”系列活动，让同学们对环保事业的各层面、对社会的各行业有更充分、更多样、更清晰的认识与了解，使同学们真正做到“心中有目标，脚下有坐标”。

附：张全简介

张全，1987年毕业于清华大学环境工程系，获学士学位。历任上海市环境科学研究院总工程师助理，上海市环保局污染控制处副处长、处长、局长助理兼污染控制处处长、副局长、党委副书记，上海市城市建设投资开发总公司副总经理。2007年9月至今，任上海市环境保护局局长、党委副书记。第十一届全国人大代表。
