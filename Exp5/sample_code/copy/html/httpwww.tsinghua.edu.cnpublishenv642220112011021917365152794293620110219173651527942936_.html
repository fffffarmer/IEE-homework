研究领域

针对新兴污染物（POPs、PPCPs等）开展污染识别表征、污染物化学和污染控制化学研究。

近期研究兴趣包括：

1. 氯/溴/氟代POPs废物的高能球磨机械（力）化学处理技术

2. 基于POCIS被动采样和质谱联用技术检测水中痕量污染物

3. 焚烧/冶金/化工过程二恶英等副产物POPs的生成/排放/控制

4. 全氟化合物（PFOS、PFOA等）的风险防控与替代技术
