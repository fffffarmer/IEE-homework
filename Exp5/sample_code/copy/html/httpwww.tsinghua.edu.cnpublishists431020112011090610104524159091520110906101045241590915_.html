我所师生参加第十五届科学哲学学术会议

 

  2011年8月24日至27日，第十五届全国科学哲学学术会议在大连理工大学伯川图书馆成功举行。本次大会为期4天，主题分别为自然科学哲学问题、社会科学哲学问题、科技伦理学、科学文化哲学问题，围绕4个主题进行了两次大会报告和4个分会场的专题讨论。

  本次大会开幕式由中国科学院研究院胡新和主持，大连理工大学校领导、科学哲学专业委员会荣誉理事长邱仁宗、大连理工大学人文与社会科学学部部长洪晓楠教授分别致辞。中国自然辩证法研究会荣誉理事长范岱年教授、美国波士顿大学哲学系曹天予教授、德国达姆施达特技术大学哲学系AlfredNordmann教授、大连理工大学刘则渊教授以及中国社会科学院哲学研究所邱仁宗教授分别做了大会主题报告，展示了科学哲学领域国内外研究的最新成果。全国各省市高校的120多名代表参加了此次会议，本次会议讨论的内容涉及了科学哲学领域的多个方面，小组分会场每位代表报告15分钟，讨论提问5分钟，小组会场汇集老中青三代专家学者，学术探讨异常热烈。

  会议报告结束后，洪晓楠教授主持了大会的闭幕式。中国科学哲学学会理事长科学哲学专业委员会胡新和做了大会总结报告，高度赞扬本次盛会，号召要继承历届科学哲学学术会议优良的学术传统，并鼓励青年一代博士生踊跃发言和积极投入科学哲学研究，会上宣布拟定山东大学作为第十六届科学哲学学术会议的主办单位。最后，主办方诚邀各位代表参观考察了旅顺口和大连市区等地。

  清华大学科学技术与社会研究所的吴彤教授、曹南燕教授、刘兵教授、王巍副教授、蒋劲松副教授和博士生苏丽、张姝艳参加了此次学术会议。
