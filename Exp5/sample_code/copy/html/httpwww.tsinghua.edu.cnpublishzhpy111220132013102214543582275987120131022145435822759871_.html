美国工程院院士、普林斯顿大学教授林寿海到访我中心

9月11～12日，美国工程院院士、普林斯顿大学教授林寿海（Sau-Hai Lam）到访我校，向热能系捐赠奖学金，并到周培源应用数学中心进行学术交流。

11日下午，林寿海向清华大学捐赠奖学金仪式暨学术报告在热能系举行。该奖学金将纳入全校奖学金管理体系，用于奖励热能系在课外科技活动中表现突出的学生。校党委常务副书记陈旭出席仪式并致辞，对林寿海表示感谢，肯定该奖学金的设立对学校培养创新人才、多样化人才的推动作用及对建设世界一流大学的重要意义，希望学生们学习林寿海严谨为学、不断进取的精神。美国工程院院士、清华大学燃烧能源中心主任罗忠敬，热能系主任李政，系党委副书记刘树红、张建胜等师生共90余人参加捐赠仪式。捐赠仪式由校基金会副秘书长赵劲松主持。

12日上午，作为计算奇异摄动理论的创始人，林寿海与我校周培源应用数学研究中心部分师生就共同感兴趣的前沿科学问题进行深入讨论。他详细介绍了最近在该理论方面的重要进展，听取了周培源应用数学中心师生在生物化学反应机理简化和多组分扩散问题上最新研究成果的介绍。林寿海还结合自身研究经历，对如何运用数学方法去解决物理以及工程问题提出建议，并就传统工科大学应该如何发展应用数学谈了个人看法。

林寿海，1954年获伦斯勒理工学院（Rensselaer Polytechnic Institute）航空工程学士学位，1958年获普林斯顿大学博士学位，后在康奈尔大学任助理教授，1960年到普林斯顿大学任职，曾任该校机械与航天工程系主任6年，1999年退休。
