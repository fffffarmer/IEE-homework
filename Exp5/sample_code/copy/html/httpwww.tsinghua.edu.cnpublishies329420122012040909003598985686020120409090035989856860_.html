孟捷 

男，1967年生，籍贯山东，出生于安徽芜湖。

教育背景

先后毕业于复旦大学、中国人民大学和北京大学。经济学博士。

工作履历

1995年起在中国人民大学经济学院等单位工作，教授、政治经济学专业博士生导师。

2007年-2011年，兼任中国人民大学书报资料中心总编辑，全面负责人大《复印报刊资料》系列学术期刊的编辑工作。2011年起为清华大学经济所教授。先后在英国曼彻斯特大学、牛津大学、日本京都大学进行长期访问研究。

学术兼职

中国政治经济学年会总干事；

全国马克思主义经济学说史学会副会长；

中国《资本论》研究会副秘书长、常务理事；

中华外国经济学说研究会常务理事

研究领域

政治经济学，经济学方法论，经济思想史

奖励与荣誉

 2009年获教育部高等学校科学研究优秀成果奖三等奖；

2006年获北京市哲学社会科学优秀成果一等奖；

2005年入选教育部“新世纪优秀人才支持计划”；

2003年获“教育部优秀青年教师资助项目”的资助；

2002年被授予“霍英东教育基金会”第8届青年教师基金；

2001年入选“北京市培养新世纪（2001-2005）社科理论人才百人工程”

“经济人假设与马克思主义经济学”，《中国社会科学》2007年第1期。
