中国社会科学院金融研究所郭金龙教授来我院讲座

2018年4月26日下午，中国社会科学院金融研究所郭金龙教授应邀来我院经济所做客“陈岱孙理论经济学讲堂”，以“金融机制与家庭财富管理”为题发表演讲。高淑娟教授主持了讲座活动。

郭金龙教授长期从事金融与保险研究，曾主持完成多项国家级、省部级重大课题项目，对于我国当前的金融机制及家庭财富管理问题具有独到的见解。

郭金龙教授在两个小时的时间里，从金融资产及金融市场的种类和作用，到我国金融市场的形成及发展历史，进行了深入透彻的分析。同时，对大家比较关心的家庭财富管理及相关话题，进行了独到的讲解。他讲到，家庭财富管理，包括家庭资产管理、家庭负债管理以及家庭收支管理等三个方面的内容。目前，大多数家庭通常比较关注资产的管理而忽视负债管理，实际上负债的管理不但必要，在一定条件下甚至比资产管理更为重要。关于家庭财富管理的目的，他认为相对于财富增长来说，更为重要的是要平衡财务收支、保持家庭财务的流动性和安全性、实现资产和负债的合理匹配，在此基础上才能使家庭财富充分享受到国民经济成长的成果，尽可能实现财富的最大化。

最后，郭教授提出了关于家庭财富管理的建议，并就相关问题与在场的师生进行了热烈讨论。

此次讲座使在座师生获取更多、更实用的金融知识的同时，更加深入地理解了财富机制的性质与运行，并帮助大家理论结合实际，运用所学经济学理论分析社会现实问题，学习科学理财的思维认识与操作方法。

                                                                                                                                    （张意茗供稿）
