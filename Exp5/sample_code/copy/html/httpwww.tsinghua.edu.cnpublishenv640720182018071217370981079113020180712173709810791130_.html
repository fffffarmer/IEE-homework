清华大学环境学院温宗国教授课题组招聘信息

因科研工作需要，清华大学环境学院温宗国教授课题组招聘博士后2名，科研助理2名，行政秘书1名。岗位要求及待遇如下。

岗位1：博士后2名

一、研究方向与合作导师：

方向1：环境管理、规划与政策分析，环境信息化与大数据等

方向2：物联网装备与系统开发、废物资源化

合作导师：清华大学环境学院，温宗国教授，博士生导师（个人主页：http://www.tsinghua.edu.cn/publish/env/6422/2011/20110219234745558329580/20110219234745558329580_.html）

二、申请条件和要求：

1、已取得博士学位，或近期能顺利答辩获得博士学位者，年龄不超过35岁（申请有关人才计划者不超过31岁）；

2、环境经济、能源经济以及环境规划管理、环境系统分析等相关学科或专业背景；

3、热爱科研工作，具有较强的系统分析思维和数据建模能力，具备独立从事科研及项目工作的能力，团队合作和组织协调能力突出；

4、有环境管理和政策分析、循环经济或节能减排等领域项目经验，或者从事过环境信息化、环境物联网装备与系统开发、环境大数据挖掘技术研发等工作；

5、发表SCI收录的学术论文1篇以上或有著作出版者；

6、具有较强的中英文写作能力和口头表达能力。

三、工作待遇：

1、国内博士后可以申请“创新人才支持计划”经费资助50万元；“优秀博士毕业生”经费资助30万元；

2、海外优秀博士后可以申请“博士后支持计划（海外）”项目，经学校统一评选，可分别获得60万元、40万元、30万元的资助；

3、工资及福利待遇优厚，按照国家和清华大学博士后相关管理办法的有关规定执行（可申请住房），根据研究任务合作导师还提供额外科研补贴。

岗位2：项目研究员2名以上

一、岗位职责：

协助开展节能减排和循环经济相关课题研究、实施和管理工作，协助完成课题申报、参与项目报告撰写等。

二、招聘要求：

1、具有环境规划与管理、环境系统分析、环境经济等相关专业背景，获得硕士及以上学位；

2、熟悉节能减排或再生资源行业，有科研项目或规划编制等项目经验；

3、参与过国家项目报告书的编写、研究、科研会议组织和管理工作；

4、发表过核心或SCI学术论文、组织协调能力突出、团队合作强者。

三、工作待遇：

按照清华大学合同制人员管理相关规定，办理五险一金，具体薪酬面谈。

岗位3：行政秘书1名

一、岗位职责：

参加行政领导会议，负责研究中心行政事务的上传下达，参与中心重大科研项目的调研工作，拟定调研报告，及时掌握研究中心主要工作的进展情况。

1、 负责研究中心行政事务的上传下达，参加组织行政领导会议，参与中心重大科研项目的调研工作，协助开展项目申请、项目验收等办公室事务。

2、协助开展中心承办的培训班、研讨会、学术交流活动以及中心公开授课的教务工作；

3、协助开展中心专著的统稿、出版工作;

4、领导交办的其他行政事务、日常报销管理等。

二、招聘要求：

1、本科及以上学历，行政秘书相关专业，或有生态环境相关学科背景，工作经验2年以上；

2、具有较强的文字撰写能力，擅于公文写作;

3、具有良好的外联、公关能力;

4、要求有一定的外语写作和交流能力。、

三、工作待遇：

按照清华大学合同制人员管理相关规定，办理五险一金，具体薪酬面谈。

此招聘长期有效，有意应聘者请将简历及代表学术论文发至hrtsinghua@163.com，并按照“姓名-专业-应聘岗位”写明邮件主题，初选合格者通知安排面试。
