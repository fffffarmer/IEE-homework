校园看点“当迪士尼遇上百老汇”——美国钢琴家兰斯·怀斯曼独奏音乐会

 

演出信息

时间：2018.9.21(周五) 19:00
地点：蒙民伟音乐厅

艺术家简介

兰斯·怀斯曼（Lance Wiseman）以他对音乐作品独特的清澈诠释及对乐谱细微变化的精心关注而闻名。他的钢琴演奏特色在于浓厚的音色、自然的分节及充满诗意的表达。他的演奏，从巴洛克风格到现代音乐，能够取悦于最挑剔的观众。

作为演奏家以及与专业乐团合作的独奏钢琴家，怀斯曼在欧洲和美国都获得了高度的赞誉。他与希腊萨洛尼卡国家交响乐团合作表演之后，马其顿报刊评论：“他的表演预示他将为国际舞台上的演奏名家。” The Voice （雅典）评论言“他真正领悟到如何意译伟大作曲家隐藏的宝藏。”

入场方式及售票信息详见清华大学艺术教育中心演出官网 www.hall.tsinghua.edu.cn
