环境学院学生会各部门简介

学生会文艺组

学生会文艺组通过开展“绿缘”系列文艺活动为我系同学提供了多个向系内外展示自己文艺才能的舞台，搭建系内同学感情沟通，与兄弟院系互动交流的桥梁与纽带。

9月末的“新生舞会”将给你大显身手的机会，让你消除刚入学的陌生感，结识、熟悉更多系内外的新老同学。第二学期，文艺组将为全系热爱唱歌的同学们量身打造“卡拉OK大赛”，让大家一展歌喉，突破自我，大家千万不要错过哦！

每年我系的文艺重头戏——学生节晚会，则是全年最值得期待的文艺盛宴。学生节整台晚会的节目都由我系老师和各班同学自编、自导、自演，将是一个向全系乃至全校展示个人风采、班级风格的大舞台。

学生会体育组

体育的魅力无处不在，生活的激情无时不有！体育源于生活，生活离不开体育。体育组就是一个既为大家提供生活服务，又为大家创造体育参与环境的部门。

在高度重视体育的清华，体育永远是你大学生活不可或缺的点缀。“至少为祖国健康工作五十年”的口号号召下的，清华的体育活动广泛的开展，男女生们每年必跑的3000米，1500米，贯彻一年的“马约翰杯”运动会，系内各种联赛都和同学们的生活融合在一起。

“马约翰杯”运动会是我校内体育比赛的总称，其中的项目小到棋牌、大至各种球类，涉及非常广泛，可谓是应有尽有。我系有非常优秀的体育传统与体育氛围，通过师生们的共同努力，曾获数次“马约翰杯”运动会总冠军。

每年十月，学校将会面对新生举办独一无二的“赤足运动会”，以班级为单位，大部分项目需要光脚参加，非常特别~

此外，系内的体育活动也丰富多彩，一年一度的系内足球赛、篮球赛、排球赛、系运动会、羽毛球联赛或者台球赛等，都是各位一展身手的好机会！

体育组愿意为新生们竭诚服务，为同学们提供更多的体育资源，营造最好的体育氛围，让体育融入同学们的生活，让体育丰富同学们的生活！

学生会生活权益组

生活权益组是与学生日常生活最紧密相关的小团队。它会带你走过女生节、男生节的唯美风景，为你解决生活中的点滴困难，让你在这里找到家一样的感觉。

每年的3月7日，是清华的女生节。这似乎是一个因三八妇女节而产生的节日，女生们不愿称自己为妇女，而女性的节日特权又不想放弃。因为女生们更年轻、更有活力，所以就将妇女节的前一天定为了“女生节”。这一天，是属于所有女生的浪漫节日。一早起来，你也许就会收到男生们送来的热腾腾的早餐；桌子上，也许就放着男生们送来的礼物、祝福卡；走廊里，也许早就被男生用彩带、千纸鹤装扮一新；校园中，到处飘扬着“祝XX女生节日快乐”“XX系女生最美丽”的条幅……只有你想不到的，没有男生们做不到的，总之，惊喜无处不在。这一天，女生们都是最幸福的“公主”。

每年的11月12日则是清华的男生节，同样在这一天男生也能够得到所有清华女生的礼物与祝福。

另外，在这两个节日当天，学生会生活权益组也会组织各式各样的活动，比如奶油蛋糕制作、电影晚会等等，让你享受到尽兴！

学生会宣传组

怕自己忙于应付新的环境而错过了美妙的新生舞会？怕自己埋头自习而忘记了学生节晚会的重要日子？怕自己没收到消息而与精彩的体育比赛擦肩而过？别担心——只要每次经过宿舍张贴栏时留意着 标志，活动信息、赛事赛果，宣传组第一时间告诉你！

已经被以上学生会各部门介绍的丰富多彩的活动弄得眼花缭乱了吧？敬请留意宣传品——抢眼绚丽的海报能告知你活动的详细信息，特色马杯文化衫会在赛前发到你手中，限量版学生节书签门票超有纪念价值，还有可爱的吉祥物“绿绿”、“缘缘”与我们一同演绎你我他的故事…~

学生会内联组

就如同每次走过桥前往彼岸，我们总会感谢架桥的人；即使我们互不相识，然而由于内联的存在，我们能及时了解你真实的需求与恳切的建议。是的，这就是内联，她在同学与学生会之间，学生会各部门之间，你和我之间架起了一座沟通的桥梁。交流、协调与共享是对内联工作最好的概括。我们促进系会各部门的交流（主席团例会、经验分享、学生会工作、日常联系工作），提升总体工作效率；收集并整理历届学生会活动资料，与大家分享宝贵经验；与各班联系人保持联系，及时了解大家的想法和建议。

我们耐心、细心、真心地与大家交流，我们渴望倾听你的心声！

学生会外联组

学生会外联组致力于通过校内、校外资源的联络与沟通，获得系内学生活动的部分经费、物资支持，以保证活动所需。

“学生节”晚会时的入场观众礼包中，精美的纪念品、学校周边优惠券；男女生节时的节日大礼包中，各式优惠、折价券，为大家带来应得的节日福利，让同学们亲身感受到节日的欢乐。这一切，都源自我们学生会外联组背后默默地工作。希望能够通过我们的努力，为大家的节日、为我系学生活动带来更多乐趣！
