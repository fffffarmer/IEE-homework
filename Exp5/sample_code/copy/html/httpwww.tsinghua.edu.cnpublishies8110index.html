清华大学华商研究中心

    国际华商纵横驰骋，促进了中国经济、亚太经济与全球经济的发展。华侨华人在中国改革开放中长期充当了先行者的角色，民营企业日益成为中国经济社会发展的原动力。观念更新与思想解放，制度创新与体制变革，都浸透着海内外华商的影响与推动。国际华商经营管理兼融东方理念与西方思想于一体，并在实践中摸索与创新，在东西方文化交融中促进了当代管理学的发展。“达则兼济天下”，成功的华商领袖体现出传统儒商的胸襟和社会责任的承担。
    在国务院侨办的大力支持和合作建设下，清华大学人文社科学院、经管学院、公管学院共同组建了校级跨学科研究机构华商研究中心。中心为企业家、学者、政府官员提供三方交流的高层平台，相互砥砺与激发，从华商探索与实践的鲜活案例中提炼企业经营思想的灵感与火花，推动原创性的经济学与管理学研究成果。
    华商研究中心的建立，将秉承“学习华商、研究华商、服务华商”的宗旨，立足学术前沿，拓展国际视野，依托清华大学经济学、管理学、国际关系等学科优势，致力于全球华商及其领导艺术、跨区域投资与贸易行为、经营理念、管理经验和方法智慧等学术领域的研究。
华商与中国经济高速列车一起奔驰，清华大学华商研究中心也将与之共同成长，力争在国际学术界中独树一帜。
    中心主要工作如下：
    (1)“华商领袖•清华讲堂”，国务院侨办与清华大学联合主办的高层论坛。中心还设有华商研究中心讲座等交流平台。 
    (2)主持国家社会科学基金重大项目、国家自然科学基金海外合作项目等重要项目；主编Global Migration and China英文丛书、社会经济史译丛、国际华商研究书系等。
    (3)承担国务院部委、省市政府部门的委托与重点项目。“中国侨资企业数据库”的开发与应用、《中国侨资企业发展年度报告》等形成富有影响的系列成果。
    (4)王永庆管理思想研究、台商研究等原创性成果富有特色。
    (5)承接侨商、台商、港商、民企、国企的捐赠、委托项目、企业咨询或人员培训,并提供政、商、学高层交流平台，提供咨询服务与交流平台。
    (6) 中心设有欧洲分部、北美分部，充分发挥当地学者、侨领和华商的力量，探讨异军突起的欧洲华商与日益壮大的北美华商。
    开展高水平的国际交流，与耶鲁大学、布朗大学、鲁汶大学、怀雅逊大学等开展实质性合作研究项目。
