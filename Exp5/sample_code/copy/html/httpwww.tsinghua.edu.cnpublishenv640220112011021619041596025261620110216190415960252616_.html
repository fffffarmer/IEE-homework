银行绿色信贷管理高级培训班

与中国其他行业一样，环境作为一种基本要素开始导入传统金融领域，并将成为未来银行业规避风险和获取持续性增长的新手段和新引擎。从环境的角度去规划和重新设计业务流程是我国新时期所有行业必须遵循的原则和趋势，尤其是银行业这样一个日益国际化和社会化的行业，必须更加认真地应对世界潮流和客户意识的提升。而且随着国家环境政策的整体性改进和执法的日趋严格，没有考虑规避环境风险的银行迟早要承受损失，这种前瞻和意识已经并不超前。

清华大学环境管理与政策研究所是国内领先开展绿色信贷政策研究和业务流程再造设计的单位，在政策研究和业务设计的基础上，现创办“银行绿色信贷管理高级培训班”，旨在帮助银行建立起“绿色信贷”环境风险控制体系，培养出第一批熟悉国家环境政策、产业政策、节能减排规划，中国特色的企业和项目环境管理程序、绿色产业发展前景，对银行环境管理能力建设有系统了解的银行环境风险管理高级人才。

课程对象

1、全国性银行和地方商业银行等金融机构高层管理人员；

2、境外金融机构驻中国高层管理人员；

3、大型企业负责人及企业内部投融资业务负责人

课程概述

目前，“绿色信贷”政策的实施对银行建立环境管理能力体系提出了迫切要求，本课程从环境与金融的基础知识着手，介绍国家的环境管理政策、产业调整政策、环保产业的发展趋势，并从专业角度重点讲解企业及高污染行业的环境风险及其防范策略，最后运用以上知识，提出银行针对项目、企业两种不同的环境风险管理流程及操作方法。通过专家详细而深入的讲解，本课程为银行业加强自身环境管理能力，建立环境风险控制体系提供了专业知识和操作指南，从而进一步推动“绿色信贷”在国内的有效实施。

学习认证

完成规定的课程并考核合格者可获得由清华大学颁发的“银行绿色信贷管理高级培训班”结业证书，加盖清华大学钢印和公章，可在清华网站查询和认证

课程体系

模块1：环境与金融

    介绍环境与金融的发展历史沿革、讲解环境问题对金融业的影响、解析环境与金融的相关概念、介绍国际动态与实践。

模块2：中国的环境政策体系与绿色信贷政策实施

    讲解中国环境现状与问题、中国环境管理与政策体系，全面解读绿色信贷政策。

模块3：中国近期产业结构调整政策

    解析控制类产业政策和激励类产业政策，介绍产业政策查询系统。

模块4：环保产业发展趋势及重点领域分析

    讲解中国环保产业发展现状与总体趋势。重点介绍城市环境基础设施建设与运营、节能减排市场、CDM项目的运行与投资、可再生能源市场风险与前景等环保产业重点领域。

模块5：绿色信贷中的环境信息获取及企业环境信息分析

    解读我国环境信息公开制度，并讲解我国企业环境信息分析方法。

模块6：我国高污染行业绿色信贷环境风险分析

    选取并分析重点环境风险行业，分析重点行业主要环境风险。

模块7：银行绿色信贷环境风险管理操作系统

    针对贷款企业，讲解银行环境风险管理体系以及风险控制流程与方法。

模块8：银行绿色信贷环境风险管理操作系统

    针对项目贷款，讲解银行环境风险管理体系以及风险控制流程与方法。
