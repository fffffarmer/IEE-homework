2018年中法联合培养环境管理高级硕士项目招生计划

根据法国教育部和清华大学关于清华中法环境与能源中心的框架协议，以及清华大学与巴黎矿校(ENSMP)、里昂国立应用科学研究院(INSA de Lyon)之间所签署的校际合作协议，三校拟合作开展环境领域的高级硕士学位培养项目。

高级硕士(Mastère Spécialisé)是由法国大学校委员会(Conférence de Grandes Ecoles)批准认可的一种高级专业培养方案，在法国主要面向具有Bac+5学历（相当于欧洲体系的硕士研究生学历）的学生或具有实践经验的业界骨干力量，旨在培养具有专业技术背景且能够解决企业实际管理问题的高级专门人才。

培养目标    

针对环境或相关专业领域的高级管理需求，培养既有坚实的理工专业基础，又掌握现代管理知识，从而具备分析和解决实际管理问题的能力，并具有创新思想、战略眼光和国际视野的复合型人才，以进一步提高其综合素质和国际竞争能力。

未来就业主要面向大中型企业的管理部门，从事环境/能源领域的高级管理或相关研究工作。

学习期限     

从2018年9月开始，学习期限总共为14个月，其具体安排为：14个月学习包括8个月的课程学习和6个月的实习，8个月的课程学习中包括4个月在中国学习，4个月在法国学习。

培养特点     

针对已具有专业技术教育背景的学生或来自企业的技术骨干，首先分别在中国和法国进行为期8个月的集中课程学习。教学计划按模块化的形式进行组织，其中包括相关的理论课程、案例研究、交互研讨、情景角色、团队项目和实地考察等内容。具体内容将会涉及到基础理论、分析方法和工具以及项目案例等几个有关方面。

课程学习结束后，将在企业进行为期6个月的专业实习，在来自企业和学校的专业教师的联合指导下，完成法国高级硕士学位论文的撰写和答辩工作。同时还要按照清华大学硕士学位相应培养方案的具体要求，完成清华大学硕士学位论文的相关研究和撰写工作。

学位授予

参加本项目的我校同学应为来自环境或相关专业的在读硕士研究生，学习期满后，在同时满足清华大学硕士和该项目培养方案要求的前提下，将分别授予清华大学颁发的硕士学位和法国大学校委员会及巴黎矿校颁发的高级硕士(Mastère Spécialisé)学位。

费用要求     

法国高级硕士培养为收费教育项目，学费平均约为1.2万欧元，人均每月生活费用约为600欧元，往返法国交通费用约为700欧元。但对于优秀的入选者，可以豁免学费。同时对生活费用部分，法方予以奖学金的资助。

申请条件 

我校与环境领域相关专业在读的一、二年级硕士研究生；

学习成绩优秀，具有开拓和进取精神；

具有从事环境管理工作的意愿、能力或潜质；

获得指导教师的同意及推荐；

具有较高的英语交流和实用水平；

愿意服从整体学位培养方案的工作计划及时间安排。
