历年年鉴


为及时反映学校办学、改革、发展的轨迹，从1997年10月开始，党史校史研究室每年编辑一卷《上海交通大学年鉴》。《年鉴》是汇集截止出版年为止的较全面、系统地记述学校各领域、各部门的情况，并附重要规章、文件选目和统计资料等的综合性参考工具书。
