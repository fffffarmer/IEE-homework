一、设立宗旨

为感谢钱易院士在我国环境科研和教育领域的杰出贡献，她的学生们在她80寿辰之际，发起并捐资成立了“钱易环境教育基金”，并在该基金支持下，设立了“钱易环境奖”，旨在激励全国积极践行环境公益、脚踏实地开展创新研究的优秀学生，激励青年一代投身环境保护事业，为国家可持续发展与生态文明建设努力奋斗。

二、申请人资格

1. 中国各中学、大专院校、科研院所的全日制在校生。（包括中学生、本科生、硕士生、博士生，专业不限）；

2. 思想健康，品行端正；

3. 热爱环境事业，具有创新精神；积极组织并践行环境公益活动，并取得突出贡献；脚踏实地地开展创新性研究，并取得显著创新性研究成果；

4.  原则上两年内获得过“钱易环境奖”的同学不得再次申报。

5.  原则上两年内获得过“唐孝炎奖”、“高廷耀奖”的同学也不得再次申报。

三、奖项设立

特等奖1名，奖金5万元人民币；

一等奖5-10名，奖金2万元人民币/每位获奖人；

二等奖10-20名，奖金1万元人民币/每位获奖人；

中学生奖5-10名，奖金6000元人民币/每位获奖人。

四、申请与评审办法

（一）申请流程

1. 个人申请

（1）申请人填写《清华大学钱易环境奖申请表》2018创新奖申请表.doc2018公益奖申 请表-高中.doc2018公益奖申请表.doc2018管理奖申请表.doc，表格下载地址：http://www.tsinghua.edu.cn/publish/env/10921/index.html（请注意选择奖项类型），包括个人基本信息、获奖、论文发表和参与活动；

（2）获奖证明复印件、参与社会活动中申请人所做贡献排名证明材料（如有多个参与者）、国际交流的相关证明材料复印件（所有证明材料均加盖教务公章）；

（3）具有创新性的作品（发表或未发表）；

（4）申请人应在申请表中如实、客观的描述自己的研究成果，评审之后获奖候选人的申请表将在原单位公示；

（5）申请人准备一页PPT（姓名+单位+主要研究成果）以便终审之后评审委员会发布。

说明：申请材料中，关于获奖、参与社会活动和国际交流等活动，需如实描述自己在其中的作用与贡献； 关于具有创新性的作品（发表或未发表），应未用于申报其它类似奖项。

 

2. 推荐人推荐

申请人需要提供两位推荐老师的推荐信（需推荐人亲笔签名，并写明有效联系方式，以便审核）。

3. 申请材料递交

（1）全部材料以电子版（或扫描件）形式作为附件（附件命名为：学校-姓名-申报方向）发送至邮箱qya@tsinghua.edu.cn。

（2）申请材料原件邮寄地址：北京市海淀区清华园1号， 100084，

清华大学环境节能楼731#， 电话：010-62780050，张凯琴老师收。

（注：邮寄纸质材料之后，在电子邮件里注明快递单号，以免出现文件丢失情况）

（3）提交截止时间：2018年 9月30日17:00之前。

（二）评审程序

1、评定：钱易环境奖评审委员会对所有申请人进行评定，确定获奖者名单，特等奖获得者将由一等奖候选人通过现场答辩评比产生；

2、公示：获奖名单在清华大学环境学院网站及微信公众号公示一周，同时贴出获奖同学的公示材料；

3、确认：钱易环境奖评审委员会将确认获奖者名单，并通知学生本人。

五、颁奖

颁奖仪式定于2018年12月，在清华大学举行，颁发奖学金和荣誉证书。具体时间地点另行通知。
