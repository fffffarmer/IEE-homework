经济所多位同学的论文在“中国马克思主义政治经济学青年论坛”上获奖

近日，经济所四位同学的论文在“第二届中国马克思主义政治经济学青年论坛”上获奖。四年级直博生孙小雨同学的论文“构建劳动价值论的数量分析框架”荣获一等奖第一名；三年级本科生刘充同学的论文“固定资本、收入分配与中国经济的微观效率：一个经验研究”荣获二等奖第一名；高晨曦是经济学所2016届本科毕业生，现在日本一桥大学师从渡边雅男学习政治经济学，他的论文“资本主义下的服务业：实物与劳动力租赁”获得三等奖；崔震是经济学所2017届本科毕业生，现在苏黎世联邦理工学院读研，他的论文“价格理论的张量分析”获得入围奖。

“中国马克思主义政治经济学青年论坛”由中国人民大学经济学院主办，拟于2017年12月在北京举办第二届论坛。自2017年6月征稿以来，面向海内外共征集127篇投稿作品。论坛评审委员会评选出一等奖作品3篇，二等奖作品5篇，三等奖作品11篇，入围奖作品35篇。

公示信息见：http://econ.ruc.edu.cn/displaynews.php?id=14305

“中国马克思主义政治经济学青年论坛”于2016年12月在北京举办首次论坛。在首次论坛中，经济学所三年级直博生孙小雨同学的论文 “关于刘易斯和布伦纳两种农业劳动力转移理论的比较——兼评关于中国经济转型的争论”荣获二等奖第一名（一等奖空缺）；二年级硕士生杨青梅同学的论文“利润率趋向下降规律的理论与争论：一个批判性综述”荣获三等奖；高晨曦是经济学所2016届本科毕业生，现在日本一桥大学师从渡边雅男学习政治经济学，他的论文“资本对消费过程的统治：消费资本论”获得三等奖。
