2017年本所硕士招生信息

 

专业方向一：
010100 哲学
01（全日制）科学技术哲学

考试科目：
①101 思想政治理论
②201 英语一 或 202俄语 或 203日语
③302 数学二 或 652 社会科学方法
④906科学技术史

备注：本专业分为科技哲学、科技社会学、科技传播和科技政策四个研究方向，承认我校任何理工专业的考试；复试时专业综合考试内容：科学技术概论。

 

专业方向二：
030300 社会学
02（全日制）应用社会学（科学技术社会学）
考试科目：
①101 思想政治理论
②201 英语一 或 202俄语 或 203日语
③302 数学二 或 652 社会科学方法
④906科学技术史

备注：复试时专业综合考试内容：科学技术概论。

 

专业方向三：
071200 科学技术史
01（全日制）科学技术史

考试科目：
①101 思想政治理论
②201 英语一 或 202俄语 或 203日语
③302 数学二 或 652 社会科学方法
④906科学技术史

备注：复试时专业综合考试内容：科学技术概论。
