121108 机械力对整合素构象和激活的调制作用

报告人: 娄继忠 研究员

报告时间: 11月8日，周四，下午3点

报告地点: 清华大学科学馆213会议室

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

摘要:    细胞与胞外基质的相互作用是包括细胞分化，细胞迁移，胚胎发育以及组织形成等许多生物过程的基础，其缺陷会引起多种疾病。这一相互作用通常是通过整合素家族蛋白介导的，而整合素蛋白又通过Talin等膜内蛋白与细胞肌动蛋白骨架相连接并构成相关化学和力学信号传导的一部分。研究表明整合素存在不同的构象，而不同构象下配体结合能力也不一致。通过生物膜力学探针测量，我们实时观察到活细胞表面LFA-1整合素的构象变化过程。同时结合分子动力学模拟和理论分析，我们得到了整合素/配体相互作用的动力学性质，明确了机械力对整合素激活过程的调制作用。
