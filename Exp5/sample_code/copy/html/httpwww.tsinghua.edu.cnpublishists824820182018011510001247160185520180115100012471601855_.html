科技所2018新春联欢会举行 胡显章等老领导出席

2018年1月12日，科技与社会所广大师生济济一堂，共庆2018新年的到来。

原校党委副书记胡显章教授回顾了他与科技所的点点滴滴。胡老师表示，当年他是分管文科的副书记，后来成为第一任人文社会科学学院的院长，通过对国际同行的调查，他认定STS是新兴交叉学科，是清华应该大力发展的新文科标志学科，更是践行清华文理会通传统的代表学科。吴彤老师补充说，胡老师曾大力帮助我所争取博士点，但博士点批准时胡老师因超龄原因无法担当博士生导师。为此，前所长曾国屏教授曾经多次打报告争取他的博导资格，都被胡老师谢绝。胡老师清正廉洁的作风品质，堪称圭臬。

创所所长魏宏森教授也回顾了科技与社会所的历史。魏老师表示，今年是科技与社会所建立40周年，科技所也已经由当初的两三个人的队伍发展到今天近百人的规模。有着这样光辉的起点，科技所师生更应不忘初心，不辱使命，为清华新百年的发展贡献力量。

老教师丁厚德教授则补充了一段科技所名称由来的历史。丁老师说，我们所的名字是老校长高景德给起的，充分体现了当初学校领导的魄力果敢和高瞻远瞩。现任所长杨舰则带领大家重温了科技所一年来所取得的成就——包括新开通识课程，科研成果、科研经费的增长，特别是顾全学校大局为各相关学科评估所做出的无私奉献。
