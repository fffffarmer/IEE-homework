经济所“2018年全国优秀大学生夏令营”成功举行

2018年7月10日至7月13日，清华大学社科学院经济学研究所组织了“2018年全国优秀大学生夏令营”活动。本次活动共收到来自全国各高校80余名学生寄来的报名材料。经过初步资料审查筛选，确定了来自清华大学、北京大学、北京航空航天大学、中央财经大学、对外经贸大学、西南财经大学、中国农业大学等7所高校，本科前三年总评成绩排名在本校同年级本专业前10%的18位优秀大学生前来参加夏令营。

这四天日程安排充实、丰富，营员们经过专题研究、学术报告、学术沙龙、面试笔试、座谈交流等环节与经济学研究所的教授们进行面对面交流，通过个人展示、互动讨论等形式最终完成选拔工作。经济所老师在政治经济学、经济史、计量经济、宏观经济、数字经济等五个方面为参营大学生做了生动的专题演讲。学生与主讲老师一起探讨学术问题，并与我所优秀在校博士及毕业的优秀校友交流清华的学习生活及职业发展规划。

最后，他们完成了笔试和面试环节。四天紧张而充实的夏令营不仅让他们熟悉清华园、社科学院和经济所，更为他们提供了一个舞台，展示他们的能力才华和为实现自己梦想而拼搏的努力。同时，在这一过程中，他们也与清华老师学生结下深厚友谊。
