巴塞尔公约不限成员名额工作组第十一次会议第二天

清华大学环境学院2018年9月13日电   瑞士时间9月4日，巴塞尔公约不限成员名额工作组第十一次会议进入第二天，会议形式和主要内容开始由单一的主会场会议逐渐转移至接触组的商讨，讨论也更加具体和激烈。国际班五位同学根据分工和兴趣，有计划的参与和跟进了多个会场的会议进展。

 本日，各国代表的核心关切和会议要点集中在了海洋塑料垃圾和微塑料这一议题上，各国就巴塞尔公约能否在这一问题上担当中心地位，如何建立相应的伙伴关系的问题展开了长时间的讨论。由于时间限制，各国代表在有关海洋垃圾接触组中只是概括性的发表了对于11/7号文件的建议和立场，但即便如此，这一部分的会议仍持续了过三个小时。在这其中仍能看到不同国家对于同一问题的不同观察角度，如喀麦隆代表非洲国家，一方面承认了海洋垃圾问题的严峻性，又反复强调陆地塑料垃圾对于非洲国家的严重影响，希望文件中也着重提及这一方面的内容。

在此之外，会议还对技术和战略框架这两大问题开展了接触组讨论，晚间讨论了加拿大和智利提交的有关公约法律执行和遵约的CRP文件。

值得一提的是，4号中午，大会主办方于万国宫前举办了海洋垃圾展（Marine Litter Exhibition）这一边会。这一方面体现了本次大会聚焦海洋垃圾问题的特点，另一方面也展示了全球环境爱好者越来越多的关注海洋微塑料问题的趋势。

根据全球环境国际班的培养方案，所有同学在本科期间都有机会参加由国家级专家带队并指导的全球公约大会。通过参加国际会议，同学们开阔了视野，实现理论学习、方法训练和职业素养的无缝对接，提高参与国际发展与全球环境问题治理的综合素养与实践能力。这种亲身参加国际大会的经历，让国际班同学们获益匪浅，感受到了作为全球环境人的任重而道远。
