诺贝尔物理学奖获得者小林诚教授在我校演讲

                 

10月21日，2008年诺贝尔物理学奖获得者小林诚教授（Makoto Kobayashi）访问我校。本次访问是应人文社科学院科技与社会研究所的邀请，在清华海外名师讲堂，面向全校师生作题为《物质与反物质对称性的破坏》的公开演讲。该演讲作为“与大师对话——诺贝尔奖获得者中国校园行”系列活动之一，得到了国家外专局的支持，同时也被列入我校百年校庆百场学术活动中。来自我校人文社科学院、工物系、物理系以及周边院校的150多名同学和老师参加了报告会。人文社科学院王巍副教授主持了报告会，会上小林城教授还与现场学生们就演讲内容进行了对话。

王巍副教授

小林诚教授

小林诚教授演讲前，副校长谢维和在丙所会见小林诚，对其来访表示欢迎，并简要介绍了我校相关学科的情况，尤其谈到了文理科交叉及科学技术哲学和科学技术史这样的交叉学科在培养创新人才培养方面的地位和作用。双方还就共同关心的中日教育问题进行了探讨和交流。作为本次活动的组成部分，小林诚教授还同物理系和工物系师生，围绕着最近相关领域前沿的最新进展进行了交流。

小林诚1944年出生于爱知县名古屋市，研究方向为基本粒子物理理论，并以卡比博－小林－益川矩阵为高能物理学界所熟知。现就职于日本筑波高能加速器研究社，现为高能加速器研究机构名誉教授、独立行政法人日本学术振兴会理事及财团法人国际高等研究所研究员。2008年，小林诚和益川敏英由于共同发现了“CP对称性破缺的起源”而获得了该年度的诺贝尔物理学奖。
