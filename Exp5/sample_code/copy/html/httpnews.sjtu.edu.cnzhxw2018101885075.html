上海交大牵头陈贵海教授负责的国家重点研发计划项目启动会召开

2018年10月17日 责任编辑：洪纳娜 郑茂

2018年10月11日，以上海交通大学为项目牵头单位、电子信息与电气工程学院陈贵海教授为项目负责人的2018年度国家重点研发计划“云计算和大数据”重点专项“基于立体精准画像的学术分类与推荐系统”项目启动会暨实施方案论证会在电子信息与电气工程学院召开。

科技部高技术中心信息处处长傅耀威、科技部高技术中心信息处项目主管贾燕红和张军、上海交通大学电子信息与电气工程学院院长关新平，各有关主管部门负责人、项目专家组专家、项目各课题负责人及学术骨干成员等30多人参加了会议。

项目启动会上，关新平首先致辞，表达了对本项重点研发项目的高度期待，希望各位专家提出宝贵意见，让项目能够做得更好，也希望大家一如既往支持交大发展。科技部傅耀威处长指出项目考察关键点，并就项目实施提出建议，希望项目组成员和学校积极沟通获得支持，做好任务分解与进度安排，落实法人责任制。

陈贵海教授代表项目组介绍了项目的立项背景、主要研究内容、考核指标和预期成果、课题设置、进度安排以及实施方案等，同时针对每个子课题，分别从课题目标、拟解决的关键科学问题、研究内容、技术路线、年度计划、工作基础及预期成果等方面汇报了各课题的工作计划。与会专家高度评价了本项目，表示项目立项必要性强，创新性突出，研发意义重大，同时也提出了必须要能够与应用结合来做出实际系统，细化评价指标，加强项目进度管理，加强课题间的交流与协同等具体要求。

该项目计划研究基于立体精准画像的学术分类与推荐系统，以解决当前人工挑选评审专家耗时费力且很难公平公正的问题。针对学者人数众多、数据庞杂对遴选评审专家的挑战，研究并实现科研行为数据库（数据）、多维学术画像（画像）、精准推荐（推荐）、可信溯源与隐私保护（安全）等一系列前沿关键技术及系统，并在科教领域进行示范应用，形成相关规范和标准，为遴选专家的“高效化、精准化、可信化” 提供技术支撑。项目将突破目前学术网络构建和专家推荐体系中的关键难题，培养一批大数据挖掘领域高精尖人才和创新团队。

针对专家组的意见和建议，各课题负责人与课题骨干成员进行了充分的讨论，在课题协调、项目进度、考核指标等方面达成共识，为项目的实施与推进奠定了坚实基础。
