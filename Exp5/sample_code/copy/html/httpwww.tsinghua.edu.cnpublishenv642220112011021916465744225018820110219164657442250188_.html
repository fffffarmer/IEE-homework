研究领域

主要从事环境污染控制化学与纳米材料学的研究，涉及到水和废水、空气的污染化学及其控制；此外，还负责管理清华大学环境质量检测中心的环境监测及相关咨询，具体包括：

1.室内空气污染化学及净化技术

（1）室内空气污染化学：研究室内空气污染物及嗅味物质的来源、释放规律、在室内介质中的分配特性，一次污染物在室内的反应以及二次污染物的生成机制

（2）空气中挥发性有机物/嗅味的去除技术：如真空紫外光催化、吸附-热脱附/催化分解、可见光催化臭氧分解、生物过滤等

（3）空气中无机污染物的去除技术：研究开发常温下高效分解臭氧的催化材料、复合型光催化材料

（4）研究开发针对不同室内环境如医院、家居、实验室、游泳馆、密闭空间的室内空气净化器

2.水污染控制化学与材料

（1）高级氧化水处理技术：研究多相光催化氧化、催化臭氧氧化、Fenton试剂法、电化学氧化及其相应的高效长寿命催化材料

（2）污染物的化学转化：研究调控水中有机污染物水解、还原、光化学降解及其中间产物（副产物）的生成

（3）水和废水深度处理集成工艺：综合运用化学转化、絮凝、吸附与生物处理达到水和废水的深度处理，优化工艺参数

3. 新兴污染物的转化与降解技术

研究全氟辛酸、全氟辛磺酸、内分泌干扰物、消毒副产物、药物和个人护理品的转化与降解技术

4. 环境质量检测服务及咨询

开展大气和废气、室内空气、水和废水、土壤、固废等方面的监测；文物保存环境监测；环境技术验证相关的检测
