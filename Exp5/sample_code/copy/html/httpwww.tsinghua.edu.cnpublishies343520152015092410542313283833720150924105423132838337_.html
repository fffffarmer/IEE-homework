社科学院经济所与剑桥大学土地经济系举行学术交流活动

2015年9月20日上午，英国剑桥大学土地经济系师生应社科学院经济所邀请，对经济所进行了学术访问，并在新斋105会议室开展了学术交流会议。

到访的嘉宾有土地经济系教师Franz Fuerst博士、Thies Lindenthal博士、Helen Bao博士及八位在读博士研究生。我所副所长王勇副教授、龙登高教授、汤珂教授及数位博士生代表与其他学院学生出席交流。会议主持人汤珂教授向来宾表示了热烈的欢迎，剑桥友人对我所的热情接待表示荣幸与感谢。

首先，Franz Fuerst博士从机构设置、培养模式、研究领域、核心任务等角度展示了土地经济系概况。该系建立于1962年，研究方向涵盖房地产金融、环境经济与政策等多方面。在教学与科研并重发展理念下，其关于城镇规划的研究在英国大学学科评估中位居榜首。

随后，经济所龙登高教授对中国地权市场的历史和土地的现代解释做了详细的介绍。接下来，剑桥大学土地经济系Jose Vallejo Bermeo、Christopher Hooton、Victoria Ormond三位博士生先后做了最新学术成果汇报展示。他们的汇报分别涉及发展中国家的住宅金融风险、评估土地政策以及银行、债务契约和房地产行业。最后，经济所杨崇圣博士生从平台经济的视角讨论了经济特区及其政策。

午餐时间，双方进行了自由而热烈的讨论，并表达了对更广泛合作的期待。交流会在愉快的氛围中圆满结束。
