第五届清华大学-东京工业大学文科教师恳谈会在我校召开

以“大学人文社会科学教育和研究的前沿探索”为主题的清华大学-东京工业大学文科教师恳谈会于9月5日下午在我校人文社会科学学院举行。 

恳谈会是“清华大学-东京工业大学联合培养研究生项目”的一个组成部分，该项目迄今已实施6年，其间清华人文社科学院和公共管理学院已派遣10余名博士和硕士研究生赴东京工业大学学习，同时接受了6名东京工业大学的留学生来清华人文社科和公共管理学习。围绕着联合培养研究生的需求，双方教师还建立了联合授课和每年一次轮流在两校举行恳谈会的制度。迄今为止，上述恳谈会已在清华举办了3次，在东京工大举办了两次。

本次恳谈会分为两个分会场同时举行。“政治·经济”分会场由人文社科学院经济研究所的高淑娟教授主持；“语言·文化”分会场由外文系冯峰教授主持。我校人文社会科学学院和公管学院的10余名教师和东京工业大学社会理工学研究科的10余名教师参加了恳谈会并分别在两个会场上进行了交流。恳谈会的综合讨论安排正式的研讨会后，与“清华大学——东京工大联合培养研究生项目”文科组负责人，科技与社会研究所杨舰副所长代表清华方面举办的联谊晚餐一并举行。清华大学日本研究中心曲德林主任到场致辞。清华-东京工大师生之间进行了进一步更热烈的交流。根据会议安排，6号全天东京工大教师根据各自的兴趣和专业，同清华师生就今后的教育、学习和研究计划进行了个别、深入的交流。
