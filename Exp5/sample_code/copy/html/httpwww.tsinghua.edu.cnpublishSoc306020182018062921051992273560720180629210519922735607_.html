原日本岩手县消费合作社联合会会长加藤善正先生应邀到我系讲座

2018年6月5日下午，原日本岩手县消费合作社联合会会长加藤善正先生应邀到我系举办了一场题为“日本消费合作社发展史回顾及对中国的启示”的讲座。本次讲座由中国人类学民族学研究会发展人类学专业委员会主办，清北消费合作社协办，系“发展人类学讲座”第2讲，由发展人类学专业委员会主任张小军教授主持。
加藤先生首先讲述了自己从事消费合作社工作的经历，从大学消合到岩手县消合建立运营，他58年来一直从事消合运动，积累了丰富的经验。消费合作社简称消合，专业农民合作社简称专合，在日本消合的知名度很高，但对其理念理解的人很少。在产业革命的背景下，工人为了谋福利建立合作社，其中只有Rochdale（英国曼切斯特）公正开拓组合幸存下来，提出出资、利用、运营三位一体的原则，这个原则为消费合作社的发展提出了有效路径。具体来说，出资者可以参加消合成为社员，一个社员出资金额占消合总资金的10%以内。社员要买消合的产品，不买的人不能参加。成员对领导人和对消合产品提出意见，如好不好，贵或便宜，好吃不好吃等；参加开会学习，介绍消合产品给朋友们；重要的事情以总会和理事会投票决定。
在此基础上，加藤先生介绍了日本消合的发展背景和具体发展情况。二战后，日本工业快速发展、物价上涨，出现了食品安全问题，工人消合、大学合作社等不断建立。消合由消费者们共同组成，代表消费者利益，不通过超市等中间商而直接与生产者沟通、商讨价格，监督产品质量。消合通过与生产商签署产销合同保障食品安全和稳定的价格，产销合作有三个原则：1. 生产者：产地向消费者公开（产地透明）；2. 栽培方式、养殖方式向消费者公开（过程透明）3. 生产者和消费者间交流和合作。目前也存在中间商与消合产品之间的竞争关系。合作社与企业不同，企业的目的是盈利，而合作社的理念是以民主平等公平为基础，实事求是，以对社会负责和尊重他人意见为信条。2016年联合国教科文组织把合作社确定为世界文化遗产，加藤先生把原因归纳为：在全球激化贫困和差距，全球变暖、气候异常，粮食危机和数亿饥饿人口，战争和内乱，能源危机和土壤荒废的背景下，只有合作社始终坚持贯彻自己的理念才能带给孩子们健康的食物，带给家庭幸福，才能保护地球。
讲座最后，加藤先生一一回应了现场师生的提问，张小军教授为其颁发了讲座纪念证书，此次讲座在现场师生热烈的掌声中结束。
