KCL政治经济学系副系主任Brian Salter教授做客清华大学科学社会学与政策学沙龙第89期

10月31日下午，伦敦大学国王学院（KCL）政治经济学系副系主任、全球生物政治研究中心主任Brian Salter教授做客我清华大学科学社会学与政策学沙龙第89期，从政治经济学买方和卖方市场以及不同创新模型的角度，解释了目前全球范围内干细胞疗法治理的“乱象”。我所王程韡博士主持了沙龙，来自我校社科学院、生命科学学院，以及中国科学院、人民大学等单位的二十余名师生参加了本次活动。

在报告中Salter教授指出，目前各国在制定干细胞疗法的规制政策时往往都遵循了科学为导向的创新模型。这种模型的终极目标是产生中立、普遍的知识，因此在临床试验方面也更加谨慎，比如往往需要12年甚至更长的时间，全球也仅有8家总有的机构真正满足了这种模式的全部要求。然而全球却有着大量的病危甚至濒死的患者需要尝试这种新的疗法。供求之间存在着严重的矛盾。相比之下，医学领域的创新却遵从着不同的理念。一切医药都应以更好地保证患者的利益，减少他们的病痛为己任。在这一理念下也产生出模型2（欧洲模式，少数患者先导），模型3（非欧洲模式，多数患者先导）以及模型4（综合模式，强调先导用户的诊疗信息向临床试验开放并作为其基础）等新实践。但很可惜，目前各国在政策上的规制并没有及时跟上实践的步伐。这是学术界和政策界需要共同关心的问题。
