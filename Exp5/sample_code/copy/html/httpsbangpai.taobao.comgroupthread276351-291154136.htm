FAQ：

有部分卖家觉得符合上述条件就可以通过审核了。天天催催审核。

上述是基本门槛，还有其它条件。 

首先，是品牌店铺？ 而不仅是有品牌的店铺。必须主营是这个品牌（店铺至少80%-100%成交占比是这个品牌或该品牌下的子品牌），并不是说店铺做一堆各种品牌，其中一部分做一个品牌，就可以称为品牌店铺了。

其次，店铺有没有品牌运营的意识。有些卖家自己商品后台发布在什么品牌下自己也搞不清楚。没有运营这个品牌的意识，怎么能称为品牌店铺。店铺全部氛围有没有宣导突出这个品牌。消费者是否能通过你的运营认知到你的品牌。品牌在全网的排序，搜索如何。或者提升如何。

第三，有没有整体的品牌运营的规划，营销推广方案，自己的设计风格。

 

当然，现在不符合不代表今后也不符合，欢迎更多希望将自己品牌运营发展的店铺逐步调整改善，而并不是为了通过某和活动为目的。
