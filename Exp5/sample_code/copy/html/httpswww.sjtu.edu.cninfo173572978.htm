2018年第五届全国现代制造集成技术学术会议通知

时间：2018年10月08日 10:24 发布人： 浏览次数：[]

[责任编辑]孙佳 [信息提供]机械与动力工程学院 

第五届全国现代制造集成技术学术会议将于2018年11月17～19日在上海隆重召开。 

本次会议在国家科技部的大力支持下，由《计算机集成制造系统》编辑部主办，上海交通大学承办。会议将围绕新时代先进制造技术与产业发展，邀请国内先进制造技术领域的多名院士和知名专家介绍先进制造前沿动态，组织行业内专家学者交流研讨，展示展览相关技术产品。热忱欢迎全国高等院校、科研院所以及制造业企业从事先进制造技术研究、开发和应用的专家、学者和技术人员光临大会。会议有关事项如下： 

一、会议主题

大融合、大变革——新时代先进制造技术与产业发展 

二、会议时间和地点

2018年11月17日～19日，上海·临港。 

三、会议内容

本次会议设置大会主旨报告、分会场主题交流，以及智能制造示范企业参观，会议同期还将举办相关技术产品展览展示。 

（一）会议交流内容 

1．先进制造业的发展形势与国家政策研究。主要交流我国先进制造业发展现状与问题，国家“国务院关于深化‘互联网+先进制造业’发展工业互联网的指导意见”、“新一代人工智能发展规划”、“‘十三五’先进制造技术领域创新专项规划”等政策解读，世界主要国家政策比较等。 

2．新一代信息技术与先进制造技术的融合。主要交流云制造、物联网、大数据、网络协同制造等新一代信息技术与先进制造技术的融合；新一代人工智能基础理论体系和关键共性技术，及其与先进制造业的融合。 

3．制造技术与装备创新升级。主要交流增材制造、激光制造、机器人、新型电子制造、高档数控机床、制造基础技术与关键部件、工业传感器、绿色制造等核心技术。 

4．新时代先进制造企业实践案例。主要交流新时代先进制造技术在企业中的具体实践与应用，以及先进制造企业转型升级的思考。 

（二）参观智能制造示范企业，与企业对接交流 

组织参会人员参观中国商飞C919总装车间和上海交大临港基地智能制造集成测试验证线，与企业现场交流。代表自愿参加。

十、参会对象及会议规模

全国高等院校、科研院所以及制造业企业从事先进制造技术研究、开发和应用的专家、学者和技术人员等，会议规模拟为300人左右。 

十一、会议费用

每位参会代表需交会议费2800元，现场缴费，可现金、可刷卡。会议期间交通住宿费用自理，住宿费680元/间。 

十二、会议地点及交通路线

1．会议地点：上海滴水湖皇冠假日酒店 

2．地址：上海市浦东新区临港城南岛1号 

3．酒店电话：总机：021-20339999；传真：021-20339977 

4．会议交通： 

（1）接机（接站）：16日下午3点至晚上8点，会务组每隔1小时在上海虹桥火车站、上海浦东国际机场有接站车，并在地铁滴水湖站设有接驳车，负责将参会代表送至滴水湖皇冠假日酒店。 

（2）自驾：上海市自驾车参会的代表，可凭会议通知或会议名称免费停车。
