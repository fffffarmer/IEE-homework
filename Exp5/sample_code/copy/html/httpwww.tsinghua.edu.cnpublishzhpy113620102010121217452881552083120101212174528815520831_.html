蛋白质结构动力学

课程简介：

蛋白质生物功能与结构动力学特征紧密相关，不同结构运动模式决定了蛋白质在生化过程中的不同生物功能。给定一种蛋白质的空间结构，通过建立结构动力学模型，进而得到蛋白质结构的运动轨迹，或者将结构整体运动分解为不同频率振动模态的叠加，能够帮助进一步理解蛋白质生物活性点在配体结合、催化过程中的运动特征。

本课程讲授蛋白质结构动力学研究的最新进展，重点分析了三种蛋白质结构动力学的物理模型和求解方法，即分子动力学方法、谐波动力学和随机动力学方法，详细讨论了谐波模型中的原子模态分析、弹性网络模型等动力学方法，并对每种动力学模型都给出了模型的适应性分析和数值方法。在本课程介绍的蛋白质结构动力学模型中，每种模型都给出了相关研究方向的未来研究展望，并介绍了蛋白质结构动力学在生物研究领域的应用。

课程特点：与同类课程相比，本课程在教学内容方面侧重于培养学生为复杂物理问题建立合理数学模型的能力，利用研究生已掌握的力学、数学方法研究蛋白质结构振动问题。通过本课程的学习，可以培养学生利用数学、力学方法研究蛋白质结构动力学的能力，提高同学对生物大分子研究的兴趣，增强跨生物、数学、力学学科的交叉研究能力，为应用数学、力学专业同学进一步的科研工作打基础。

教学大纲

1 引言 （介绍课程内容、主要学习目标、通过本课程学习达到的要求）（1学时）

2 蛋白质结构的基本知识  (讲授蛋白质结构的基础知识，要求学生了解掌握蛋白质结构及折叠过程的基本概念) （5学时）

         2.1 自然界中的20种氨基酸

      2.2 氨基酸侧链的几何结构和物理属性

      2.3 多肽链及蛋白质结构

      2.4 蛋白质结构的非共价作用力

      2.5 蛋白质结构的折叠过程

3 蛋白质结构的分子动力学(Molecular Dynamics)模型 (讲授MD模型的基本原理、算法，要求掌握MD的计算步骤) （6学时）

      3.1 蛋白质结构势能和作用力计算

      3.2 分子动力学的Verlet 算法

      3.2 边界条件

4 蛋白质结构的谐波动力学(Harmonic Dynamics)模型 (讲授HD模型的基本原理，要求掌握HD模型的计算、分析方法) （8学时）

      4.1 原子模态分析（Normal Mode Analysis）方法

      4.2 弹性网络模型（Elastic Network Model）

5 蛋白质结构的随机动力学(Stochastic Dynamics)模型 (讲授SD模型的基本原理和方法，要求掌握CSAW的数值模拟) （8学时）

     5.1 蛋白质系统的Langevin 方程

     5.2 蛋白质结构动力学的CSAW模型

6 蛋白质结构动力学方法的应用 (了解蛋白质结构动力学在各生物研究领域的应用原理) （4学时）

     6.1 离子通道的振动模态分析

     6.2 改进晶体结构B系数

     6.3 酶特异性的动力学基础

开课教师：孙卫涛
