2018亚青赛：

2018年10月20日晚U19国足0：1负于塔吉克斯坦队。

比赛中，上半时，U19国足中场大将徐皓阳的任意球颇具威胁；下半时，他一次失误导致失球。

10月21日，我用新浪微博的私信给徐皓阳发了两条信息，鼓励他 “要学会用坚强面对挫折！“

“告诉你的队友，你们身后至少有上百万的国足忠实球迷的支持！球迷最关注的不是比分，而是你们是否能坚强地面对困难！”


                            
   

                                                            
                                                                                                          
10月23日，17:00，亚青赛：U19国足将迎战沙特队！生死战！ 10月22日，我用新浪微博的私信给蒋圣龙等U19国脚发了条信息，鼓励他们 “明天的比赛，心态很重要。要做到乐观、冷静、坚强” 。
           

      
                                                          
                                                           



我想，其实大多数球迷都期盼所有梦想都开花..........


----------------------------------------------------


我其实真想给中国管足球的人说一句：与其组织训练中超的U25球员，还不如从U15球员抓起！

          我相信中国足球的前景是光明的！

           虽然，道路可能会有些曲折。



只要没人乱折腾，中国足坛就算是“鸡窝”，也终将飞出金凤凰！
