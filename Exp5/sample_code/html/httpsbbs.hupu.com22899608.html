集结Reebok两大核心科技！FUSIUM RUN 开箱评测 由  Winnieyyy 发表在虎扑装备·运动装备 https://bbs.hupu.com/gear
Reebok在专业健身迷和健身爱好者的心目中一直有着比较高的影响力。还记得Reebok之前联合专业机构发表的Reebok Floatride Space Boot SB-01太空鞋履吗？主推新世代航太科技的缓震中底Floatride Foam，加上类似拳击鞋款的超高鞋筒，整体新颖帅气，但是说实话，我是驾驭不了这样的鞋子的……
 

现今运动品牌推出的运动鞋款总要融入更多的潮流元素，以往专业与休闲穿搭是两件事情，但是Reebok推出的轻量避震跑鞋FUSIUM RUN结合新世代航太科技缓震中底Floatride Foam，加上轻量科技与编织鞋面技术，是一款兼具实用与酷炫外观的超帅鞋款呢！
 

要挑战自我，就不能向热浪低头! 夏日出行，无论户外休闲或烈阳狂奔，首要装备就是一双可靠舒适运动鞋，在炎热天气下仍可发挥良好功能。而这次Reebok为夏季跑步训练推出的新款跑鞋Reebok FUSIUM RUN则结合了目前Reebok最新的两大科技，具体是什么呢？一起来看看

Reebok FUSIUM RUN 开箱
 
鞋盒
 
标签

一如既往的MADE IN CHINA，同时需要注意的是，Reebok的42码对应欧码也就是EUR码是42.5码，建议大家购买的时候参照脚长CM码选择尺码。
 
开箱


Flexweave科技鞋面
在运动服装行业，仅仅制造设计外表美观的鞋子和服装是不够的，因为性能才是重中之重。这就是为什么其他知名体育品牌也一直在试验和开发新的改进材料。
 

这双FUSIUM RUN采用的 Flexweave 科技鞋面 在17年9月就已发布，其首创的8 字型缠绕的结构给鞋面产生了独特的空间。在鞋面中段两侧用上Flexweave 科技鞋面的高韧度纤维，既加强稳定性亦保持了灵活性能。而且鞋头采用了柔软针织材质，高度的贴脚之余，也能增加透气度，极为适合夏日天气。

Floatride科技中底
如果把一双跑鞋分为三个部分：鞋面（Upper），中底（Midsole），大底（Outsole）。如果把一只鞋子看成是一辆车的话，那么鞋面相当于车身，中底则相当于悬挂系统，大底则是轮胎，可见，中底是跑鞋的灵魂，也是区别各大厂商的核心标志，立身之本。
 

传统的中底大多采用 EVA（Ethylene Vinyl Acetate）材质。EVA材料具有良好的柔软性和橡胶一样的弹性，加上相对低廉的价格，使之成为各大鞋厂广泛使用的中底材料之一。相信很多跑友都发现，一双EVA中底的跑鞋在跑上一定的里程之后，脚感就会变硬，这是因为EVA在经过跑友们的反复踩踏之后，泡沫中的空气被挤出去了，这样一来，被“压制”的EVA是无法变回原来的形状的，缓冲的能力也会大不如前。而由著名科技公司德国巴斯夫研发生产的Floatride科技中底由颗粒压制而成，很大程度上解决了这个问题。它比传统的EVA中底质量要轻，却仍然保证了很好的回弹性，在缓震上表现不俗。这样的材质被用在太空鞋上也就不足为奇了！

去年，Reebok将Floatride科应用到了跑鞋上。Reebok Floatride 这款先行者跑鞋在 17年5月获得《Runner’s World跑者世界》杂志美国版的“2017夏季最最佳首秀”跑鞋。9月获得《跑者世界》杂志中国版的“2017最佳首秀”跑鞋。足以证明Floatride这一材质在跑鞋上出色性能了。
 

本次新推出的FUSIUM RUN同样也是如此，倾斜式后跟设计则为脚跟落地时提供更多缓冲空间。同时前脚掌也有一定程度的向外扩展，保证了落地的平稳性。
 


鞋底
这次 Reebok FUSIUM RUN 采用了分隔式样的外底设计，能将跑步时的冲击平均分布，提升跑步时的舒适性，亦能加强支撑及稳定性。从图中可以看到跑鞋的底部各个区域的设计明显不同。
 

这次新设计的大底有针对性地依据跑步时足底压力的分布状态而设计，可以看到在后跟及外侧摩擦较多的地方设计了非常密集的小孔，可以增加耐磨性。而运动引导凹陷沟槽相比以往的跑鞋设计明显加宽加深，再轻量化的同时又不丧失灵活性是非常不错的设计。
 

此外，橡胶大底加强耐用性之余，更带来了适当的抓地摩擦力。

产品实际测试
上脚FUSIUM RUN的第一个感受，居然这鞋没有鞋垫？是的FUSIUM RUN确实没有设计鞋垫，他直接将传统的鞋垫与中底及鞋面缝合在了一起，这样的设计是不是更适合夏天赤足穿着呢？
 

鞋舌设计采用了目前主流的一体式设计，保证了鞋舌在运动过程中不会歪斜，减轻了移动摩擦给脚背带来压力。

从试穿体验上来说，FUSIUM RUN继承了Reebok一如既往的风格，前后掌回弹适中，有力度的同时也不会感觉到卸力，保证了跑者在跑步过程中不会因为缓冲性能太好以至于加速感偏弱的问题。
 

前脚掌跨宽度的设计得益于弹性鞋面以及中底的外扩设计，保证了充足的活动空间，加上绝佳的透气性在夏天穿着会感到非常舒服。Reebok FUSIUM RUN在鞋面中段两侧用上Flexweave科技鞋面，既加强稳定性亦保持了灵活性能，包裹性能还是非常出色的，同时也有一定的足弓支撑，并且在脚后跟也加强了固定。
 


总结
总体来看，Reebok的这双跑鞋，主打特色计时轻量与性能合二为一的概念。同时其699元上市价格可以说在目前的跑鞋市场竞争中充满潜力，相信当更多跑者尝试过这双鞋之后，一定会对他的缓冲性能以及鞋面的透气性产生好感。
 

近几年Reebok在跑鞋科技研发上走得很稳健，每款产品都可以说是十分用心的，无论是价格还是设计都是比较适合大众跑者。
跑步作为一种生活方式已经深入人心，希望Reebok未来能带给大家更好产品。
