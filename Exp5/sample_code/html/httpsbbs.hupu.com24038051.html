[流言板]踢球者：拜仁对费基尔感兴趣，首席球探已经开始现场考察 由  漂移过境 发表在虎扑足球·拜仁专区 https://bbs.hupu.com/bayern

虎扑10月22日讯 根据《踢球者》报道，今夏里昂球星费基尔转会利物浦失败，但是他仍然在转会市场上，拜仁已经派出他们的首席球探劳伦特-布瑟尔前往里昂进行现场考察。

布瑟尔本月初现场观看了里昂2:2战平顿涅茨克矿工的比赛，拜仁对费基尔的兴趣已经持续了一段时间。

费基尔在今夏转会利物浦失败，当时费基尔已经同利物浦完成了合同签字，但是随后里昂官方宣布转会失败，费基尔之后跟随法国国家队拿到了世界杯，他同里昂的合同要到2020年到期，因此他仍然在转会市场之中，他的预计转会费为6000万欧至7000万欧，值得一提的是，里贝里和费基尔都是同一个经纪人。
