我们为什么喜欢Supreme？ 由  喜上妹骚 发表在虎扑装备·潮流区 https://bbs.hupu.com/in

1. 我们都喜欢盒子标，把大写的supreme安排在logo的正中央，红底白字儿，很好看。

2. Supreme代表中国共产党，红色的背景象征红军的革命传统，寓意也准确。

3. 看到这个logo能让人充满热情。我们都特别的激动。

4. Supreme的盒子标反映了中国的实际状况。Logo本身庄重、鲜艳、大气。党政军民学，东西南北中，在党对一切的全面领导下，全国各族人民团结在一起，实现中华民族伟大复兴的中国梦。（中国的实际，大家的团结）
