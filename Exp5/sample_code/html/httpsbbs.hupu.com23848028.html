我球衣伙伴的其中之一@竺正豪 也是一个对老球衣及其钻研的另类。他那天微信我说，“我们喜欢的那些球员现在都已经5、60岁了。”是的，也许有些更老。所以，我也觉得能够有机会和J博士见上一面，做个访谈聊上30分钟，是何等难得又珍贵的事。

 


在寒暄之后的第一个问题，就让J博士带我们回到旧时光，回忆一下当年为费城76人效力时的难忘片段。而68岁的J博士就像是那些睿智又酷酷的老人一样，略缓慢的语速微笑着说，“Back to the old days? Sure, sure.”

 


1983年是费城76人历史上的最后一个冠军。J博士说到那一刻，眼神中不自主地流露出骄傲的光芒。“那年摩西马龙是常规赛MVP，而我是全明星赛MVP。我们有四个全明星，我、摩西、莫里斯-奇克斯和安德鲁-托尼。”纵使活动前做了许多功课，甚至看了三个纪录片，J博士一开口就告诉我了一些前所未知的信息。当季四个全明星，如今的宇宙勇大家脑补一下，就知道当时的76人阵容是何等华丽丽的存在。


 说到现在的76人，J博士也是极尽溢美。“他们一直说的那个‘过程’，现在看起来很不错了。我觉得他们目前状态正盛，也希望球迷们多关注这支年轻的球队。”我听着听着就想到2013年，那年的勇士来到上海，呼声远在科比的湖人之下，但我已经十分喜欢当时的勇士。篮球就好像是一首交响乐，正确的打球方式会被优美地流露。这支76人其实有一些当时勇士的样子。

 


在现场J博士告诉我，76人新晋上任的总经理埃尔顿-布兰德已经联系了J博士，希望今年季后赛他能够在赛前进入更衣室给年轻的球队一些关于季后赛建议和激励。J博士在纽约篮网获得两个ABA冠军后，ABA与NBA合并，并且他被交易至76人，却先后三次在NBA的总决赛中铩羽。最终，J博士在83年努力搏出一个冠军，成就了ABA和NBA双料夺魁的成就。

 


J博士也说，每一次失败都意味着仍有进步空间。“I learned a lot not as champion but as a runner-up”他更坦诚地补充道，“a loser.”是的，没有任何做作的架子，硬扯的面子。一个将近古稀的老人，历尽铅华道清真理，失败没什么好怕，面对失败我们应该变得更好。所以，我真的相信像J博士这样的赢家可以在季后赛的更衣室里，给到这支年轻的76人许多的中肯建议。

 


现场环节里还有国内插画师为Dr.J制特意作的精美图画。她的诠释也非常好，这幅长着翅膀羽毛的J博士插画既象征Doctor在场上标志性的滞空翱翔，也意为在场下作为NBA表率的传奇巨星，他的优雅形象如同天使一般。的确，J博士在70年代就是NBA的门面和招牌，也是始终被公认为举止谦和的绅士。所以当我伏在他耳边轻声传译时，也看他开心地不住点头。

 


其实还聊了非常多的话题，说到他和艾弗森如同家人般的关系，也说到罚球线起跳扣篮，他分享的概念是篮球运动员在变得更强壮的同时也要兼顾灵活性，这也是他能获得更好的弹跳和速度的心得。最后活动结束时，J博士在我身上的球衣签名，比起现在球员大多的刷刷几笔，我觉得他签了好久好久——Julius “Dr.J” Erving 2018。



好了，我知道姿势是个槽点！！




其实整个活动远不止这半小时，之前还有关于球衣的分享，@game_issued 专程从香港赶过来给现场的大家分享了很多故事，现在估计有在去加州的路上了。致敬！当然现场展出的76人专题球衣，还有我的好基友@aijewelz 终于在自己的家门口和好兄弟一起做了球衣展出！还是开头那句话，当年买第一件球衣的时候会想到有这样的时刻吗？#不忘初衷#

 关于NBA冠军主题展，J博士参观后也是赞不绝口。他说最喜欢的部分还是冠军展对于早期历史的铺陈和介绍。他觉得那些远古时代的球员也同样十分伟大，并值得纪念。因为正是有了前辈打下的基础才有了如今的联盟。所以，这么好的展览各位筒子就不要错过啦！哈哈哈！
