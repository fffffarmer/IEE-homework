虎扑10月22日讯 中国女足球员王霜自加盟法国巴黎圣日耳曼女足以来，表现非常突出。今天出版的《人民日报》刊文指出，尽管还在磨合与适应的过程中，但王霜的亮眼表现已足以让人们在中国球员身上体味到久违的荣耀。王霜留洋的可贵之处在于，本身能力就很强却还保持积极进取的心态，已经赢得好名声依然勤勉好学、谦虚谨慎。在王霜身上，人们不仅可以欣赏到她的精彩球技，还可以感受到她的远大志向。

《人民日报》文章全文如下：

对中国足球来说，能不能涌现一批又一批志存高远勇于实践的优秀球员，恐怕也是评判中国足球能否打好翻身仗的重要一环

最近一段时间，中国球员王霜火了。网络上，几段王霜代表巴黎圣日耳曼女足出战联赛及欧冠并打进漂亮进球的视频广为流传。

23岁的王霜因其在几次大赛上的优异表现而得到更多关注。参加完雅加达亚运会后，她毅然放弃国内联赛的高薪、选择到自己感兴趣的欧洲俱乐部历练。声名鹊起之时勇于放下薪酬上的诱惑去欧洲联赛闯荡，是因为她心中还有更大的梦想。而在巴黎圣日耳曼足球俱乐部，尽管还在不断磨合、不断适应，但王霜的亮眼表现已足以让人们在中国球员身上体味到久违的荣耀。

中国足球球员留洋，更多是国外俱乐部出于打开中国市场考虑而签约各自感兴趣的球员。这些球员来到国外俱乐部或者很难得到上场机会，或者得到有限的上场机会也表现平平。而国内联赛薪资水平的一涨再涨，无形中也对球员的留洋意向带来直接影响。在国内联赛轻轻松松就可以拿高薪，谁还愿意吃苦受累在国外联赛中坐冷板凳呢？但这样一种不肯到更高水平联赛闯荡的守成心态，很难对球员水平的锤炼与提升带来积极影响。长此以往，球员水平得不到切实提升、优秀球员难以脱颖而出也就可能成为现实。

王霜留洋的可贵之处在于，本身能力就很强却还保持积极进取的心态，已经赢得好名声依然勤勉好学、谦虚谨慎。在王霜身上，人们不仅可以欣赏到她的精彩球技，还可以感受到她的远大志向。作为当前国内很有潜质的女足球员，王霜的追求与梦想，让人们深切感受到新一代女足球员身上的勃勃生机。

志存高远才能行稳致远。对王霜来说，她正通过自身努力尝试着将梦想变成现实；对中国足球来说，能不能涌现一批又一批志存高远勇于实践的优秀球员，恐怕也是评判中国足球能否打好翻身仗的重要一环。
