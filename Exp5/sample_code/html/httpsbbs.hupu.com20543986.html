韦德之道6，开箱+上脚+实战评测（已实战） 由  Satorpia 发表在虎扑装备·运动装备 https://bbs.hupu.com/gear
实战表现：体重63KG，打法不好说，看谁守我，能突死就突死，能投死就投死，模板老大蜗壳，场地很平整的室外塑胶场（最考验抓地的场地）。
先放一张图，这张图的意思就是，我有多双鞋做对比：


缓震：4.5颗星，跟AJ30类似，前软后硬，后跟的CUSHION，用力跺脚的话，还是会感到震脚的，至于实战表现还行，前掌DRIVE FOAM和云科技，表现非常不错，比AJ30软，但是没那么弹，至少缓震方面，这双鞋在后卫鞋的表现里算是顶尖的了，如果后跟的表现能再好一点的话，完全能打5颗星，至于缓震材料的持久性，那就不得而知了。

包裹：不打分，理由是，这双鞋对瘦脚不太友好，前掌不宽也不瘦，鞋带系统较鸡肋，如果是一双肥龙脚，或者穿很厚的袜子，估计包裹能有质的飞跃，但只能说，这双鞋的包裹对我来说只是够用而已，毕竟我是连AJ30, KOBE AD NXT，和CE 2017 PK都能穿的下的人。

支撑：4颗星，支撑倒是非常不错，魔术贴和外侧的菊花标志以及碳版，能够给予双脚非常有力的支撑，而且鞋面也能锁住双脚，只可惜因为包裹对我不太友好，所以没能打5颗星。

透气：我的实战鞋只在打球时穿，所以从来不在乎透气性，不打分，不过WOW6的鞋面不是很厚，所以透气应该不会差。

脚感：5颗星，穿起来没有硬伤，不压脚，不咯脚，感觉非常舒适，脚感很适合拿来压马路，而且重量适中，不重不轻，对于后卫来说，绝对不会嫌重。

抓地：3.5颗星，（10.22更新，我今天穿AJ30都会出现刹不住的情况，看来最近场地特别滑啊，但是WOW6依旧是不如AJ30，我不会改变评分，但WOW6的抓地绝对不差，只能怪场地太滑了……）我觉得这双鞋最大的败笔就是鞋底的纹路，如果是在自己做动作，那么抓地还是很给力的，差在哪？差在与人有身体接触的时候，这鞋底有概率会滑，虽然场地是最滑的场地类型，但是表现与AJ30和CE 2017 PK差了一截。也许在干净的木地板会表现不错。我不是说这双鞋抓地差，而是这双鞋的抓地没能达到与这双鞋匹配的水平，但也绝对够用，我相信至少比AJ31强不少。

耐磨：4颗星，实战2个小时，看不出磨损，不过还是建议不要硬刚水泥。

最大的缺点：鞋带系统不行，可调节范围小。

总评：WOW系列实战性能最好的一双鞋，除了抓地差了一点点，从实战性能来说，挑不出什么毛病，而且脚感不错，WOW系列我穿过808和3代，我敢肯定的说，WOW6的脚感比前两双要好得多，也许也是压马路最舒服的一双鞋。至于性价比，原价1099，不敢说值，但也没有太溢价，这双鞋我觉得最合适的价格是8-900，毕竟我的毒液6都得花600多买的。
有关缓震，至少目前从这双鞋的表现，DRIVE FOAM+云科技已经有不输AD和NIKE的表现，如果持久性的问题得到解决，那么在缓震方面，国产鞋已经完全不输国外大牌了。

-------------------------------------------------------------分割线---------------------------------------------------------------
实战评测等我下午实战之后再写感想。
第一次做球鞋评测，鞋子购于深圳来福士WADE STORE。
早上9点半就到了，发现商场没开门，就去楼下星巴克喝咖啡了，到了10点用手机刷了李宁的官网，结果秒没，乖乖上楼买，进店铺时已经10点过了七八分钟那样，一个大个和一个大妈正在买鞋，大个和大妈不是一伙的，好像一共提走了2双还是3双鞋，都是买韦德之道6，最后一双43码被我给买了，问了店铺小哥，店里一共只到了10双，我走之前只剩3双了，都是大码和小码，42到44估计是没了，1099，刷卡走人。
进入正题，先说一下我的体重，早上空腹刚测63公斤，我的脚是瘦脚……（玛德先吐槽虎扑，明明图是正的，上传之后就歪歪扭扭的，还转不了）：

提袋，没什么好说的：


盒子：抽屉式改成合盖式，硬度和厚度都不如之前的盒子，减配无疑



送的镂空鞋袋和贴纸：


正主：


后跟内侧:李宁商标和云标和CUSHION和DRIVE FOAM，中底材料应该是云，估计是云4吧


小标签，锻炼一下颈椎：



鞋头做了防踢设计，写有李宁标志和WOW全拼：


侧面大菊花，没什么好说的，大家都看得多了，顺便说一下，中低的颜色是渐变的。


鞋底材质不知道是水晶还是什么的，有点透明，鞋底纹路是的确是战斗机样式，无数个类似美国F117战斗机的小飞机。中间就是大家熟悉的碳板，前掌能看到一些深色的东西，应该就是DriveFoam。


魔术贴不确定是否是碳纤维，不过李宁是出了名的碳纤维控，我估计是，黑色和透明的都是橡胶材质。


魔术贴内侧一边是罗马数字6和英文的SUN RISE晨曦。


鞋子是前掌内靴设计，不过鞋带系统的前半边是一体的，后半边才是分开的，鞋舌是韦德的小菊花LOGO。


后跟英文左脚DWYANE和右脚WADE。提带是三色的。鞋舌内侧是骚紫色的菊花标志和WOWSIX，鞋舌的填充还行，中等，不厚也不薄。


鞋垫内侧，依旧是小飞机，鞋垫软软的，橡胶的感觉。


正面韦德签名处是突起的，感觉应该不会被磨掉。

采用新的编织鞋面，质感不错，很多通气孔，估计透气效果不错，还有一排排的“减速带”


鞋的内侧用了一定的热熔胶覆盖鞋面。


补几张上脚图：






上脚对比，与几乎预定2017最佳球鞋的CE 2017 PK做对比：
缓震：我感受不出区别，不论是前掌还是后掌（修改，又认真的踩了踩，后掌还是CE 2017 PK软弹一点，不过WOW6也不差，不仔细感受真的觉得差别不大），不知道是我体重太轻还是WOW6的缓震技术太好，反正我是几乎感受不出区别。
包裹和支撑：被CE 2017 PK完爆。WOW6的包裹一般，前掌不算大，中等，但是我是瘦脚，所以前掌依旧有一点的空间。不过对于宽脚的人，估计感受应该不错，首先说明一点，长度没偏，就是李宁43码的长度，但是鞋带系统好看大于实际，有点勒不紧，穿一双STANCE估计会好很多，而韦德首战则是穿了2双袜子，不知为什么这么设计，详细的感受等我下午实战回来再说。
抓地：做了一些简单的侧移，感觉不错，但我实战回来再做感想。
