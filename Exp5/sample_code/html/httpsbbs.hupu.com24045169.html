楼主大三，十三号晚上在学校操场打球然后被这个学妹加了QQ， (我当时不知情，是她找了场下另一拨的同学要我的q)

小弟已经有女朋友了所以就把这件事给女朋友说清楚了，然后给这个学妹也说清楚了，聊了几句后就放在列表躺尸。后面十七号这个学妹发了说说，内容大概就是没有带校卡借了某个路上男生的卡，凭我多年经验，这条说说字里行间都是暧昧正在撩汉子的气息，果不其然在昨天晚上她俩就公开在一起了。

我正感叹现在的大一小年轻怎么这么着急呢，突然一想不对啊，因为就在两天前她还在学校表白墙下面疯狂求偶撩汉……这俩人十七号认识开始眉来眼去搞暧昧，昨天在一起，中间短短五天这个学妹还在表白墙上广撒网……

表白墙每天消息太多了，只找到了一张她评论的图片，福利上某个jr女朋友吧还有曼源女神
