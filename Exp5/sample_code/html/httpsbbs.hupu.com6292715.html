 先借用几张不知道是那位大哥的图，接下来就是我自己的了。 
















皮子质感超级好，又厚又柔软，到现在都是很有光泽。



鞋后跟是不是有点椰子2的影子。





鞋舌现在来看还是很厚实的也很大，能够很好的包裹住脚踝。






刚一拆来，我靠，我还以为当年NIKE就这么坑爹了，后掌无气垫？一大块像ADIDAS鞋子里的橡胶，后来才知道我错了，原来后掌是Air sole，我一直都以为这鞋是前后ZOOM的，因为介绍就没有提到Air sole

ZOOM已经氧化的只剩中间的材质了，但是中的这个棉片还是很厚实的。





这双鞋真的很好打，速度快，鞋底舒服，透气好，记得基德当年也穿过这双鞋，真希望能够复刻！
