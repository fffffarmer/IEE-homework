   据虎扑一位朋友说因为要发售一耐克什么鞋，或者什么原因，国内又推迟到8月25日发售。目前到底什么情况还未得知，13熊猫本来原计划就是8月25, 8月4日是乔5发售，如果全部赶在一天，也不利于市场火热程度，看目前形式，国外4号是一定发售了，国外一旦发售，国外公司货就很快到达国内，会间接拉低熊猫价格，可以说有利有弊，好的一方面是能够稀释国内价格，不好的是东莞A级货会在这段时间大赚一笔！
 
  不管如何吧，劝所有想买熊猫的小白白门，你们买的预售在各大网站的鞋一定要小心喽！尽量去大店购买，因为你们没见过和公司货基本一致的假厂子鞋。再次提醒小心上当！
