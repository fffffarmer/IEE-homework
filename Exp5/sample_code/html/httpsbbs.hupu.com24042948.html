我和街上很多朋友一样从小看着美猴王长大，以前一直以为六小龄童老师就是我们心中的无所不能，惩恶扬善的孙悟空。后来确实在网上看了一些六小龄童老师的采访，前后有漏洞，包括和杨洁导演的恩怨，也跟风黑过六小龄童老师很长时间

今天翻看六小龄童老师的微博，发现前几条微博一条评论都没有，应该是把微博评论给关了，直接原因应该是这个

可能后来有工作人员告诉了六小龄童老师这句话本来的意思，之前可能六老师还以为是善意的提醒，还在底下回应了抱拳，大概能想象出六老师知道真相后的失落和愤怒，评论关了

他确实有一些不光彩的行为，但是我现在再看，他无非就是想挣点钱，也没有做什么伤天害理的事，年轻的时候可能做过一些让杨洁导演失望的事，可是后来杨洁导演都原谅了，我们网友又何必揪着不放呢。

他都快60岁了，而且塑造的美猴王也是给我们童年有很大影响角色，我觉得网友们还是善良一些吧，
