  每次逛装备区JRS们最困扰的问题：BOOST怎么清洗，本人也有好多BOOST鞋，平时一直穿，起初脏了也是头疼后来朋友推荐了一个神器洗过后效果明显，废话不多上图
加水倒粉后泡沫状
洗后样子，效果非常好



洗完后的污水

1.先找个脸盆水放至浸没鞋底差不多倒多少自己斟酌起泡起差不多就行。
2.然后把鞋子放进去底面泡个20-30分钟左右，同时可以拿布和刷子刷一下鞋面如需清洗也是同理。

总结：BOOST鞋的保养确实麻烦，最好的办法还是及时清理，不要真的等到全变黑发黄彻底氧化后再指望一次洗成新鞋的样子那是根本不可能的。
