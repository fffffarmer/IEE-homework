PUMA推出了一款混合动力跑鞋 满身都是黑科技 由  wzy866 发表在虎扑装备·运动装备 https://bbs.hupu.com/gear

每到年中，各大运动品牌开始相继更新发布自家的跑鞋。PUMA也不例外，这一次他带来了一双前所未见的跑鞋，采用了时下最流行的设计语言：混搭。



PUMA跑鞋黑科技的前世今生

最近几年，随着跑步的人越来越多，在作为一种健康的生活方式之后，大家对跑鞋无论是从外观还是性能上要求都越来越高，这也间接的催生出各种各样的“黑科技”。自从跑鞋被设计出以来，这其中最为重要的部分“中底”一直是大家关注的焦点，因为中底承担着缓冲性能、回弹性能、稳定性能这三个硬性指标。


早在2014年，彪马就推出了NRGY中底黑科技，这是一种叫做发泡微球（Infinergy）的材料，这种材料将普通的TPU材料进行了发泡处理，每一个大小5mm左右的TPU颗粒像爆米花一样膨胀到原来的10倍大小，因而可随双足伸展屈曲，提供随行而动的支撑效果，每走一步，这些能量球就会自由移动，静止时则会聚拢成形。总体上，这种中底具有了弹性更足、缓冲更强和更耐久的特点。


在NRGY之后，PUMA在15年推出了搭载IGNITE科技的高性能跑鞋，这也一举使得PUMA在专业跑鞋领域大放光彩。IGNITE在推出之时，采用了巴斯夫高性能材料Elastopan和Cellasto。Elastopan是一种多组分聚氨酯(泡沫)组合料，其形状、颜色和质感等方面可进行自由设计。Cellasto是一种具有优异物理性质的微孔聚氨酯弹性体，其能够有效阻隔各种振动。


简单的来说IGNITE科技在外形上得到了改变，而且材料可塑性更强，同时又能涂装各种各样的颜色，最最主要的就是其看起来和摸上去不在那种粗粗的颗粒物了，也就是说美感和手感都得到了提升。


而这也正式宣告了PUMA拥有了更加强大的技术基础，去参与跑鞋市场的厮杀，也确实获得了一席之地。

HYBRID 混合动力

如果把一双跑鞋分为三个部分：鞋面（Upper），中底（Midsole），大底（Outsole）。如果把一只鞋子看成是一辆车的话，那么鞋面相当于车身，中底则相当于悬挂系统，大底则是轮胎，可见，中底是跑鞋的灵魂，也是区别各大厂商的核心标志，立身之本。


一般价格便宜的入门级跑鞋的中底绝大部分都是使用EVA泡沫的，这样的跑鞋一般直接在EVA原料上通过裁剪成型制成中底。一双EVA中底的跑鞋在跑上一定的里程之后，脚感就会变硬，这是因为EVA在经过反复踩踏之后，泡沫中的空气被挤出去了，这样一来，被“压制”的EVA是无法变回原来的形状的，也无法提供和缓冲的能力也大不如前，所以下脚感觉就硬了。


而这一次，全球首个融合IGNITE和NRGY两大顶尖创新技术于鞋底的鞋款， hybrid Rocket runner 终于来了。NRGY能量球颗粒自带即时缓冲和舒适弹力，IGNITE泡沫中底提供缓震支撑，两大混合动力相互作用，打造PUMA HYBRID专属的王牌中底科技。


最明显的变化在于两者结合之后，中底泡棉的外观发生了巨大变化，会看到有突起的一个个白点，猜测应该就是NRGY这项科技，基础材料为IGNITE。

细节设计

如果用一句话来描述这双混合动力跑鞋的话：EvoKNIT鞋面提供即时舒适感，袜子般的手感，NRGY提供从鞋跟到鞋头的优质缓冲，IGNITE 提供高能量回弹。而这三项技术恰恰都是PUMA的核心科技。


在造型设计上PUMA HYBRID Rocket 采用分体式中底设计，前掌以及后跟结合上 NRGY 和 IGNITE 技术，提供出色的缓震保护和能量回馈性能，并且在中足加入 EVA 稳定块作巩固，配合上后跟 TPU 稳定片以及 evoKNIT 针织鞋身，赋予更好的包覆性和稳定性。同时一体式的袜套式设计也能够帮助跑者在运动过程中，保持脚踝的稳定。


同时在足弓的两侧，额外添加了硬质的tpu稳定片，tpu稳定片同时与鞋带相结合，也能够增加包裹性及稳定性。


主打配色这款，同时也是上手的这款，采用了前后中底不同色的设计，前脚掌采用黑色基地搭配白色爆米花，后脚掌采用红色基地加上白色爆米花的配色，更有动感和混合动力的感觉。根据PUMA方面的介绍，这款主打跑鞋同样有丰富的配色供选择，看来颜色炸裂是2018年大家对跑鞋设计上的一大突破啊！


在外底部分，PUMA HYBRID Rocket 则根据中底的颜色设计搭配了相应颜色的大底。大底采用全香蕉覆盖，从肉眼看上去，还是有一定厚度的，从以往PUMA跑鞋的性能来看，耐磨性想必也是相当不错。

总结

近年来，PUMA在跑鞋领域不断推陈出新，即便面临来自Nike和Adidas的夹击，也显得游刃有余，底气十足，现如今将手中的三大科技结合而成的混合动力跑鞋，也必能在市场中博得一席之地。
