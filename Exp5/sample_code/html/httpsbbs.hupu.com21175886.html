「真」无线时代中，运动耳机的自我修养

「真」无线时代中，运动耳机的自我修养 由  虎扑跑步区 发表在虎扑装备·运动装备 https://bbs.hupu.com/gear

作为一个不折不扣的跑步爱好者，也接触过不少运动耳机，不过对于 Jaybird 这个品牌，之前真是没有太多耳闻。但拿到这次的 Jaybird  RUN 产品，当目光落到包装封面的那一刻，着实眼前一亮！



这款 Jaybird RUN 包装封面上的两位代言人，一个是Timothy Olson，另一个 Rory Bosio，二人都是美国顶尖的耐力运动员，越野跑领域的双子星。越野大神代言，瞬间提升了对产品的好感度，尽管还没试用。


Timothy Olson


Rory Bosio


当然，代言人的选择只是提升了品牌形象，产品性能怎么样还要自己体验。在开始这款 Jaybird RUN 测评之前，我们先捋了一下运动蓝牙耳机的侧重点，特别是针对跑步，在我看来依次是：音质>舒适度>稳定性>播放时长>其他性能。针对这些要素，接下来我们一一测评。



 音质惊喜 超预期 

对于耳机而言，重中之重是听感，即音质，但是在跑步这种特定情况下，音质可能不再那么重要。因为在跑步时，整个人的注意力集中在协调四肢、调整呼吸和低头看路上，就算再怎么三心二意，剩下的一对耳朵也听不出个所以然来。所以对于跑者，音乐氛围的营造是否恰当，才是选择跑步耳机时应该考虑的指标。



第一次使用这款 Jaybird Run 时，它的音质很让我吃惊，让人感觉这并不是一款运动耳机。果然，在翻阅说明书后发现，它的耳机参数很是优异。在这里不多说专业术语，反正你听后就会觉得它三音均衡，低音饱满，高音亮堂，人声与乐器分离清晰，层次感丰富。特别是在这样的低温寒冬，出门打开音乐就能激活大脑，让身体在律动中点燃运动激情。


跑者还可以通过APP来自定义你喜欢的音效

不过它也有一些小缺点，戴上快走会有迈步产生的震动声，但是跑步时却不明显 ；降噪效果比较好，但这是个双刃剑，有时休息可以用来降噪，夜间在偏僻的地方跑步还是不安全。


 舒适和稳定 跑步耳机的基本素养 

可以说，舒适度和稳定性应该是一个跑步耳机需要到达的首要标准。在长时间跑步中，就算是项链这样细小的累赘都会让身体感觉不适，所以现在运动耳机市场纷纷进入“真无线”时代，减少一切累赘感。



耳朵里塞着这么一对将近15g的小家伙，确实足够轻量，而且有多种耳塞、耳翼选择。在我加速冲刺时，它依然在耳内稳定如初。其实只要跑前佩戴妥当，跑步中根本无须担心掉落，我甚至一度忘记它的存在，大脑仅仅感受到音乐在身体中流淌的律动。


耳翼和耳塞配置了不同规格，所用硅胶材质亲肤舒适


 充电5分钟 畅听1小时 

无线跑步耳机对播放时长的要求还是有的，毕竟没电了是个扫兴的事。单就这款耳机而言，充满电后可以播放约4小时。充电盒在满电的状态下还能为耳机提供8小时的续航，可以说还不错。



充电盒小巧精致，出门放在兜里很是方便

但是如果你想要戴着它挑战一次马拉松，就意味着你需要是400以内的选手，更不用说一些长距离的越野赛事了，所以这是这款耳机的短板。当然这也是整个无线耳机产品的通病，我接触过最长的待机可能也就在6小时左右了，我们只能期待行业的技术革新了。


 傻瓜耳机 操作容易 

Jaybird RUN 的操作也是非常便捷的，一切都围绕两个按键。3秒开机，6秒链接蓝牙设备，按钮并不突出，但是容易触按，在跑动中稍微按动就能实现所需操作。



对于苹果手机用户来说，还有一点值得一提，这款耳机能够连接Siri。在跑步中，点歌换曲接电话是常有的事情，有了Siri这些就方便极了。


 蓝牙连接不担心  8米完全没问题 

相比较有线耳机，蓝牙耳机的连接性也是需要考虑的因素。有时跑步，我们会将手机放在腰后，这可能会导致蓝牙连接不良，音乐中断会直接影响跑步心情，甚至需要停下脚步调整手机位置，或者直接将手机拿在手上。

笔者特地测试了这款耳机的蓝牙连接性时，将手机和耳机分离至8米都不是问题，更别说只是把手机放于身上。

 双纳米防水 严格工艺不留憾 

耳机的防水功能主要体现在夏季和男生，极少女生夏天跑步会大汗淋漓，但大部分跑者会在热天里给自己洒水降温。所以跑步蓝牙耳机在极度潮湿甚至面临水花的直接冲洗，如果防水性不佳，那么直接就byebye了。



作为一个定义为专业跑步耳机的产品，把Jaybird Run与其他品牌的跑步耳机作比较，可能它并不完美，例如它不支持监控跑步数据、APP不能对接国内社交软件等。不过如今跑步APP目不暇接的情况下，你不需要一个用来听歌的耳机还能监控你的步数步幅和心率，它只要做好自己的本职工作就已经足够优秀了。


所以这样一款自我修养良好的运动耳机打动你了吗？
