换换口味，Jordan 3 黑水泥背包 由  風村 发表在虎扑装备·运动装备 https://bbs.hupu.com/gear
前些天打折的时候购于finish line。整体还是很不错的，还原乔三的细节也是很用心。
闲话少叙，上图：


正面呈现一个长方形，有乔三后跟的WiFi信号，爆裂纹以及经典的五个鞋带孔设计。



还有经典的吊牌设计，可惜不是和黑水泥一样的Nike标识。




上部的吊环运用爆裂纹设计，底部则是借鉴了乔三鞋底的圆形设计。



背带下半部一个依然借鉴了乔三的鞋带孔，而另外一个借鉴了黑水泥后跟的Nike老标，而上半部分则是由经典的爆裂纹覆盖。




里面则是很标准的15寸笔记本的夹层和两个口袋便于存放一下小的物件。



底部的拉链来开，能够存放一双球鞋，也是Nike背包比较普遍的特点之一。45.5的乔一亲测可以，但46的黑水泥稍显拥挤。

既然设计来源于黑水泥，那必须和灵感之源来个亲密接触。






后跟的WiFi信号对比。



老屁股。



鞋带孔的设计。




鞋底圆形纹路的经典设计。

这款背包有黑，白两种配色。我独对黑色情有独钟。

背包借鉴了乔三许多很经典的设计，细节比较丰富，容量也足够实用。十分推荐。
