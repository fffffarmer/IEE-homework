汉密尔顿：不明白最后策略怎么会变成这样的，我们真是自找麻烦 由  风吹过山 发表在虎扑·F1赛车场 https://bbs.hupu.com/f1
   “我不太确定策略最后怎么会是这个样子的，我们让自己陷入了如此的困境。”
    第二次进站以后，汉密尔顿落后莱科宁12秒，汉密尔顿说：“12秒太多了，我追不上他了。通常来说我认为车队有能力做得更好，这（策略）有点让我吃惊，但你不可能每次都有正确的策略。”
    沃尔夫也质疑了车队的策略，“我们必须重新思考一下是否有一些地方我们本可以做得更好。”
    汉密尔顿继续：“从我们在第11圈进站开始，我就知道我们必须2停了，我们真的是自找的（陷入如此艰难的境地），当我出来以后，我落后莱科宁9秒，我必须去追击他，轮胎在此过程中被用尽了。莱科宁进站换上新胎以后马上就开始追近了我。之后我2停了，出来以后落后12秒，这让我们陷入了如此的困难即使想竞争都很困难，我不是太清楚车队为什么选择这个2停的策略，但事后总结来说，可能我们应该选择用紫胎起步。起步的时候，我和莱科宁的速度明显不一样，我明显比他慢，他的轮胎比我软，我没办法在第一个stint跟上他，这是艰难的一天。”
     “恭喜莱科宁，我知道他上次赢得分站冠军已经是很久以前的事了，但是他今天驾驶地太棒了，我为他感到开心。同时也恭喜维斯塔潘，从第18位起步，他完成了一场伟大的比赛。”
     “作为一只车队整体表现来说，我们今天并不出色，通常来讲我们不是这样的，所以我们会回去看看战术板，重新部署，我们过去曾经有过很棒的连续性表现，但是这个周末我们就像是打了个双柏忌。”
