这个综艺节目火了是因为之前有几名韩国女主播在直播中玩过一次“夹大腿”的游戏，当时引起了关注，然后被众多综艺节目列为大长腿妹子们的重头戏。“夹大腿”其实是考验大腿力量的一种测试。

游戏的规则是两个面对面的坐着，然后其中一个妹子夹住另一个妹子的腿，然后听到开始的声音后，被夹住腿的妹子要用力撑开夹她的腿，而夹住腿的另一个妹子就要用力夹住，在这个时候，谁的大腿更有力量，或者功夫最好就显而易见了。

如果一个妹子没夹住她的大腿，被另一个妹子用力撑开了，那么她穿的短裙就会暴露她隐藏在暗处的那些小秘密哦。

其实这个综艺节目中，最大的看点并不是谁的腿最有力的，而都是在期待着那个被夹住的妹子，让她用力撑开那双美腿，让那些勾人的小秘密暴露在大家的眼前。
