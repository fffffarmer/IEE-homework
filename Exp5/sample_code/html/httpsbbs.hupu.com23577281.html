首度联手！Jordan Brand发布巴黎圣日耳曼足球俱乐部联名系列单品 由  狮王LEO 发表在虎扑装备·运动装备 https://bbs.hupu.com/gear

Jordan品牌与巴黎圣日耳曼足球俱乐部达成合作，这也是Jordan品牌首次与足球俱乐部的正式合作。“Jordan品牌和巴黎圣日耳曼俱乐部都在体育和时尚领域占据着独特的地位，因此双方此次合作称得上是天作之合。”迈克尔·乔丹说道。

Jordan品牌 x 巴黎圣日耳曼合作带来的联名系列将涵盖实战及休闲在内的全方位产品体验。整个系列包括两件特别版的比赛服、球鞋及配套服装。比赛服以Nike Vaporknit Match球衣为设计基础，采用简约的黑白配色并搭配红色作为点缀。这两件球衣将会做为俱乐部的第三和第四出场比赛服。守门员的球衣则大胆采用了红外线红为主色，以致敬Jordan品牌的这个经典配色。

同样以黑白两色为主，这次联名合作的鞋产品和其他服装系列采用了与比赛服相近的配色方案，并同样加入了红色元素，它不仅是Jordan品牌的经典配色，也同样是是巴黎圣日耳曼球衣的主要配色之一。此外，设计师还别出心裁地将两个品牌经典标志做了结合：把Jumpman标志放置在巴黎圣日耳曼俱乐部徽章的中心并将其融入在联名产品上，成为此系列的一大亮点。

Jordan品牌 x 巴黎圣日耳曼联名系列将于9月15日正式发售；巴黎圣日耳曼球员也将于9月18日穿着Jordan品牌球衣亮相欧冠赛场。
