－－－－－－－－－－－－－－－－－－－－－－－－－－－－－


        二十二年，68款市售发行的配色。相信这其中的某个配色也许和大家有着不同故事，它可能见证了你年少时的一段经历，也可能陪伴你渡过了一段难忘的岁月——那双你省吃俭用，花费了一个学期零花钱买到的第一双Air Jordan 11现在还珍藏着吗？那个曾经陪你一起排队买鞋的小伙伴现在还常联系吗？


        我相信对于所有喜爱Sneaker文化的朋友来说，Air Jordan 11所代表的并不仅仅是一个金额数字这么简单。每一次的发售，每一款配色都凝聚了鞋迷们无价的付出和内心的渴望。


        希望通过这一次的整理，能够对大家在今后查询和收集Air Jordan 11时所有帮助。
