1，我碰到他们当时是6月份左右，邹雨宸恢复情况还可以，每天都在做伤腿的脚踝训练和腿部力量训练和一些体能减值，因为膝关节受伤所以需要提高踝关节力量，当时邹雨宸体脂含量还是很高的，这个教练会悄悄的跟体育医院的医生说你看他这个肚子，肥的不行，不过每天训练还是很认真的，汗出的很多
2，这个教练听说是从台湾找来单独带他的，效果应该还可以吧，训练还是比较有针对性的，碰见的几次基本都是踝关节的训练为主，应该是在室外进行膝关节的力量训练。
3，后来没多久邹就去国家队报到了，因为他让人来医院把他之前练习腿部力量跳跃的垫子拿走拿去北体了。而之前我一直在用这个垫子做恢复训练，当时是6月底
4，这个视频我7,8月份就在抖音看过了，一眼就认出这个教练了
再后来就没碰见过他了，我还纳闷按理说恢复的还可以，怎么到了国家队就没消息了，找媒体朋友打听了才知道又受伤了。。。
真的是为他感到遗憾，邹的体格真心很好，不受伤的话绝对是中国男篮的第四内线的首选，甚至冲击NBA也有很大机会
真的是万分遗憾，祝他早日康复，重返赛场
