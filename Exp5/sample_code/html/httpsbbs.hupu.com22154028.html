    拆卡永远十拆九肿，专收才会细水长流。这句话同样适用于上组。前段时间上了几组imm和手提，既然是高端系列，价格肯定不低，谁让我那么喜欢西蒙斯，想搏一下76人。后来证明，想法确实太过于优质，即便很幸运的随机到了76人，还要看能否拆得出西蒙斯的卡，概率低之又低，而这次的结果，直接把我干退坑了。于是决定，从现在开始到年底，不再拆卡，不再上组（这个月的国宝也果断无视！），只偶尔收一些小散包和小卡就OK了，专心弄卡淘回血。来吧，正在清理各个群的累积，这次就先晒一下在lucky家上的几组imm和手提的大肿成果

第一张是来自1617immaculte collection 凯尔特人2轮新秀，迪米特里厄斯 杰克逊的新秀横版双球衣，话说这哥们前一段的新闻是被我破费签约，而现在的东部半决赛，正好是对阵老东家凯尔特人，不过并没有进入季后赛轮换名单，不出意外，应该最终也会成为边缘球员



第二张来自让我最为心痛的手提箱系列，上了两组，竟然只随机到庄神的板签，编号7/15，不说了，全是泪



第三张算是稍稍有些安慰了，1617immaculate系列，老汉的JERSEY NUMBER PATCH，编号11/23，切的应该是球衣背号，看了下ebay，截标价通常在1000-3000RMB之间，算是比较好的老汉的卡了，先留着吧
