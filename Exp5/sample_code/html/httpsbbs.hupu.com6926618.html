巴意盎然@匹克巴蒂尔八代Peak Battier VIII实战评测 由  百人斩少年 发表在虎扑装备·运动装备 https://bbs.hupu.com/gear

匹克巴蒂尔八代的广告铺天盖地，在央视展播、网络视频广告中JRS一定看了不少，包括球鞋杂志上面也都有相关的评测软文展示。我嘛喜欢巴蒂尔很久了，虽然不是最喜欢的球员，但是巴蒂尔的球场风格、睿智的头脑都很欣赏，虽然之前的七代球鞋没有上脚过（倒是贩卖过一批银色的巴蒂尔五代OTZ），不过这双在广告以及它本身的姿色诱惑下购入的第八代球鞋给我带来的感觉还是很不错的。


在评测之前说一下自己的球员类型，以前都写Anthony Carter，老家伙已经退役了，再加上自己的玻璃人属性、苦练之后无限精准的外线投射，现在只能做到外线组织、突破分球、空位三分、关键球还有扛站在外线的大吨位兄弟，总体说来就是失去速度、没有篮下终结能力的超弱化版本Chanucey Billups......

开始评测！

这双鞋实际上就周六晚上和老伙伴们在内场高强度（准专业队水平）打了两个小时，上脚的第一感觉不是特别好，前排的鞋带太紧了，脱下来之后松了鞋带，把脚踝附近的三排鞋带系紧之后就特别舒服啦。这种舒适性是什么呢？跟随PEAK的一贯性，球鞋的楦型很适合亚洲人的脚型，近期入的Nike id Zoom Five B和好久之前买的李宁Defend Everyday就是中掌过紧，我都是把鞋垫中掌高出来的部分修剪掉一部分，把鞋舌与鞋体之间的松紧带剪掉穿起来脚才能不疼（毕竟有玻璃人属性），这双巴蒂尔八代就不会有这样的现象，不过一定要把鞋带的宽松度放到适合自己的脚才可以，脚瘦的朋友就可以缠紧一点。


不过嘛，从照片里面大家也能看出来在实战之后，鞋面回头顺着鞋带洞方向的折痕，而且是只在两个小时内就积累下了这样的折痕，这部分鞋面所用的材料不软不硬，但如果经年累月下来绝对非常难看，说不定会有漆皮一样断裂的现象出现，这个算是美中不足。

 

大量使用漆皮材质没有使得鞋面变得坚硬，这个要归功于球鞋各处皮面的接洽处设计的很合理，球鞋前面的Shane Battier字体也很好看，这几天巴蒂尔自己吐槽自己说原本的姓氏是“Battle”以后写上战役的话，会更容易让人产生硬汉、铁血的印象。





柔软度极好的硅胶鞋垫加上三重缓震技术，巴蒂尔八代的整体脚感非常舒服，缓震性也不会像之前写过评测的洛瑞二代那样之后后跟才能感觉到，三重缓震技术成熟之后给人带来的感觉是全方位的缓震，相反由于球鞋设计的方向、球员球场风格的限制，启动时候这双球鞋带来的助力没办法让人印象深刻，不过好在本身就是为了巴蒂尔这样的无球跑动射手而设计，喜欢快速启动的朋友更适合Hyperfuse、D·ROSE或者匹克闪电这样的。







外底性能方面，这双匹克巴蒂尔八代做的很不错，前掌内侧后掌外侧使用了大面积的凸面人字纹、前掌外侧和后掌内侧则使用了横向的平面闪电纹，在向前发力的时候能够牢牢抓住体育馆久未打理的木地板，侧向横移的时候也不会有滑出去的不踏实的感觉，不过个人觉得这是双内场鞋，外底使用的是比较柔软的橡胶，塑胶场地也没问题，面对水泥地可能消耗的就比较快啦。


虽然鞋面布满了网洞，但是透气性并没有Hyperfuse之类的材料那么爽快，不过对于北方的冬天而言，倒是好事儿。匹克巴蒂尔八代的保护性不容多言，中帮球鞋、漆皮面料鞋面，加以后跟整块的皮料内包TPU，只要把靠近脚踝的鞋带全部系好，不违背这双球鞋设计的初衷正常使用，应该不会造成任何运动伤害。


综上所述，个人认为PEAK Battier VIII在新赛季众球鞋的性价比角逐中不容小觑，作为国产球鞋延续时间最长的球星个人款式，也势必拥有更多的拥趸。虽然只是短短实战了两个小时，不过足够让人对它的稳定性印象深刻了，如果你需要一双长期的内场训练、比赛用鞋，而你的风格也不是突破一波流的家伙，那么这双鞋再适合你不过啦。


喜欢的朋友记得点赞并且推荐哟~

还有我是在TB搜货号直接找的最低价，感觉往后300左右入手很容易，不急着买的兄弟可以等到春节左右，到时候应该会有更多配色上市的。
