      热火队2013-14赛季完整赛程表

北京时间10月30日，NBA新赛季开打，热火主场迎战公牛。

11月2日，热火客场挑战篮网。

然后热火在12月11日碰到不容小觑的步行者。

12月26日，圣诞大战，热火客场面对老对手湖人。

1月10日对阵尼克斯，从尼克斯起，热火将连续打6个客场，都是东部的对手，
接下来的是篮网、奇才、76人、山猫和老鹰。

从2月7日起，热火开始西部之旅，连续6个客场，对手分别是：快船、爵士、太阳、勇士、小牛和雷霆。跟东部的相比，这次旅行非常艰难。

1月27日，与总决赛对手马刺狭路相逢，3月7日，还将在圣安东尼奥再战一场。

3月17日，热火才首次碰到焕然一新的火箭。

最后热火的收官战很轻松，4月17日，主场对76人。
