易快BOT抢鞋软件伪评测：想靠此赚外快？太天真了，先看完这篇文章冷静冷静吧（一） 由  那就这个 发表在虎扑装备·运动装备 https://bbs.hupu.com/gear

为什么说是伪评测，因为此文完全是笔者本人一家之言，更多的是抒发个人的情绪在里面，并没有很多干货值得分享，唯一的目的就是希望看到此文的JR们擦亮一下自己的眼睛：旁门左道不可取,摆正心态，老老实实陪跑，说不定哪天就中了。（家里有矿的，时薪过万的，铁定认为此BOT就是真理的JR们倒是可以试一下，除此之外不建议对这玩意儿有一毛钱的投入）。

BOT抢鞋软件，经常抢鞋的JR们肯定对这个概念不陌生。正是因为每次SNKRS发布新款球鞋潮鞋的限量限购，导致大量球鞋爱好者们都成为陪跑族，所以有一些“身怀绝技”的个人或者团队开发出了抢鞋插件或者软件程序。

本文不讨论BOT软件在公序良俗或者社会道德层面上所引发的争论，凡事存在即合理，既然这东西存在，那它就有存在的价值或者意义，区别无非就是看受益方是哪些人。

BOT抢购软件在国外比较盛行，近些年来已经形成了相当的产业规模，从上游的开发团队到中下游的大贩子再到下游的散货倒爷，基本上都有各自领域的职业潜规则，甚至还有一些专门的评测团体或个人专门以此进行引导性推广。

说回到国内的bot市场，各种所谓的私人BOT，团队个性化开发BOT层出不穷，但其中充斥着大量的掮客甚至鸡鸣狗盗之徒在不明真相的群众里薅一把羊毛就迅速跑路，导致国内的BOT情况五花八门，乌烟瘴气，毫无行业规则。（虽然干这个的本身也上不了什么台面）。

笔者本人对于BOT或多或少有所了解，恰好笔者本人也入手了这款bot，于是恶心的故事就开始了。

 

故事的主角讲的就是国内的一款看似很火的BOT软件，名叫“易快BOT抢鞋协议软件”。希望通过此文给一些蠢蠢欲动的JR们提个醒：如果这玩意儿真的如这个开发团队所言的那么灵，那国内的倒爷们何必还花大价钱去跑海外渠道？若是坛子里有已经购买的JR，可以对照一下笔者所言是否属实，也欢迎跟帖回帖，共同揭露此BOT团伙儿的坑人套路。

 

首先定下一个结论：总的来说此BOT是没用的垃圾。接下来我几个方面进行论证为什么称这个软件是垃圾一个。

一、产品团队简介

二、软件宣传功效及实际使用体验

三、团队售后服务


一、这是个什么团队？

该团队的推广主要是通过百度贴吧进行投放的，比如耐克吧，球鞋吧，抢购吧等类似的贴吧进行投放，（玩贴吧的JR应该都知道，上面提到耐克吧球鞋吧早已沦陷，基本上被PT鞋的假货卖家所控制，该团队把广告投放到此类贴吧，可以想象此团队本身是什么样的定位），发帖者也很明显是刚申请不久的贴吧小号。如下图：




继续挖掘，还能搜出来此团队竟然还有自己的官方博客网站，文中的“坑翁”正是贴吧里联系QQ的那个人，如下图。在他们官网贴出的“战绩”里能看到，一些大家陪跑的款式，他们团队竟然能用区区几十个几百个账号就能轻松撸获几双甚至十几双，看着很逼真很眼馋吧，勿急，请接着往下看。





甚至文章下面还有“用户”评论，但是明眼人一看就知道只都是他们团队自己的小号在给自己宣传贴金而已：




 

他们有若干个QQ群，有售前宣传群，付款过后会把你拉入他们的所谓的售后群来彰显他们一条龙服务。但是群主和管理员基本上都是这几个人，就是这么一群人，其中昵称为飙升的这个人就是上图博客里的坑翁。如下图：



值得注意的是，这几个群里都有他们安排的演员，也就是托儿，假扮是购买过他们BOT的顾客，每逢SNKRS发鞋尤其是陪跑鞋款，这几个人总会炫耀自己的“战绩”，自己又中了多少双多少双这样的，其中有一个昵称为疯子的人，尤为明显。这是此人的头像



每次发完鞋子之后，群里买过BOT的顾客，都会怨声载道，没有一个中签的，唯独这个叫做疯子的托儿，每次都炫耀自己中了多少，然后是感谢易快，真厉害之类的话语。每次都无一例外。笔者入手此BOT时间已经很不短了，从未中过一次，哪怕是哪些不热门的款式，甚至有时候自己手动用手机抽很随意的就中了，但使用该软件批量抢购从未中签过任何一双。每次找作者团队质询，都会以“你的账号不行”，“你的IP有问题”等等其他理由给搪塞过去。

   这是个什么样的团队？我看也就会玩一些这样的把戏罢了。

bot的话 国内的可以说都是垃圾。

目前排名第一的毫无疑问是Cyber 第二是EVE 但是eve最近的的表现一般，Serena x OW 被PD给虐了 supreme方面被新星Sieu虐了，但是更新的快，地位暂时还是没办法撼动。
剩下的像PD tripio kicksmoji wop 这些 算是第三梯队吧 各有各的好 但都不稳定，而且很多是snkrs或者adi专属。

目前比较看好的是backdoor的bot adisplash在这次yeezy表现不错 虽然发售刚开始出现过不了splash的现象，但是反应非常快半小时内更新两次，成功解决了问题，我加了几双进购物车 但由于前一天买了很多双700了 就都没有付款。

最后说一下重点 这些所有的bot都是sold out了 国内几乎是不可能买到的 而且因为墙的限制，在操作过程中会有很多问题，还要租server 要买proxy 要付费进好的cook group，这三项的开支每个月大概200-300美金。
而且二手的bot套路多，好的bot价格非常高，比如cyber 转卖价格在3000美金左右。

总之 bot的水很深 如果不是专业的，没必要碰。
