        此轮比赛中，各队外援在进攻端可谓予取予求，共有四位外援砍下40+，打出爆炸数据同时也为球迷奉献了一场精彩比赛。其中青岛雄鹰乔纳森-吉布森揽下47分8板，深圳烈豹肯尼-伯顿砍下42分5板，北控皮埃尔-杰克逊轰下46分6板6助，浙江金牛马库斯-邓蒙独得41分6板7助。

        你如何看待首轮比赛各队外援的爆炸表现？谁能延续上一场的火热表现继续打出40+？
