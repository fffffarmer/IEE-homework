【四十不惑】诺维茨基的Air Pure Profile Max 由  gt228 发表在虎扑装备·运动装备 https://bbs.hupu.com/gear

    今天是2018年6月19日，诺维茨基40岁的生日。尽管由于时差原因，当地时间目前还是18号的夜晚，但几小时后，相信全球的独行侠球迷都将开始以各种形式为偶像庆生。
    其实按原计划，这一帖本想发一下王治郅的Air Max Elite，但进入六月后猛然记起这个月有他的生日，于是调整顺序将Air Max Elite留到下次，今天就蹭个热度，温故一双诺维茨基早期的成名战靴： Air Pure Profile Max。
    

    之前诺维茨基的鞋已经整理过若干双，其余几双在他退役前还会陆续发完。以前在一些帖子中也曾提到，他本人对穿什么鞋并不在乎，耐克也从来没给他出过签名鞋，不过从这双2001~02赛季的Air Pure Profile Max开始，耐克给诺维茨基的待遇有所提高，将他设定为这类中高端实战款的“主要代言人”。不知道现在还有多少人能记得它，当时国内的售价是930元，除诺维茨基以外 国王的佩贾、太阳的便士哈达威等人都有上脚。

    若是单纯从篮球鞋的角度来看，Air Pure Profile Max无论定位，配置，还是历史地位，与前面那几帖相比都有些相形见绌，可写的东西其实并没有太多。但它在我眼中依旧有一种特殊的存在感，能帮我串连起许多2001~02赛季发生的故事，因此这篇帖子涉及的内容恐怕也将比较杂乱。
    对小牛球迷来讲，那是十分美好的一年，情况稍微有点类似于这个赛季的76人，季后赛也打到了第二轮，球队的未来充满希望。诺维茨基也正是在那个赛季跻身联盟的一线球星。


   
    那年我刚上初中，对篮球运动员的印象只是“高个黑人，每个长得都差不多”，知道名字的只有乔丹，甚至连姚明都还不认识。但已经开始初步接触篮球，那会在操场上，如果哪个同学投篮很准，或是喜欢“骑马射箭”，那么在他进球之后，一些比较皮的同学总喜欢眉飞色舞地冲他喊一句“嚯~~~你是诺维茨基吗！”。当时真的完全不懂他在说什么，但这个名字挺顺口的，我一下就记住了（不像我妈，老人家至今仍叫他莱温斯基），并且心中默默标注：这个人＝投篮准。
    后来在班里看同学的杂志 才第一次知道诺维茨基原来是个金发碧眼的长头发白人，与另一位白人搭档，掺杂在球场上一众黑人当中格外显眼，瞬间给我留下了深刻的印象。然后从2002~03赛季开始看懵懵懂懂地看NBA转播，小牛队狂轰乱炸的比赛风格很容易吸引刚刚入门的菜鸟球迷，14连胜的开局让我认定它是支强队，加之自幼对于蓝色莫名的偏爱，很快便成为了一名达拉斯的铁粉。记忆中与纳什芬利满场狂奔的德克还是意气风发，挥斥方遒；而似乎就在转眼之间，当年那个二十出头的德国小伙子，今天已经整整四十岁了。



     也是在2001~02赛季，小牛在库班的带领下完成了蜕变：将主场搬到新的球馆（美航中心），同时更换了新队标，设计了新球衣，主场的白/海军蓝配色一直沿用至今。尽管在一些主流媒体的“最美球衣”排名中，达拉斯往往只是被排在比较中游的位置，但在我眼中 它无疑就是最好看的......
    白色的比重几乎占了90%以上，与袖口的海军蓝形成强烈的对比，再以宝矿蓝作为细节线条的点缀，使这件球衣整体看上去很干净。胸前“DALLAS”的字体设计散发着那种21世纪初期特有的现代感，记得上学时选修过一门字体设计的短期课程，其中一个关于“审美疲劳”的基本规律记忆犹新：多数情况下，如果一个品牌的产品使用寿命很长，比如电视，冰箱，空调这种动辄就用个十多年的东西，那么它的字体设计则应该尽量简洁，让人即使每天都看很多次也不会有腻的感觉；而电影，游戏，唱片封面中那些文字则要想方设法弄的很燃 或者很有个性，以给人一时的视觉冲击，但同时人们对它的新鲜感也消耗的很快。
    球衣的领域似乎也是这个道理，街头篮球那些花里胡哨的东西好像很炫酷，但实际并不禁看；而NBA比较致力于打造球队文化和传统，像湖人，凯尔特人这种有历史底蕴的豪门从不轻易更换Logo，球衣的设计也往往多年不变。相比之下小牛的历史虽然没那么长，但库班接手之对这方面的经营十分用心：多年来除了拉丁之夜，圣诞大战这些特殊情况，达拉斯的主场球衣从来不变（上赛季换回耐克之后不再规定主场必须穿白色，所以变的有点乱......），客场也几乎很少推出新款，并且大多数还只变配色不变版型。2011年夺冠的加成 让这套球衣注定成为球队历史上经典，现阶段恐怕也还会长期使用。   


    17年前的这双Air Pure Profile Max也成了诺维茨基脚上第一双与新球衣搭配的战靴。耐克给他的配色与主场球衣放在一起简直浑然天成，白蓝两色的比例都如此一致。这双鞋本身独特的设计 造成内外侧色彩强烈的反差，是我心中这件白色球衣的绝配，其完美程度甚至不亚于同时期纳什的Air JetFlight。

    
    不知各位第一次看到诺维茨基都是什么时候，和我年龄相仿的人可能还有一点印象：一贯以长发造型示人的诺维茨基曾经一度把头发剃的很短很短，基本只剩薄薄的一层。2011年杨毅讲那个《大话美职篮》的时候还提到过当年诺维茨基剃头这段故事，估计是从王治郅那听来的。我后来根据他讲的内容查了查赛程和照片，发现同样也是Air Pure Profile Max那个赛季发生的事：当地时间的2002年1月15号，小牛客场116比107战胜了老鹰，那场比赛诺维茨基的脑袋还是这样的 


    然而到了1月17号打灰熊，他就变成这样了......


    造成这一现象的幕后推手就是下图这位干翻德克的老人：唐-尼尔森。



    老教练早就觉得诺维茨基的长发会影响投篮的视线...于是那场打完老鹰 回到更衣室 便把德克叫来说“头发太长了吧，要不剪剪”。诺维茨基似乎是不太愿意，不过嘴上还是说“行，回头我找个地方剪剪”，心想就这么糊弄过去，说不定老头过几天就把这事给忘了。 没想到老头这时来了一句“还找什么地方啊，现在就来吧~”随着一声令下，瞬间冲出来几个队友就把诺维茨基给按住了......估计是布拉德利，拉弗伦茨他们，接着直接就押送到卫生间的洗手池...这时诺维茨基挣扎着从镜子里看到后面走过来一个人，手里拿着一把剃头推子......
    那便是诺维茨基曾经的队友：墨西哥人纳胡拉，也是现任NBA发展联盟德克萨斯传奇队的主教练。据大郅回忆，纳胡拉相当注重自己的形象，每场比赛前都要对着小镜子把头发梳理的油光水滑，因此大郅还爱称他为“发哥”。所以平时自己预备个推子也是情理之中。赶上他手脚也太麻利了一点，诺维茨基还没怎么反抗，现场已是“纷纷鳞甲飞，顷刻遍宇宙”，一头金发就这么改了板寸了。
    折腾完之后，老尼尔森对已经懵逼的德克露出了满意的笑容“不错，这回轻易挡不住视线了！ 你去把那些头发捡起来，咱不能白剃了”。然后让诺维茨基把自己的头发分装在45个小袋里，签名，并且每袋以100美元的价格出售，将这些收益捐给达拉斯一个治疗乳腺癌的基金会（老尼尔森的妻子当时就患有乳腺癌）。据说那些头发十分抢手，眼看就卖的差不多了，剩下最后5袋的时候老尼尔森又说，这5袋就别卖给普通球迷了，找个有钱的吧。于是以1000美元一袋的价格把这5袋卖给了小牛队的上一任老板：小罗斯·佩罗特。（佩洛特家族在德克萨斯可谓富甲天下，有兴趣的朋友可以去了解一下，他的父亲老佩洛特被称为IT业最具传奇色彩的人物，并且在上世纪90年代将美国政坛掀得天翻地覆。）5000美元对他来讲实在算不上什么。于是这45袋头发最终卖了9000美金，事后老尼尔森把德克叫过来说“看见了吗，这算是我给你指了一条致富的路......”

    100美金的价格我个人感觉还比较合理，就是有点不太卫生......作为一个有洁癖的人，我宁愿选择花100美金去买一双Air Pure Profile Max......
    这话千万别信，我自己说着都亏心......但没办法，从小就不太会写作文，不这么说我实在想不出其他的手段 来把话题重新扯回到这双鞋上...... 毕竟它表面上还是本帖的主角，接下来还是仔细看看这双Air Pure Profile Max。
    本世纪初那几年，耐克篮球鞋每年都会推出几款比较偏实战的，后掌往往是Max Air的中高端篮球鞋。我个人习惯把它们看做是一个系列，这双Air Pure Profile Max便是早期典型的代表作，它们的外观设计大多比较规矩，未必会让人眼前一亮，但普遍都十分耐看，鞋型和质感我认为也远远好于如今这些同级别的实战鞋。鞋带往往选择了实用的快速系鞋带系统。


    同时喜欢在一些细节部位点缀一些金属质感，这点在最后一两个鞋带孔的设计上表现得尤其突出。印象里除了这双以外，与它时间差不多的Air Max Chosen， Air Uptempo Sensation，Air Max Mythical，Air Uptempo Maximum等等无一例外都是如此。这款Air Pure Profile Max及上述的一些其他款式中都带有“六道斜线”的MAX AIR标志，也是属于那个时期的一个特殊记忆。



    内外侧鞋面的反差是这双鞋外观设计的最大亮点。若是单从外侧来看，第一眼恐怕不少人都会觉得这双鞋设计的未免也太无聊了，仅仅是一块大大的皮革而已，但不少人翻看到内侧便会收回之前的偏见；再多换几个角度则发现这双鞋其实并不那么简单，还真有点意思，只是需要从整体观察才能体会出它的美感。


    我个人尤其喜欢前半身 内外侧过渡的部分，设计师的企图似乎是先保持内侧的神秘感，然后有意地将你的视线从外侧向内侧引导，从而形成与外侧的强烈对比，这才明白它的外秀慧中，对这双鞋的好感度顿时上升一个等级。



    内侧将两种质感的皮革合理地拼接，大面积采用了这种比较浅层的磨砂皮面，上面规律地压印着横竖的“短四杠”，给我感觉可能是有意在与外底纹路相呼应。


    当时耐克喜欢在篮球鞋上特意标出“NIKE BASKETBALL”的字样，可惜印象中已经许多年没见过了......感觉它现在倒是有必要回归一下，因为如今这些设计已经让人越来越难分辨它是篮球鞋还是足球鞋了...


    这个轻质聚亚安酯中底设计在当年也是被许多球鞋通用的，反正不是中底便是外底，基本就是几样式个来回组合着用，某个角度上讲这算也是一种偷工减料，比如这双Air Pure Profile Max，它的中底与Air Jet Flight是完全相同的，只是把气垫改为了开窗式；而外底纹路则又与Air Uptempo Sensation完全一样......这么一来也导致那几年同等价位的许多球鞋脚感都大同小异...幸好还都算比较舒服。 



    Air Pure Profile Max的外侧鞋面使用了大面积的人造鞣皮，网状的双行车线将它分割为数个菱形是这双鞋外侧最明显的特征。


    皮面本身有一定的厚度，摸上去有点像一种皮质的沙发，十分有质感。虽然我从未在一些资料上查到过这双鞋的设计灵感，但近几年每当我看到这个鞋面，都强烈地觉得它很像一个什么东西，但当时又想不起具体是什么，可谓是百思不得其解......


    直到2018年新年那几天，一位朋友送了我这份礼物。把它打开的那一瞬间，我心中一切的困惑全都消失了......




    不错，正是这种被捆绑着的火腿肉......不知道当年那位设计师爱不爱吃，或是他家里有亲戚就生产这个...我认为并非完全没可能...否则他怎么能将这种设计如此完美地还原在了球鞋身上...


    再说优质的火腿为什么要用绳子绑起来，其主要目的是为了让猪肉保持一个稳定的形状，并且可以适当将其束缚的狠一点，这也可以保证最终火腿肉质的紧实。他给我这个镰仓的猪肉与鹿儿岛的相比还有些不同，在口感上体现的也比较明显，我个人是比较喜欢直接切片，配以奶酪之类的东西，很容易吃饱，也省事。


    或者就是把它放在炒饭里，但就要相对切的厚一些，否则炒的时候容易 
    这TM又说到哪来了.....


    真接不上了
    

    脚踝，脚踝填充物比较厚实，毕竟这双鞋主要是提供给锋线上的球员，保护性还是相当重要的一环。把它穿在脚上，的确能实实在在地感受到它的厚度，并且值得一提的是，这双鞋本身还不是很重，可以说完全不会影响你奔跑的速度，只是透气性就不予评价了。


    后掌缓震采用开窗式Max Air气垫。由于本人体重原因，对Max Air基本踩不出什么感觉，但 Air Pure Profile Max和Air Max Chosen是我印象中少数的几款能在后期明显感受到Max Air弹性的球鞋，前掌缓震也不错。


    硬质橡胶外底与外置中掌TPU托盘也是当年的经典组合。外底采用横竖排列的格状纹路，抓地表现基本可以说中规中矩，但印象中并不十分耐磨，加上当时基本都是打室外塑胶场地，一两个学期基本也就磨平了。如果是室内，恐怕这双鞋会有更好的表现。



    后掌最下方这个耐克的Logo倒是意外地成为了这双鞋被人记住的一个点，当然，其中也有一部分人记的是Air Uptempo Sensation...


    再说到当年的诺维茨基，其实好几张经典的照片中，诺维茨基穿的都是这双Air Pure Profile Max，只是被挡住了而已...比如下面这张 小牛当年的“国际纵队”，左一穿着BB4的小伙子就是上文提到过给诺维茨基剃头的墨西哥人纳胡拉。那个时候国际球员在NBA还没有得到如今这般的认可，小牛队在这方面可谓是联盟的先驱，这跟库班，老尼尔森以及管理层共同打造的环境有直接的关系，据说对他们国际球员都十分的照顾，并且鼓励他们打出自己的风格。
    也正所谓“前人栽树，后人乘凉”，从某种意义上来讲，正是当年诺维茨基，帕克，吉诺比利，姚明他们职业生涯的成功，才推动了之后NBA在全球范围的人才选拔。如今的字母哥，波神，西蒙斯 恩比德以及3天后即将被某支球队选中的东契奇都着实赶上了一个美好的时代。


    下面这张照片中的阵容就比上面那张强大太多太多了...上学那会总是幻想能打造一支绝对无敌的超级球队，比如下面这五个人再加上奥尼尔，姚明，纳什基德艾佛森，麦迪卡特皮尔斯......简直可以碾压全世界......以上这些人，现在只剩下诺维茨基和卡特了。


    那是2002年费城全明星时期的媒体照，也是诺维茨基和纳什职业生涯第一次入选全明星，而且西部的主教练正是小牛的老尼尔森，可见那个赛季他们的状态有多么火热。从照片中也能看出，经过一个月的时间，诺维茨基的头发也长了一点......
    那届全明星是本世纪目前为止 最后一次没有统一东西部队服的全明星，球星们都是穿着各自母队的球衣，场面上看着倒挺有意思....耐克也为参加全明星的国际球员们在球鞋上特意绣了各自祖国的国旗。


    那场比赛我后来看过录像，记得下半场开始好像两边就有点认真打了，算得上十分精彩的一次全明星。而我个人更关注的便是比赛中出现的那些球鞋，其中许多都是我非常中意的款式，条件允许之后最终也都如愿收入了手中。



    除了诺维茨基和佩贾的Air Pure Profile Max以外，比如我个人最爱的 纳什的Air JetFlight。



    基德，小奥尼尔，布兰德，戴维斯的Shox Stunner。（https://bbs.hupu.com/18123627.html）


    下一帖会细说的 皮尔斯的 Air Max Elite。


    邓肯的Air Max Duncan II。


    科比的 The Kobe II。（https://bbs.hupu.com/18368965.html）


    韦伯的Dada CDubbz。

    弗朗西斯的 X-BEAM 。（https://bbs.hupu.com/18019093.html）


    沃克的PRO MODEL 2G。（https://bbs.hupu.com/21142449.html）


    佩顿的Air Zoom GP IV。


    以及乔丹的Air Jordan 17 Low。



    说到这又想起一个事：三年前，在ebay上曾经出现过一双Air Pure Profile Max，居然就是照片中这双有国旗的，诺维茨基全明星赛的GU，上面还有纳什，皮尔斯，邓肯等等几乎当年所有耐克旗下球星的签名（当然也包括诺维茨基自己的，但没有乔丹），这自然让我兴奋不已，但一口价是6500刀，面对这个数字，我当场就理直气壮的怂了。
    然而我看它一段时间之后并没有卖出去，卖家把形式改为了拍卖，起拍价3300刀。坦白的说，在我看来这双鞋不止值这个价，但3000刀是当年我心中的一个坎，超过3000的基本不会考虑，可心中又很清楚那双Air Pure Profile Max一旦错过，恐怕此生便没有第二次得到的机会...于是打算等等，如果一直没人出价，看它是否还能再便宜一些。
    但情况并没按我希望的方向发展，没过几天就有人出价了，这个混蛋让我一下陷入了被动的局面，开始了极其强烈的思想斗争...当时还与多位朋友商讨过入手的可能性，其实说白了就是想多找几个人鼓鼓劲，给自己一个下决心的借口。
    最终，在那双鞋落锤的当天，他们纷纷跑过来问我“那双鞋结果怎么样了？”
“你别说，结果我TM一咬牙一跺脚！”
“你出价了？！”
“我还是出家吧”  

    我当时如果真的入了手，今天这个帖子哪还用得着写什么火腿...数签名就数不完了......身外之物...有缘无分，实不可强求......最终出价的也就那一个人，在眼睁睁目送它被拍走之后，其实没过多久也就释然了......
    这次经历之后 我对买鞋这件事的态度都发生了一些变化...我从前常常幻想要把所有好东西都据为己有，而且我似乎还有一定程度的强迫症，所以刚刚工作那个时期败家败的相当厉害....说到底，只为玩一个“人无我有，人有我精”，通过物质上的攀比去获得一种优越感，面对他人的种种羡慕和嫉妒，往往还道貌岸然地装作很淡定，很谦虚，实际内心则享受的很。

    然而这种想法是很危险的，物质的东西接触的越多，贪婪也越随之膨胀，这是人类的本性，一旦蔓延开来是很难往回收的。这种贪婪折磨得人们内心无法安宁，久而久之，便导致了整个大环境的浮躁。卖鞋的厂家在这个时候绝不会同情我们，这正是他们希望看到的，所以要趁火打劫，趁着这批人头脑还热的时候多卖他们几双，于是利用各种营销方式轮番轰炸大家的钱包，有些款式还特意少生产一些，以营造出一种供不应求的景象，这时候贩子又应运而生，整个市场越弄越乱。
    近几年我个人倒是很赞同一种心态叫“过眼即是拥有”，因为发现这世上还有太多自己永远都无法得到的东西，能看过它几眼，知道它的存在其实就足矣了。何况现在讨论的根本不是那些传世珍宝，只是流水线上造出来的球鞋而已，完全不必看的太重。

    我错过的 是一双当年所有全明星签名的，诺维茨基第一次入选全明星时在正赛穿过的球鞋，可以说是仅此一双，1/1的限量。但如今再想起这些我并没有格外的懊悔，反而只觉得当时自己那个挣扎的状态挺有意思，最后等于省下了3000多美元，没什么不好。现在经常看一些朋友 因为没抽中签，错过了一双什么联名限量款而难受个半天，其实我认为这并不值得，连买鞋需要抽签这件事我都认为不太正常，因为这些货的量早晚都是人为可控的，近期这个市场确实火爆，但等到哪天真正走上正轨了，人们也不再这么疯狂了，哪天他一着急 曾经那些限量款说造就给你造个多少万双，到时候可能原价 甚至打折就能买到一双，岂不美哉。归根到底 消费者这边最重要的还是应该尽量保持平稳的心态，大家都“过眼即拥有”了，相信这许多的怪现象便会自然消失。



    对于我而言，当年得到纳什GU的JetFlight之后，似乎就渐渐地理解“知足”了，如今GU，GI之类的已经几乎不怎么关注，PE那些也比从前少了许多，再这么下去，可能距离所谓的“出坑”也就不太遥远了。


    若干年前我还曾经画过一幅诺维茨基和纳什，画中的他们正巧穿的正是Air Pure Profile Max与Jet Flight。以如今的眼光回首再看，这张的许多细节都相当不到位，自以为实在拿不出手了。




    不过仍有好心的朋友帮我将它镭射在了一块木板上，并且拿给诺维茨基本人签了名。
 

    在这之前其实也拿过一些画去让他签，不过在木板上的还仅此一次，也是我个人心中十分喜爱的藏品，把它和其他一些纪念品一起摆在了玄关，不论出门还是回家，它们都能在一定程度上美化我一天的心情。


    我个人始终认为，崇拜偶像，买他的鞋和球衣自然非常重要；但若是力所能及的也能为他做点什么，甚至和他本人有些互动，无疑是件更有意思的事。可惜长时间以来我总是眼高手低，想的很多，做的却很少，无形中浪费了不少宝贵的时间。
    不过随着意识到诺维茨基离退役越来越近，再不开始动手，恐怕以后便再没有机会了。于是最近四五年时间，我的业余时间一直在制作诺维茨基一些经典时刻的动画。虽然每天工作就是用电脑弄这些，但我个人还是更喜欢用铅笔的感觉，能力一般，水平有限，自己也没有什么其他特殊技能，无非就是一张一张画。打算今年全部整理完，然后赶在他退役前 能有机会去给他本人看看，最好再顺便握个手什么的...这件事目前也都在按计划顺利的运作之中。今天赶上这个日子，虽然装备区不是聊这些的场合，不妨也在此上传其中一个片段聊以助兴。






    最后我其实并不打算在此对诺维茨基隔空说一句“生日快乐”，因为首先他本人看不见，更看不懂...这实际上就是说给其他人来看的。
    就像前几天父亲节，我看不少人都在社交软件上发个卡通人物的图片，配上个可爱的字体写上“爸爸节日快乐“”爸爸我永远爱你”之类的话，下面是一片点赞，但以我在现实中对那个人的了解，我想他（她）绝对没亲口去跟爸爸说这句话，他们爸爸可能甚至都不知道有个什么父亲节......这并不完全因为我们中国人不擅长当面去表达感情，也是现在这个时代，许多事都已经拧巴了。所以我还是晚一点在推特上直接给他留个言什么的，好歹他应该能扫上一眼。
    
    40岁，现在让我感慨的倒不是他这个岁数，好像也不是时间如此飞快就过去了，更多的 似乎是因为即将面对一个已知的结局 而有些不知所措，尝试过很多次，但仍然无法想象他离开NBA之后自己将是什么状态。只知道现在的愿望只有一个：让这一切都结束的再晚一点吧。
