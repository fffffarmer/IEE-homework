进击的Reebok！DMX的复古潮流势头凶猛

进击的Reebok！DMX的复古潮流势头凶猛 由  识货妞 发表在虎扑装备·运动装备 https://bbs.hupu.com/gear
（原文链接： 进击的Reebok！DMX的复古潮流势头凶猛）
或许对于我们国内的球迷和鞋迷来说，伴随着Reebok这个品牌的名字一起映入脑海的应该就是那个神奇的阿伦艾弗森了。在注定是伟大一代的1996年选秀里，作为状元秀的AI立即就享受到了个人签名鞋的待遇。Reebok question 1的意义无需多言，艾弗森晃过神的那个Crossover，姑且称这个动作是AI的“The Crossover”，而这双鞋就是“The”时刻的见证者。 



在经历了那样一个成功的新秀赛季，Rookie of the year，连续五场得分40+，还有无数像“The Crossover”的高光时刻，Reebok和艾弗森给了“Question”系列一个“Answer”，艾弗森的球鞋Answer系列由此而生，而Reebok也毫无保留地将当时自己手中最好的科技牌DMX搭载其中，这双Reebok Answer 1应该就是很多球迷朋友对于DMX的初印象和初体验了。



Reebok Answer 1

2001年，Reebok answer 4支撑着艾弗森拿到自己的唯一一个常规赛MVP,并且打入总决赛在历史最佳组合之一的ok身上拿走了奥尼尔和科比当年季后赛的唯一一场失利。



无论是answer系列的第一双还是第四双，在充满意义的背景下，这两双鞋绝对是每一位艾弗森球迷的复古潮流首选。



就像现在nike、adidas都要先把缓震科技放置在跑鞋上面，待成熟后再将其投入其他运动领域一样，当年的Reebok也是率先将DMX放置在了一双跑鞋里，创造了一双外观在当时看来极具未来感的DMX Run 10。在去年2017年即DMX科技推出20周年之际，这双当时的未来之鞋迎来复刻，从20年前成功的回到未来。在去年的一阵老爹鞋的复古风潮中，这双DMX Run 10似乎有点不太合群。在老爹鞋中它好像没有那么“老”，整个鞋子充满了不属于“老爹”那个时代的未来感。而在现在的一群轻量化鞋子中，它又显得有些许笨重，或许20年前的设计师们对于未来的展望速度终究还是输给了时间这个家伙。



Reebok DMX Run 10

复古潮流对于不同的人也有不同的解读。有些人认为复古复古自然就一定要忠于元年，原汁原味地还原当年的风格和感觉。而有些人则觉得既然是当代的复古潮流，那在复古的基础上增加现在科技的舒适和便捷，把复古和现代结合起来，何乐而不为呢。



Reebok似乎敏锐地感受到了这两种不同的年轻观点，于是在带来原汁原味的DMX Run 10的同时还带了一双新鞋：DMX fusion



Reebok DMX Fusion于2018年5月正式发布，鞋底外形和科技依旧沿袭元年DMX Run 10的模样，而鞋面则整体全部更换了模板，改成了更轻的ultraknit编织鞋面，舒适性提升的同时保持了它应有的时代未来感。



Reebok Dmx  Fusion

当然在老爹鞋的复古潮流里，Reebok肯定不会拘于一个鞋款。同样搭载DMX的Reebok Daytona DMX也很吸引复古爱好者的眼球。多款配色满足多种不同复古风的搭配，在DMX的强劲势头里，这双daytona dmx也绝对不容小觑。



Reebok Daytona DMX
