其实詹姆斯的第一个总冠军并没有觉得那么的兴奋和激动，感觉就是理所当然的事情，雷霆太年轻！接着随着2012年7月，37岁的雷·阿伦以自由球员身份加盟迈阿密热火队！夏天雷阿伦决定离开效力了5个赛季的凯尔特人，对于君子雷而言这是一个艰难的决定，因为他在绿军有美好的回忆，也有真挚的兄弟情义，08年他们曾一起捧起总冠军奖杯，然而12年夏天，雷阿伦宣布加盟死敌热火，绿军三巨头感情降至冰点。随后无论是加内特，还是皮尔斯退役夜都对雷阿伦避而不谈，甚至怨气十足。但是在堂哥眼里，当年的君子雷看似为总冠军加盟热火，实则和绿军矛盾太深只能被迫离开！很明显雷需要证明自己！可是岁月不饶人已经37了。。基本上快退役了可以说是。。。很多人说他是来了去抱大腿的........没想到成为最粗大腿！！！他的一击3分让马刺GDP悔恨总决赛！！！特别是邓肯真的太遗憾了！往地上重重一拍看的叫人。。。。大家都觉得当时的热火要输了！！！记得当时好多观众席的人都提前离场了，这个是真的，不信可以找视频！！！那个3分进了我在家看人直接从沙发上跳起来的，06年开始看的NBA到那个时候从来没有那么激动，只是觉得这些人特别厉害，但是热火这个总冠军让我感动了！这就是NBA的魅力：Anything is possible ！！！！

然后詹姆斯也站出来了大帽伊戈达拉！尼玛这追帽能力还有谁？？？？ 
好像詹姆斯还有个扣篮被格林拉下来。。。LOVE爱神以为进了。。。激动的跳起来了直接。。。可惜没进被犯规了，摔下来还不轻当时非常担心啊！！！ 

最后赢了篮球往天上一扔哭了。。。。詹姆斯抱着LOVE表情都可以当表情包了！泪流满面！！！然后跪在地板上我相信过好多年以后绝对是经典！！！最后采访的时候说cleveland this is for you 泪奔了！！！ 

欧文2 
1抓地力=耐磨度：底纹路深，前掌内侧外底上翻，变向时抓地非常不错，不过黑金是水晶底耐磨应该一般！前掌纹路形状像指纹识别，抓地优秀。外底较软所以耐磨只能7分。 透气性主要靠鞋舌！夏天穿一定热！冬天应该还好! 
2缓震:感觉欧文鞋中规中矩的吧，欧文那么快的速度和变向太软影响启动速度！7.5分！不会拖泥带水影响启动！不会太硬，也不是那种软的！ 
3包裹性：如处女般。。。哈哈哈 这双鞋比较好的地方了！感觉魔术贴起了很大的作用！！！总体来说还是特别稳定的！10分满分！ 
4抗扭曲：没有碳板的鞋子里面算是比较好的啦！大面积的魔术贴固定脚面发挥了极大的作用！做了一些变向并没有夸张的变形。8.5分 
5脚踝保护：整个鞋子包的都比较紧感觉也不会很差，但是毕竟是后卫鞋还是弄个护踝比较好！毕竟职业球员都打绷带的。。。。8分！ 

6：情怀：不给满分对不起欧文的3分啊！！！！10分！！！！ 








战士10

2缓震:：后掌ZOOM厚度达到14mm！前掌的大块扇形zoom也有8mm！ 9分！ 
3包裹：有的人压脚背根本穿不进去哈哈哈。。。这鞋都没有鞋带只有魔术贴，只适合瘦的人穿。。或者脚背不高的人！但是只要穿进去其实包裹性还可以的！就是这鞋心有点高，你们懂的，我给8.5分吧！ 
4防侧翻：鞋子前掌后掌连接处有一片较大的tpu片来抗扭转 底上，有一个防侧翻的设计。大底在那个地方向外延伸了一小块，防止在突破中双脚外翻导致受伤 这个看人有人觉得可以 有人觉得一般。。。我给个8分吧 
5脚踝：很多人吐槽的地方。。。重心太高容易受伤。。。7分吧。。。 透气性：大棉鞋。。 

6情怀：那一帽就价值10分！总冠军啊！！！
