JS迷彩 
http://item.taobao.com/item.htm?id=19452628643

这是2011年秋冬的作品，现在不太容易买到而且价格也没什么优势，设计感相对比其他JS产品来说比较弱化，但也绝对是好看的款式，不是像@300km 这样的JS高端密的话不建议购买多件，省着买爱疯5吧。




修身西装（多色）
http://item.taobao.com/item.htm?id=13930213532&
价格：108元

原谅我推荐这款被贴上“韩版”tag的西装，不过帅哥的衣柜里怎么没有一件修身款式的休闲西装。如果你觉得黑色太正经？那还有其他很多颜色可以选择，图中关谷君所穿的这件蓝色西装就是不错的选择，年轻有活力又不会太张扬，内搭衬衫或者Tee均可。如果你一年穿不了几次，建议先从价格比较便宜的入手，当然如果预算较多的话也可以购买更好的毕竟一分价钱一分货~




牛津纺7分袖衬衫 日单 （UNITED ARROWS ：green label relaxing）
http://item.taobao.com/item.htm?id=15256787025& 
价格：88元
 
光看图片就已经足够有说服力了吧！牛津纺这种面料的质地一般的布料厚实硬挺，上身效果比较好，7分袖介于长袖和短袖之间，没有长袖那么正紧也没有短袖的夏日感那么强烈，可以说是个恰恰好的时髦的长度。口袋比较特别选用的是黑白条纹针织制成，简洁中不乏细节，88元的价格买到这样的设计和质地实在划算！



水洗磨白 牛仔衬衫 
http://item.taobao.com/item.htm?id=18339268517& 
价格：110元

马上入秋没有几件衬衫傍身怎么可以？什么？格子，素色，条纹都看腻了！牛仔衬衫这两年大行其道，如果你还没有一件的话...恭喜你跻身不时髦男青年的队伍，新学期继续当女神的备胎吧！
另外，眼镜是他的招牌，喜欢的可以备一副！由于眼镜的海洋实在太深邃，识货哥水性不好就不多探究啦，大家根据自己喜好来。

其实识货哥我见过一次王传君本尊，被他的光芒闪瞎了眼！其实关谷君也绝对算得上是潮男了，光是剧终衣服就换了不知道多少套，这边就不一一阐述啦，相信各位看下来心里也大概有个数了，这周我们还有一位重磅级任务“好男人就是我，我就是曾小贤”也会出场喔~
