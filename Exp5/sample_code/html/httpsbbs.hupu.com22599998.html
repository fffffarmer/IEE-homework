微信公众号：贝加乐言

本文翻译自国外网站，首发于微信公众号贝加乐言，文末附有原文链接，可以点击查看。

凯文•杜兰特在这个领域是最受欢迎的球员之一，也是收藏者们的最爱。作为金州勇士队的超巨，得益于今年再次拿到总冠军，他继续将自己的新秀卡价值推到了一个新高度。目前杜兰特已经非常稳定的处于顶级球员之列，接下来如果他获得更多的总冠军，那么这些新秀卡的价格也将随之走高。

杜兰特职业生涯早期从西雅图超音速队起步，这个时期的新秀卡（还有球鞋）被疯狂收藏。之后他在雷霆队待了几年，在那里学到了经验，亲眼见证了几个失败的赛季，那是一支围绕着他建立起来的球队，但尽管在俄城他的身边堆积着众多天赋球员，包括曾经的队友威少，他最终还是选择了加盟勇士，跟库里一起，只为更有希望获得总冠军。

这名超级巨星现在由帕尼尼（Panini）为其独家发行相关产品，不过在他新秀赛季时还不是这样。因此市面上现在有多个品牌超过30种杜兰特新秀卡，价格迥异，以下榜单中的这些是当中的佼佼者。

这些杜兰特新秀卡几乎包含了最好的一些产品和几乎全部主流的新秀签字卡。抛开设计偏好不说，玩家应该可以从中找到自己喜爱的一些产品。当然，这些卡价格也比其余的要高，个别的要以极其昂贵的价格才能买到。

15. 2007-08 Upper Deck Kevin Durant RC #234 SP

这是一款旗舰级产品，由UD出品，对于玩家有限的预算来说，它是一个关键选择。卡面是杜兰特选秀日的图像，整体的布局稍微被底部凌乱的字母有所打乱。这个系列中有几张备受瞩目的新秀球员的新秀卡发行量都不高，杜兰特是其中之一，相对的稀缺性让这张卡的价值略微高一些。

14. 2007-08 Finest Kevin Durant RC #71

Finest篮球系列是个基本上没有什么设计感可言的系列。这种传统在2007-08 Finest篮球系列上得到延续，如果说这张卡上耀眼的亮红色不符合你的口味，那它还有几个折射的平行版本替换掉了红色，包括基础版折射，银（319编），蓝（199编），绿（149编），黑（75编），金（25编）以及X-Factor版本（15编）。

13. 2007-08 Bowman Chrome Kevin Durant RC #111 #/2,999

尽管有一个较高的发行量，杜兰特这张Bowman Chrome新秀卡仍是一个稳妥的选择。Chrome品牌在这个领域一向值得信赖，玩家们总是能被其惊艳的产品吸引到。不过要我来设计这张卡的话，我不会用选秀板来作背景。它也有几个流行的平行版本：基础卡折射（299编），黑（199编），金（99编）以及X-Factor版本（50编）。

12. 2007-08 Topps Chrome Kevin Durant RC #131

跟Finest和Bowman Chrome一样，Topps Chrome也是一个有着许多平行版本的中端产品。在整个榜单里它有着最简洁的设计，整张新秀卡只有一个大的人物动作画面和箔边。平行版本包括折射（1499编），橙（199编），白（99编）及X-Factor版本（50编）。

11. 2007-08 Topps Triple Threads Kevin Durant RC #135 #/99

作为最稀有的基础版杜兰特新秀卡，2007-08 Topps Triple Threads系列用低编弥补了没有签字这一不足。不过虽然99编的限量比较低，但考虑到平行版本——Sepia（66编）、Emerald（33编）、Gold（3编）、Platinum（1/1）及四张模板一编在内，总数超过200张。

10. 2007-08 UD Sweet Shot Kevin Durant RC #101 Autograph #/299

限定区域签字一般来说很少有效果很好的，2007-08 UD Sweet Shot系列恰巧可以作为一个说明为什么会是这样完美的范例。这张卡签字没有直接签在卡上或贴纸上，而是将一块带有签字的材料融入了卡的设计。尽管字迹有消褪的问题，但还是起到了让被遮住的签字看上去就像在Sweet Shot字样下的效果。无论如何，即便是最便宜的杜兰特新秀签字卡，玩家们也始终对它有兴趣。

9. 2007-08 Fleer Hot Prospects Kevin Durant RC #123 Autograph Patch #/399

从榜单整体来看，这张卡有着高限量和贴纸签的缺点，但不妨碍这张Hot Prospects新秀卡是一张不错的卡。当然，从另一方面来说，它也有优点——更容易获得，价格也更合理。红色平行版本（25编）和蓝色平行版本（10编）则要更珍贵一些。

8. 2007-08 SPx Kevin Durant RC #101 Autograph Jersey #/299

这张2007-08 SPx新秀卡可以谈论的点非常多。过重的箔版痕迹和昏暗的图像让你又爱又恨。那块X小型球衣切片增添了亮点，但贴纸签又是让人失望之处。跟上头那张Hot Prospects新秀卡一样，这张卡也是用普通马克笔签在贴纸上，但在这么小的一块区域上看上去不太协调。Radiance平行版本（25编）和Spectrum平行版本（10编）不太容易获得。

7. 2007-08 SP Rookie Threads Kevin Durant RC #49 Autograph Jersey #/199

这张卡在杜兰特新秀卡里有点标新立异，2007-08 SP Rookie Threads这张新秀卡采用了塑料材质，让签字呈现出一种褪散效果。签字直接签在卡上，显著地位于下面那块异形球衣切片上方。金色平行（50编）和银色平行（1/1）更加稀有。

6. 2007-08 Upper Deck Premier Kevin Durant RC #101 Autograph Jersey #/199

UD Premier系列在这个领域很难说得上是一个大品牌，不过UD在持有NBA版权期间还是出品了一些不错的卡。这张硬笔签名卡绝大多数盒子都可以开出来，限量只有199编。大块异形球衣切片是整张卡的焦点，杜兰特的小图像被藏在角落里，看起来好像是事后加上去的一样。基础卡的平行版本有铜（99编），绿（49编），蓝（25编），红（15编）以及金（1/1）。

这个榜单往下还有一张跟这张有着相似外观的UD Black系列的卡，不过那张卡的结构布局更暗一些。

5. 2007-08 Chronology Kevin Durant RC #215 Autograph #/99




限量99编，Chronology系列在卡面边框里采用了动作画面。银色墨迹的卡签又给这张有着漂亮外观的卡带来了新的亮点。系列里有多种杜兰特签字，从而有一个庞大的清单，包括No.104 Mastery（25编），No.135 Green Jersey（99编）以及No.159 Draft photo（99编）。不过这张序号为215，穿着黄色球衣的新秀卡，通常被认为是官方新秀卡。每个版本都有一个带绿边的金箔平行。

4. 2007-08 Upper Deck Ultimate Collection Kevin Durant RC #115 Autograph #/150

2007-08 Upper Deck Ultimate Collection系列对于想要杜兰特低编卡签的玩家来说是个极好的选择。基础版本的限量只有150编，金（50编）和箔版（10编）的平行版本也为收藏者提供了额外的选择。

3. 2007-08 SP Authentic Kevin Durant RC #152 Autograph Patch #/299


SP Authentic系列杜兰特新秀卡或许是这个榜单上最有价值的一张卡。卡签，球衣切片以及出色的外观，玩家们对一张新秀卡的期待这张卡都满足了。299编的限量避免了价格过于昂贵但其实已经相当高了。还有一个零售版保留了主体设计，但去掉了球衣切片，限量级也提高到399编。

2. 2007-08 UD Black Kevin Durant RC #106 Autograph Jersey #/99

球衣切片大小可着卡面来，UD Black系列是一个顶级之选。这个大片的球衣实物使用数字切割模型拼出数字“2007”，限量只有99编，UD Black系列在使用金色墨水进行硬笔签方面迈出了前瞻性的一步。

1. 2007-08 Exquisite Collection Kevin Durant RC #94 Autograph Jersey #/99

最后一张，也是至关重要的一张，2007-08 Exquisite Collection系列这张杜兰特新秀卡是众多收藏者心目中的首要之选。低编、卡签和Patch切片三个因素融为一体，让所有其他杜兰特的新秀卡难以望其项背。

全文完。
