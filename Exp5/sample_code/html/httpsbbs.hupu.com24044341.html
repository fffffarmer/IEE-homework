[流言板]一图流：C罗梦剧场留念，“永远是家一样的感觉” 由  林木有枝 发表在虎扑足球·曼联专区 https://bbs.hupu.com/manutd

虎扑10月23日讯 北京时间本周三凌晨，尤文图斯即将在老特拉福德球场面对曼联。而队内的明星球员C罗也将故地重游，回到了此前驰骋过6年的梦剧场。

C罗于2003-09年在曼联效力了6年，随后他转会前往皇马，在马德里度过了9个春秋，如今他又来到了尤文图斯。

阔别9年之后，C罗也老特拉福德球场合影留念，并且在推特上表示：“感谢大家温暖的欢迎，我在这儿一直有家的感觉。”
