为何中超突然那么多大老板进来烧钱，十几年前，申花虹口的票5毛一张。没什么人看，比坐次公交车便宜，这才是中国足球常态。原因就是现在中国头号老板喜欢足球。比如和曼城球星合影。有句古话怎么说，楚王好细腰，宫中多饿死。许家印，马云之流急着往里砸钱就是献媚，和大老板保持一致，说明我也是喜欢球的。有共同语言。王首富去年被整，后来也投大连一方了。还砸钱搞了个四国赛，邀请威尔士贝尔来，本来想拍马，结果国脚弱智搞砸了，惨败。足协官员出来说，这不是国家队，整顿纹身等等。足协官员会管这些事吗？这肯定是大老板看了这场比赛，心情很差，血压升高后的原话，当然他不会自己找足协官员训斥。让个秘书把话传到。足协官员原话放送。足协官员现在疯狂急功近利要成绩，就是这个原因。国家队再踢不好，这些职业球员捞钱也到头了，取消职业联赛也有可能
