虎扑10月23日讯 目前保持着两战全胜的开拓者坐镇主场迎战开局两战皆墨的奇才，首节比赛麦科勒姆5次出手全部打铁仅依靠罚球拿到2分，另外一位外线得分手利拉德虽然拿到8分2助攻，但仅命中一记运动战进球，虽然两位得分手运动战哑火，但凭借着多点开花的表现，他们首节32-28领先奇才。次节麦科勒姆依旧找不到手感，这一节他6投1中状态依旧糟糕，奇才这边比尔迎来大爆发，他这一节命中三记三分一人砍下15分，奇才也在最后时刻追上比分，半场战罢，双方62-62战成平手。

易边再战，麦科勒姆的打铁还在继续，这一节依旧是5投1中进攻端无法帮助到球队，奇才这一节虽然一直保持着微弱优势，但进攻端状态同样糟糕无法拉开比分，三节结束时奇才1分领先开拓者。末节双方依旧难分胜负悬念一直留到最后，最后的对飙非常精彩但比分一直平手，斯陶斯卡斯最后的三分进球曾帮助开拓者取得领先，但比尔大心脏回敬一记三分将比赛拖入加时。

加时赛中麦科勒姆和努尔基奇相继取得进球，但奇才这边莫里斯和乌布雷外线两记三分帮助奇才占据主动，莫里斯最后时刻成为功臣，他先是打成2+1，随后命中致命三分杀死比赛。最终，奇才125-124险胜开拓者迎来新赛季首胜。

奇才数据：布拉德利-比尔25分8篮板8助攻；约翰-沃尔16分4篮板9助攻；奥托-波特16分10篮板2抢断；马基夫-莫里斯28分9篮板；凯利-乌布雷22分6篮板；杰夫-格林12分6篮板。

开拓者数据：达米安-利拉德29分7篮板8助攻；优素福-努尔基奇22分18篮板；CJ-麦科勒姆25投仅5中拿到13分4篮板4助攻；艾尔-法鲁克-阿米奴16分15篮板；尼克-斯陶斯卡斯15分8篮板；埃文-特纳10分7篮板3助攻。

[伤病名单]

奇才：德怀特-霍华德臀部受伤缺战。

开拓者：全员健康。
